Summary :
협력업체 관리 요약 카드와 필터 표시 마감

Description :
협력업체 관리 화면의 상단 요약 카드를 필터와 분리해 전체 데이터 기준으로만 표시하도록 정리했다. 요약 카드의 어두운 블록형 배경을 제거하고 테마 surface, border, accent 기준의 밝은 카드 톤으로 보정했다. 영어 업체 추가 버튼과 필터 버튼 줄바꿈을 방지하고, 필터 라벨과 설명 문구는 테마 텍스트 토큰 기준으로 정리했다.
