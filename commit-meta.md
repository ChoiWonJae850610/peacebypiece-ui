Summary :
직접 그리기 전체 화면 작업창 전환

Description :
직접 그리기 화면을 일반 모달형에서 전체 화면 작업창 형태로 전환하고, 상단 설명 문구를 제거해 태블릿 가로/세로 화면에서 캔버스 영역을 더 넓게 사용할 수 있도록 정리했다. 하단 도구 모음과 저장 액션은 항상 접근 가능한 영역에 배치하고, 기존 PNG 저장 및 디자인 첨부 업로드 흐름은 유지했다.
