Summary :
고객관리자 메인 대기목록 스크롤과 대표이미지 표시 보정

Description :
고객관리자 메인 화면에서 상단 대기 영역 높이를 고정하고 좌측 대기목록만 스크롤되도록 보정했다. 주요 대기 현황 카드는 4개 항목이 스크롤 없이 보이도록 유지하고, 작업지시서 미리보기는 대표 이미지 키와 저장 키를 활용해 표시되도록 보완했다. 수량 표시는 클라이언트 i18n 기준으로 다시 포맷되도록 정리했다.
