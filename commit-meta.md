Summary :
개인 테마 회귀 점검용 Cold Winter 테마 추가

Description :
개인 설정 테마 전환 범위 회귀 점검을 위해 cold-winter 테마 파일을 추가하고 theme registry, theme type, 개인 설정 i18n 라벨에 연결했다. 작업지시서와 관리자 화면의 semantic token이 default-light, beige-atelier, cold-winter 전환을 따라가는지 확인할 수 있도록 회귀 점검 문서와 token 체크 항목을 보강했다.
