# 0.10.24 저장소 현황 카드와 휴지통 목록 높이 균형 보정

## 목적

0.10.23에서 휴지통 목록 표시량을 확보하면서 상단 저장소 현황 카드 3개가 지나치게 낮아 보이는 문제를 보정한다.

## 변경

- 저장소 현황 카드 3개의 최소 높이와 내부 여백을 소폭 복원했다.
- 요금제 용량 그래프와 파일 유형 도넛 차트를 약간 키웠다.
- 파일 운영 요약 행 간격을 약간 복원했다.
- 휴지통 목록은 기존 내부 스크롤 구조를 유지하되, 상단 카드가 커진 만큼 표시 행 수가 약 1~2줄 줄어들도록 했다.

## 유지

- 외부 “저장소 사용 현황” wrapper는 복구하지 않는다.
- 새로고침 버튼 위치와 토스트 피드백은 유지한다.
- 복원/선택 삭제/비우기/R2 purge/감사 로그 흐름은 변경하지 않는다.
- DB schema는 변경하지 않는다.
