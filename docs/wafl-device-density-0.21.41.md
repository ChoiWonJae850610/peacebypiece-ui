# WAFL device density 0.21.41

## 목표

PC 기준으로만 맞춘 WAFL Foundation 적용을 모바일·태블릿 compact 화면까지 확장한다.

## 정리 범위

- 멤버관리 compact list row / meta box density
- 저장소 휴지통 compact row / empty state density
- 설정 회사 파일 preview box의 모바일·태블릿 높이
- 개인설정 모달 input / choice card / withdrawal card density

## 기준

- shape는 control로 고정한다.
- 모바일에서는 padding과 preview height를 낮춘다.
- 태블릿 이상에서는 기존 정보량을 유지한다.
- color swatch, progress bar, chart dot처럼 의미가 있는 원형/진행률 UI는 예외로 유지한다.
