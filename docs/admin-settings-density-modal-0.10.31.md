# 0.10.31 고객관리자 환경설정 밀도와 모달 흐름 보정

## 목적

0.10.30에서 재구성한 고객관리자 환경설정 허브를 더 실무형 화면으로 다듬는다.

## 변경 기준

- 환경설정 상단 메뉴 카드의 높이와 여백을 줄인다.
- 기준정보 설정 외 메뉴는 별도 본문 placeholder로 전환하지 않고 즉시 안내 모달만 연다.
- 기준정보 설정 영역은 계속 본문에 유지한다.
- 권한 관리는 환경설정이 아니라 멤버관리 화면에서 다루는 방향을 유지한다.

## 제외

- 실제 알림 발송 구현 없음
- 실제 결제/계정/회원 탈퇴 구현 없음
- 개발 건의 DB 저장 구현 없음
- DB schema 변경 없음
- 감사 로그 흐름 변경 없음
