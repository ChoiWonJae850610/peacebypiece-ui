# 작업지시서 발주정보/생산구성 금액 요약 회귀 보정 (0.11.89)

## 목적

작업지시서 상세의 발주정보와 생산구성 요약 금액이 PC, 태블릿, 모바일에서 같은 계산 기준으로 표시되도록 정리한다.

## 반영 기준

- 발주정보 요약 금액은 발주 수량 기반 공임비 합계와 로스비 합계를 더한 값이다.
- 생산구성 요약 금액은 원단/부자재 금액 합계와 외주공정 금액 합계를 더한 값이다.
- 생산구성 요약 문구는 공통 formatting helper를 사용한다.
- 태블릿 생산구성 테이블은 한 줄 row와 좌우 스크롤 구조를 유지한다.

## 확인 항목

- PC 발주정보 요약에 건수, 장수, 검수 완료, 총 금액이 함께 표시되는지 확인한다.
- 모바일/태블릿 발주정보 합계 카드에 총 금액이 표시되는지 확인한다.
- PC/모바일/태블릿 생산구성 요약에 원단/부자재 건수, 외주공정 건수, 총 금액이 같은 기준으로 표시되는지 확인한다.
- 비용 요약 카드의 총합과 발주정보/생산구성 section 금액이 각각의 계산 범위에 맞게 표시되는지 확인한다.
