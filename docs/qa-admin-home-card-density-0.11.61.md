# 0.11.61 고객관리자 홈 카드 레이아웃 확대

## 목적

고객관리자 홈의 운영 관리 카드가 화면 대비 작고 하단 공간이 비어 보이는 문제를 보정한다.

## 반영 범위

- `/admin` 운영 관리 카드 영역
- `components/admin/dashboard/AdminConsoleSections.tsx`

## 변경 내용

- 운영 관리 섹션에 최소 화면 높이를 부여해 하단 공간을 더 적극적으로 사용하도록 조정했다.
- 카드 grid를 2열/3열 중심으로 정리하고 6열 축소 표시를 제거했다.
- 카드 padding, 최소 높이, 제목 크기, 설명 크기, action badge 크기를 확대했다.
- 카드 목록이 같은 높이로 배치되도록 `auto-rows-fr`와 `flex-1` 구조를 적용했다.

## 유지한 것

- 카드 링크 대상 변경 없음
- 권한 기반 카드 노출 조건 변경 없음
- 고객관리자 IA 변경 없음
- 시스템관리자 화면 변경 없음
- DB/API 변경 없음

## 확인 항목

1. `/admin` 진입
2. 운영 관리 카드가 기존보다 크게 보이는지 확인
3. PC 폭에서 카드가 화면 하단까지 더 가득 찬 느낌으로 배치되는지 확인
4. 협력업체 관리, 저장소 관리, 통계정보, 환경설정, 멤버 관리, 작업지시서 업무 화면 링크 이동 확인
5. 모바일 폭에서 카드가 세로로 자연스럽게 쌓이는지 확인
