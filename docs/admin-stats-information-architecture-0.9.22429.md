# 0.9.22429 관리자 통계 화면 정보 구조 재배치

## 목표

통계정보 화면을 누적/전체 지표 우선 구조로 재정렬하고, 기간별 분석은 하단에서 조작하는 영역으로 이동한다.

## 변경 내용

- 상단 summary 카드 4개 아래에 전체 통계 영역을 먼저 배치했다.
  - 생산품 유형 비율
  - 선택항목 품목 TOP5
  - 업체 성과
  - 업체별 납기·검수 지표
- 기존 작업흐름 분석 카드는 제거했다.
- 기간별 분석 영역은 하단으로 이동했다.
- 기간 선택 영역은 헤더 라인에 압축 배치했다.
  - 7일 / 30일
  - 초기화 아이콘
  - 적용 버튼
  - DayPicker popover 트리거
- 날짜 직접 입력 필드는 추가하지 않았다.
- 기간 요약 항목을 클릭 가능한 필터로 변경했다.
- 선택한 기간 요약 항목에 따라 TOP5 카드 제목과 데이터가 바뀐다.

## 기간 요약 연동 TOP5 기준

- 완료 작업지시서
  - TOP5 제목: 발주수량 상위 TOP5
  - 기준: 선택 기간 안에서 완료된 작업지시서의 발주수량 합계
- 리오더
  - TOP5 제목: 리오더 TOP5
  - 기준: 선택 기간 안에서 리오더 작업지시서 수
- 불량 작업지시서
  - TOP5 제목: 불량 작업지시서 TOP5
  - 기준: 선택 기간 안에서 불량 또는 재작업으로 표시된 작업지시서 수

## DB 변경

없음.

## 후속 확인

- 불량 수량 전용 컬럼이 생기면 불량 TOP5 기준을 “작업지시서 수”에서 “불량 수량 합계”로 변경할 수 있다.
- 현재는 `status = rejected` 또는 `is_rework = true` 기준을 사용한다.
