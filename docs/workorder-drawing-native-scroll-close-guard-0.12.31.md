# 0.12.31 native 직접 그리기 모달 스크롤/닫힘 방지 보정

## 목적

직접 그리기 모달에서 하단 툴바와 저장 버튼이 처음부터 보이지 않고, 모달 내부 스크롤을 내려야 조작 가능한 문제를 보정한다. tablet 사용 중 실수로 브라우저 뒤로가기나 모달 외부 터치로 작업이 중단되지 않도록 accidental close 방지도 함께 정리한다.

## 반영 내용

- 모달 body를 flex 기반 고정 레이아웃으로 정리
- 모달 body의 외부 세로 스크롤을 차단
- 캔버스 영역은 남는 높이를 차지하게 하고, 하단 툴바와 footer는 처음부터 보이도록 조정
- 직접 그리기 모달의 backdrop click 닫기 금지 유지
- 모달 열림 중 브라우저 뒤로가기 가드 추가
- 뒤로가기 입력 시 페이지 이동 대신 모달을 유지하고 안내 메시지 표시
- 저장하지 않은 그림이 있는 상태에서 닫기 버튼을 누르면 확인 UI 표시
- 저장 완료 시 기존 PNG 디자인 첨부 업로드 흐름 유지

## 유지한 정책

- native canvas 직접 그리기를 기본 기능으로 유지
- tldraw 고급 그리기 development + feature flag 정책 유지
- DB schema 변경 없음
- R2/첨부 API/휴지통/purge 흐름 변경 없음

## 확인 항목

1. 직접 그리기 모달 첫 진입 시 하단 툴바가 바로 보이는지 확인
2. 저장/전체 지우기 footer가 스크롤 없이 보이는지 확인
3. 캔버스가 너무 작아지지 않았는지 PC/tablet에서 확인
4. tablet에서 모달 바깥쪽을 터치해도 닫히지 않는지 확인
5. 모달이 열린 상태에서 브라우저 뒤로가기를 눌러도 페이지가 이동하지 않는지 확인
6. 그린 내용이 있는 상태에서 닫기 버튼을 누르면 확인 UI가 뜨는지 확인
7. 저장 후 PNG가 기존 디자인 첨부 목록에 추가되는지 확인
