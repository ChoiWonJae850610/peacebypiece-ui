# 협력업체관리 필터 toolbar 실제 폭 확장 보정 — 0.18.48

## 목적

0.18.47 적용 후에도 협력업체관리 검색/유형/사용 상태 필터가 왼쪽에 붙어 보이는 문제를 보정한다.

## 변경

- `PartnerMasterFilters` 내부 최상위 grid에 `w-full`을 추가했다.
- `AdminFilterBar`가 flex container이므로, 내부 grid가 실제 bar 전체 폭을 차지하지 못하던 구조를 보정했다.
- 1120px 이상에서는 검색 영역이 `minmax(0, 1fr)`로 남는 폭을 차지하고, 유형/상태 select 그룹은 400px 고정 폭으로 우측에 배치되도록 조정했다.
- 720px 이상 1120px 미만에서는 기존처럼 검색창 1줄 + 유형/상태 select 우측 정렬 흐름을 유지한다.
- 720px 미만에서는 검색창 전체폭 + 유형/상태 2열 흐름을 유지한다.

## 유지

- 현재 목록 count 제거 상태 유지
- 협력업체 목록 wide table / compact card 반응형 구조 유지
- 검색/필터/정렬/등록/수정 모달 흐름 변경 없음
- WorkspaceShell 스크롤 구조 변경 없음
- DB/API/협력업체 저장 흐름 변경 없음
