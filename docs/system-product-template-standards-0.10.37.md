# 0.10.37 시스템관리자 생산품 유형 기본 템플릿 1차 화면

## 목표

시스템관리자 기준정보 관리 흐름에 생산품 유형 기본 템플릿 화면을 추가한다.

단위 표준과 외주공정 유형은 시스템 표준 원장으로 관리하고 고객사는 사용 여부만 선택한다. 반면 생산품 유형은 고객사마다 분류 방식이 다를 수 있으므로 고객관리자 직접 관리를 유지한다. 시스템관리자는 신규 고객사 생성 시 복사할 기본 템플릿만 관리한다.

## 반영 내용

- `/system/standards/product-templates` 경로 추가
- 생산품 유형 기본 템플릿 1차 화면 추가
- 1차 → 2차 → 3차 계층 미리보기 구성
- 시스템 기준정보 설계 화면에서 템플릿 화면 링크 연결
- 시스템 콘솔 빠른 링크에 생산품 유형 템플릿 추가
- 실제 CRUD, DB schema, 고객사 생성 시 복사 로직은 후속 버전으로 분리

## 변경하지 않은 것

- 고객관리자 생산품 유형 저장 로직
- 단위 표준/외주공정 유형 고객사 사용 여부 저장 로직
- DB schema
- 감사 로그 흐름
