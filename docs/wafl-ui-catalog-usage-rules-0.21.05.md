# WAFL UI Catalog 0.21.05

## 목적
`/ui` 카탈로그를 새 화면 개발자가 바로 판단할 수 있는 사용 기준 화면으로 보강했다.

## 변경 내용
- Usage rules 섹션을 추가했다.
- 컴포넌트를 언제 쓰는지/언제 쓰지 않는지 구분했다.
- 직접 className 사용 금지 기준을 추가했다.
- rounded/shadow/border/bg 직접 조합의 대체 기준을 정리했다.
- data-wafl-component naming 규칙을 추가했다.
- 새 화면 개발 체크리스트를 추가했다.
- debug outline 재활성화 위치를 안내했다.

## 운영 메모
- `/ui` 접근 제한 임시 해제 상태는 유지한다.
- WAFL debug outline 비활성 상태는 유지한다.
- 기존 업무 화면 로직은 변경하지 않는다.
