# 0.20.33 환경설정 모바일 구조 정리

## 목표

환경설정 화면의 모바일 구조를 별도 화면 전용 Shell로 분리하지 않고, 기존 WorkspaceShell과 WAFL 공통 설정 컴포넌트 위에서 보정한다.

## 반영 범위

- 환경설정 상단 메뉴 탭을 모바일에서 가로 스크롤 가능한 카드형 탭으로 보정
- 현재 선택된 환경설정 섹션을 모바일 요약 카드로 표시
- 설정 공통 카드와 섹션 그룹의 모바일 padding, title, description 밀도 보정
- 회사 파일 카드의 모바일 미리보기 높이와 버튼 영역 정리
- PC/태블릿 환경설정 화면 구조는 기존 카드/탭 구조 유지

## 변경하지 않은 범위

- 상태전환 로직
- 권한 로직
- API
- DB schema
- R2 업로드/미리보기 처리
- 회사 파일 저장/검토 상태 처리
