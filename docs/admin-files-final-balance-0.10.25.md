# 0.10.25 저장소 관리 최종 높이 균형 보정

## 목표

저장소 관리 화면에서 상단 현황 카드 3개가 너무 작아 보이고 휴지통 목록이 과도하게 많이 보이던 균형을 조정한다.

## 변경 내용

- 상단 현황 카드 3개의 최소 높이와 내부 여백을 소폭 확대했다.
- 요금제 용량 그래프와 파일 유형 차트를 약간 키웠다.
- 파일 운영 요약 행 간격을 약간 복원했다.
- 상단 카드가 커진 만큼 휴지통 목록 표시량이 약 1줄 줄어드는 방향으로 조정했다.
- 새로고침 클릭 시 완료 메시지는 표시하지 않고 진행 메시지만 표시하도록 했다.

## 유지한 것

- 외부 “저장소 사용 현황” wrapper는 복구하지 않는다.
- 새로고침 버튼 위치는 휴지통 액션 라인에 유지한다.
- 휴지통 내부 스크롤 구조를 유지한다.
- 복원, 선택 삭제, 비우기, R2 purge, 감사 로그 흐름은 변경하지 않는다.
