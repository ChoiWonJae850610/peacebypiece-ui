# 0.9.22391 저장소 휴지통 정렬/row 톤 보정

## 목적

고객관리자 저장소 휴지통 목록에서 컬럼 헤더 클릭 정렬을 추가하고, 선택 가능한 항목과 작업지시서에 묶여 선택할 수 없는 첨부파일 row의 시각 톤을 분리한다.

## 반영 내용

- 휴지통 목록 컬럼 헤더 클릭 정렬 추가
  - 대상
  - 삭제일시
  - 작업지시서
  - 유형
  - 크기
- 컬럼을 다시 클릭하면 오름차순/내림차순이 전환된다.
- 작업지시서에 함께 묶인 첨부파일은 부모 작업지시서 row 아래에 유지되도록 그룹 단위 정렬을 적용했다.
- 대상 컬럼 헤더는 가운데 정렬하고, 목록 내부 대상명은 좌측 정렬을 유지했다.
- 선택 가능한 작업지시서/개별 첨부파일 row와 선택할 수 없는 묶음 첨부파일 row의 배경 톤을 분리했다.

## 변경하지 않은 것

- DB schema 변경 없음
- R2 Worker 흐름 변경 없음
- 휴지통 복원/선택 삭제/비우기 로직 변경 없음
- package.json / package-lock.json 변경 없음
