# 0.9.223793 저장소 휴지통 테이블 컬럼 고정 정렬 보정

## 변경 목적

고객관리자 저장소 휴지통 목록에서 작업지시서 row와 첨부파일 row가 섞일 때 선택, 대상, 삭제일시, 작업지시서명, 유형, 용량 컬럼이 서로 독립적인 컬럼처럼 보이지 않는 문제를 보정한다.

## 변경 내용

- 휴지통 테이블 grid 컬럼을 고정 폭 중심으로 재정의했다.
- 선택 컬럼은 checkbox가 있든 대체 표시가 있든 동일한 크기와 세로 중앙 정렬을 사용한다.
- 대상 컬럼은 작업지시서와 첨부파일 모두 좌측 정렬로 통일했다.
- 첨부파일의 계층 표시와 들여쓰기는 대상 컬럼 내부에서만 처리한다.
- 삭제일시, 작업지시서명, 유형, 용량 컬럼은 가운데 정렬을 유지한다.
- 묶음 첨부 row의 좌측 강조선은 border가 아니라 inset shadow로 처리해 컬럼 위치가 밀리지 않도록 했다.

## 검증 기준

1. `/admin/files` 저장소 관리 화면에 진입한다.
2. 휴지통에 작업지시서 row와 첨부파일 row가 함께 보이도록 한다.
3. 선택 컬럼의 체크박스와 대체 표시가 같은 세로선에 맞는지 확인한다.
4. 대상 컬럼의 작업지시서명과 파일명이 모두 좌측 기준으로 정렬되는지 확인한다.
5. 삭제일시/작업지시서명/유형/용량 컬럼이 각 row에서 같은 위치에 유지되는지 확인한다.
