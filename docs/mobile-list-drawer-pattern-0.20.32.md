# 0.20.32 모바일 목록 드로어 패턴 통일

## 목적

작업지시서와 원단·부자재 발주 모바일 화면에서 별도 목록 화면으로 돌아가는 흐름을 줄이고, 현재 상세 화면을 유지한 채 목록 드로어를 열어 항목을 전환하는 구조로 통일한다.

## 작업지시서

- 기존 모바일 목록 화면 전환 구조를 제거했다.
- 모바일 본문은 항상 상세 영역을 유지한다.
- 작업지시서 목록은 기존 상단 메뉴 버튼으로 여는 모바일 드로어를 사용한다.
- 하단 고정 액션바에서는 목록 복귀 버튼을 제거하고 첨부, 디자인, 메모 진입 버튼만 유지한다.
- 첨부/디자인/메모 full-size sheet 흐름은 유지한다.

## 원단·부자재 발주

- 모바일에서 별도 목록 화면으로 돌아가는 구조를 제거했다.
- 발주 상세 화면을 유지한 채 발주서 목록을 left sheet drawer로 연다.
- 발주서 선택 시 목록 drawer만 닫고 상세 화면을 교체한다.
- 하단 고정 액션바는 발주서 목록 drawer, 자재 선택 sheet, PDF·납기 예약 위치로 정리했다.

## 유지 사항

- PC 3패널 구조 유지
- 태블릿 2패널 구조 유지
- 상태전환 로직 변경 없음
- 권한 로직 변경 없음
- API/DB/R2/첨부/메모/휴지통/purge 흐름 변경 없음
