# 0.11.21 관리자 협력업체 관리 공통 UI 적용

## 목적

관리자 협력업체 관리 화면의 주요 버튼과 상태 라벨을 관리자 공통 UI 컴포넌트 기준으로 맞춘다.

## 변경 범위

- 협력업체 생성 버튼을 `AdminButton`으로 전환
- 협력업체 필터 chip 버튼을 `AdminButton`으로 전환
- 협력업체 목록의 활성/비활성 및 유형 라벨을 `AdminStatusBadge`로 전환
- 협력업체 등록/수정 모달 footer 버튼을 `AdminButton`으로 전환
- 협력업체 사용 상태 라벨을 `AdminStatusBadge`로 전환
- 외주공정 이동 버튼과 삭제 확인 모달 버튼을 `AdminButton`으로 전환

## 비변경 범위

- 협력업체 CRUD API
- 협력업체 필터/검색 로직
- 외주공정 활성/비활성 이동 로직
- DB schema
- i18n key 구조

## 확인 필요

- `/admin/partners`에서 생성/수정 모달 열기
- 협력업체 유형 필터와 상태 필터 동작
- 협력업체 목록 라벨 표시
- 외주공정 관리 모달의 이동/저장/초기화 동작
- 삭제 확인 모달 버튼 표시
