# 0.9.223792 — 저장소 휴지통 선택 컬럼 정렬 보정

## 목적

고객관리자 저장소 휴지통 목록에서 작업지시서 row와 첨부파일 row가 섞여 표시될 때 선택 컬럼의 체크박스/대체 표시가 줄 중앙에 맞지 않아 보이는 문제를 보정한다.

## 반영 내용

- 선택 컬럼 전용 셀 클래스를 분리했다.
- 선택 컬럼 셀에 동일한 최소 높이와 세로 중앙 정렬 기준을 적용했다.
- 작업지시서 row, 첨부파일 row, 묶음 첨부 row가 섞여도 선택 컬럼의 시각 기준선이 흔들리지 않도록 정리했다.

## 확인 항목

1. /admin/files 휴지통 목록에 진입한다.
2. 작업지시서 row와 첨부파일 row가 섞인 상태를 확인한다.
3. 선택 컬럼 체크박스가 각 row 중앙에 맞게 보이는지 확인한다.
4. 대상 컬럼과 삭제일시/작업지시서/유형/용량 컬럼 정렬이 기존 의도대로 유지되는지 확인한다.
