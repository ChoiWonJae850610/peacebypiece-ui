# 0.9.220 — 작업지시서 상세 UX 섹션 정리

## 목적
작업지시서 PC 상세 화면에서 비용 요약, 발주 정보, 생산 구성 섹션의 시각 위계를 정리한다.
이번 버전은 원단/부자재 발주 화면을 새로 만들지 않고, 기존 상세 화면의 정보 밀도와 빈 상태 표현을 안정화하는 작업이다.

## 변경 범위
- 비용 요약 섹션에 그룹 제목과 설명을 추가한다.
- 총 비용과 단가를 먼저 확인할 수 있도록 비용 요약 카드를 재정리한다.
- 발주 정보 섹션에 그룹 제목과 설명을 추가한다.
- 생산 구성 섹션에 그룹 제목과 설명을 추가한다.
- 발주/원단/외주 테이블에서 데이터가 없을 때 빈 상태 행을 표시한다.
- 좁은 PC 패널에서 테이블이 잘리지 않도록 내부 가로 스크롤을 허용한다.
- 버튼과 카드 여백을 소폭 줄여 상세 화면 정보 밀도를 안정화한다.

## 제외 범위
- 원단/부자재 발주 화면 신규 구현 없음
- DB schema 변경 없음
- API route 변경 없음
- 저장 로직 변경 없음
- 권한/요금제 로직 변경 없음

## SQL DDL 필요 여부
불필요

## 전체 리셋 필요 여부
불필요

## 테스트 케이스
1. `/worker` 접속 후 작업지시서 상세 화면을 확인한다.
2. 비용 요약, 발주 정보, 생산 구성 그룹 제목이 표시되는지 확인한다.
3. 총 비용 카드가 비용 요약의 핵심 정보로 먼저 보이는지 확인한다.
4. 발주 정보가 비어 있을 때 빈 상태 행이 표시되는지 확인한다.
5. 원단/부자재 정보가 비어 있을 때 빈 상태 행이 표시되는지 확인한다.
6. 외주공정 정보가 비어 있을 때 빈 상태 행이 표시되는지 확인한다.
7. 좁은 PC 폭에서 발주/생산 구성 테이블이 카드 밖으로 깨지지 않는지 확인한다.
8. 기존 추가/삭제/편집 동작이 유지되는지 확인한다.
