# 0.9.22411 저장소관리 화면 i18n 잔여 문구 정리

## 목적

고객관리자 `/admin/files` 저장소관리 화면과 휴지통 상세 모달에서 영어 문구와 첨부 중심 표현이 섞여 보이는 문제를 정리한다.

## 반영 내용

- 저장소 요약 카드의 `Storage usage`, `File operations`, `File type` 계열 문구를 i18n key 기반 한국어 도메인 문구로 정리했다.
- 휴지통 상세 모달의 `Close`, `Restore`, `Request deletion`, `TYPE`, `SIZE`, `WORK ORDER`, `DELETED` 계열 문구가 `filesList` dictionary를 타도록 정리했다.
- 작업지시서 상세 모달의 `묶음 처리 첨부`, `연결 파일 용량`, `첨부된 파일/메모` 표현을 `문서/디자인/메모`, `파일 용량` 기준으로 보정했다.
- 문서/디자인 목록과 빈 상태 문구의 fallback을 `첨부파일` 중심에서 `문서/디자인` 중심으로 보정했다.
- `en/admin.ts`의 저장소관리 도메인 문구도 현재 고객관리자 화면 기준 한국어 문구로 보정해 언어 설정이 영어로 남아 있어도 저장소관리 화면에서 혼합 표기가 줄어들도록 했다.

## 제외한 내용

- 삭제일시 시간대 차이 문제는 이 버전에서 처리하지 않았다.
- 다음 버전에서 DB 저장 시간, 서버 변환 시간, 화면 표시 시간의 기준을 분리해 점검한다.
