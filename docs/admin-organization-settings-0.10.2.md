# PeaceByPiece 0.10.2 — 관리자 조직 설정 재정리

## 목표

`/admin/settings`를 고객사 전체에 적용되는 조직 설정 화면으로 고정한다. 개인 언어, 색상, 화면 밀도, 기본 진입 화면은 `/me/settings`에서 관리한다.

## 적용 내용

- `/admin/settings` 상단을 조직 설정 요약 카드로 교체했다.
- 개인 설정으로 분리된 언어/색상 설정을 조직 설정 상단에서 제거했다.
- 조직 설정 범위를 파일 정책, 저장소 정책, 알림 정책, 기준정보, 멤버 권한 후보로 정리했다.
- 개인 설정 이동 버튼을 `/me/settings`로 연결했다.
- 조직 설정 표시 문구를 i18n에 추가했다.

## 유지한 범위

- DB schema 변경 없음.
- 권한 DB, 멤버 초대, 권한 관리 UI는 아직 구현하지 않음.
- 기존 파일 정책, 기준정보, 사용자/권한 미리보기 영역은 유지.
- `/admin/settings` 내부에서 실제 저장 로직은 새로 추가하지 않음.

## 후속 작업

- `/me/settings` 사용성 보정.
- 멤버관리/권한관리 설계.
- 권한 기반 카드 노출과 조직 설정 권한 연결.
