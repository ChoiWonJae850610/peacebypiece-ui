# 0.9.145 공통 UI/레이아웃 리팩토링 1차

## 목적

0.9.77 복구 라인의 기존 화면을 유지하면서 관리자 설정 모달군의 사용/미사용 표시 방식을 우선 통일한다.
이번 버전은 관리자 환경설정 페이지에서 눈에 띄던 토글/버튼/배지 혼재 문제를 줄이는 1차 리팩토링이다.

## 적용 범위

- 파일 정책 모달
- 알림 정책 모달
- 생산품 유형 모달
- 단위 표준 모달

## 변경 원칙

1. 사용/미사용 상태 변경은 공통 `AdminUsageToggle`을 사용한다.
2. 토글 크기는 기존 `StatusToggle size="sm"` 기준으로 통일한다.
3. 텍스트 배지만 눌러 상태를 바꾸던 UI는 토글 기반으로 맞춘다.
4. 행 선택과 사용 여부 변경은 분리한다.
5. 저장 중인 모달에서는 가능한 범위에서 토글/입력을 비활성화한다.
6. 기능 동작은 기존 상태 저장 구조를 유지한다.

## 이번 버전에서 하지 않은 일

- 관리자 화면 전체 디자인 교체
- 작업지시서 UI 구조 변경
- 시스템관리자 purge 기능 변경
- DB schema 변경
- package 변경

## 다음 리팩토링 후보

- 외주 공정 유형 모달의 좌우 이동 버튼/리스트 높이 통일
- 관리자 파일관리 테이블의 액션 버튼 높이 통일
- admin/system 공통 카드 헤더와 요약 카드 컴포넌트 정리
- modal footer 버튼 정렬/폭 기준 문서화
