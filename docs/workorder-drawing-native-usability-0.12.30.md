# 작업지시서 native 직접 그리기 사용성 안정화 (0.12.30)

## 목표

직접 그리기 모달의 tablet 사용성을 안정화한다.

## 반영 내용

- 지우개 크기를 기존 굵기 선택과 연결했다.
- 펜/지우개/도형 선택 상태에 따라 굵기 선택 label을 다르게 표시한다.
  - 펜: 굵기
  - 지우개: 지우개 크기
  - 직선/화살표/사각형/원: 선 굵기
- 지우개 cursor 원형 표시 크기를 실제 지우개 크기와 맞췄다.
- 모달 body의 외부 스크롤을 줄이고, 캔버스와 하단 toolbar가 처음 열릴 때 함께 보이도록 layout을 조정했다.
- 캔버스 영역은 남는 높이를 차지하도록 하고 하단 toolbar는 모달 내부에서 계속 접근 가능하게 유지했다.

## 유지한 정책

- 텍스트 삽입은 추가하지 않았다.
- 기본 의류 템플릿 배경은 추가하지 않았다.
- 기존 PNG 저장 및 디자인 첨부 업로드/R2 흐름은 변경하지 않았다.
- tldraw 고급 그리기의 development + feature flag 정책은 변경하지 않았다.
