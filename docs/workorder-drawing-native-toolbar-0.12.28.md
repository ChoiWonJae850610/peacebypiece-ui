# 작업지시서 native 직접 그리기 툴바 UX 정리 (0.12.28)

## 목적

직접 그리기 기능의 도구가 늘어나면서 텍스트 버튼 중심 UI가 복잡해졌다. 이번 버전에서는 native canvas의 기본 기능은 유지하면서, 태블릿에서 빠르게 선택할 수 있도록 툴바를 아이콘 중심으로 정리한다.

## 변경 기준

- 텍스트 삽입 기능은 추가하지 않는다.
- 기존 펜, 지우개, 직선, 화살표, 사각형, 원/타원 기능은 유지한다.
- 기존 색상, 굵기, undo, redo, 전체 지우기, PNG 저장 흐름은 유지한다.
- 기존 디자인 첨부 업로드/R2 저장 흐름은 변경하지 않는다.
- tldraw 고급 그리기 development feature flag 정책은 변경하지 않는다.

## UX 변경

### 도구 버튼

도구 선택 버튼을 텍스트 버튼에서 아이콘 버튼으로 변경한다.

대상 도구:

- 펜
- 지우개
- 직선
- 화살표
- 사각형
- 원/타원

각 버튼은 `aria-label`과 `title`을 유지해 접근성과 툴팁을 보존한다.

### 색상 선택

색상 선택은 현재 색상 네모 버튼 하나로 축약한다.

- 버튼 클릭 시 색상 선택 popover 표시
- 검정 / 빨강 / 파랑 / 초록 유지
- 지우개 선택 시 색상 선택 비활성화

### 굵기 선택

굵기 선택은 별도 버튼 목록에서 dropdown 방식으로 변경한다.

- 굵기 버튼 클릭 시 얇게 / 기본 / 굵게 / 넓게 선택지 표시
- 선택지는 실제 선 두께를 시각적으로 보여주는 preview bar를 포함한다.

### 실선/점선

실선/점선은 두 개 버튼이 아니라 하나의 toggle 버튼으로 변경한다.

- 현재 실선이면 실선 아이콘 표시
- 클릭 시 점선으로 변경
- 현재 점선이면 점선 아이콘 표시
- 클릭 시 실선으로 변경
- 펜/지우개 도구 선택 시 비활성화
- 직선/화살표/사각형/원 도구에서만 활성화

### 기록 버튼

- 되돌리기: 왼쪽 방향 undo 아이콘
- 다시 실행: 오른쪽 방향 redo 아이콘
- 실행할 수 없는 상태에서는 비활성화

### 저장/초기화

- 전체 지우기와 디자인으로 저장 버튼에 아이콘을 추가한다.
- 빈 캔버스에서는 기존처럼 저장 버튼이 비활성화된다.

## 테스트 항목

- 펜 선택 시 실선/점선 toggle 비활성화 확인
- 지우개 선택 시 색상 선택과 실선/점선 toggle 비활성화 확인
- 직선/화살표/사각형/원 선택 시 색상, 굵기, 실선/점선이 적용되는지 확인
- 굵기 dropdown 선택 후 펜과 도형에 반영되는지 확인
- 색상 popover 선택 후 펜과 도형에 반영되는지 확인
- undo/redo 기존 동작 유지 확인
- 전체 지우기 후 undo로 복원되는지 확인
- PNG 저장 후 기존 디자인 첨부 목록에 추가되는지 확인

## 다음 후보

0.12.29 이후에는 native canvas에서 다음 항목을 검토할 수 있다.

- 닫기 전 미저장 그림 확인 모달
- 저장 전 미리보기
- 캔버스 배경 가이드라인
- 도형 드래그 중 preview 색상/투명도 보정
- 모바일 전용 툴바 축약
