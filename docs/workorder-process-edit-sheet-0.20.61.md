# 0.20.61 작업지시서 공정 편집 시트 전환

## Scope

- 작업지시서 공정 탭의 제작/추가 공정 입력을 상시 노출 필드에서 목록 카드 + 편집 모달 구조로 전환했습니다.
- 제작 공정과 추가 공정은 같은 `WorkOrderProcessEditSheet`에서 추가/수정합니다.
- 모바일/태블릿/PC 모두 같은 편집 흐름을 사용합니다.

## Policy

- 공정 탭 본문은 공정 목록 확인에 집중합니다.
- 공정 추가/수정 입력은 별도 모달에서 처리합니다.
- 모바일/태블릿 모달 입력 필드는 `pbp-mobile-no-zoom` 범위를 사용하고 `enterKeyHint="done"`을 적용합니다.
- 기존 저장/API/DB/R2/권한/상태전환 흐름은 변경하지 않았습니다.
