# 0.18.21 작업지시서 모바일/태블릿 생산 공장 선택 필드 보정

## 목적

작업지시서 PC 화면에서는 생산 공장을 선택할 수 있지만 모바일/태블릿 생산 공장 카드에서는 선택 필드가 보이지 않는 불일치를 보정한다.

## 반영 내용

- 모바일 생산 공장 카드에서 납기 입력 필드를 제거하고 생산 공장 선택 EditableValue를 노출했다.
- 태블릿 생산 공장 카드에서 납기 입력 필드를 제거하고 생산 공장 선택 EditableValue를 노출했다.
- 선택 옵션은 기존 PC와 동일하게 factoryOptions를 사용한다.
- 공장 등록, 협력업체관리, DB/API/R2 흐름은 변경하지 않았다.

## 후속 확인

- 모바일 작업지시서 상세에서 생산 공장 선택 필드가 보이는지 확인한다.
- 태블릿 작업지시서 상세에서 생산 공장 선택 필드가 보이는지 확인한다.
- 선택 후 저장 흐름이 PC와 동일하게 동작하는지 확인한다.
