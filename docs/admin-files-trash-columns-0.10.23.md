# 0.10.23 저장소 휴지통 테이블 컬럼 보정

## 목적

저장소 관리 화면의 휴지통 테이블에서 긴 작업지시서명, 긴 파일명, 긴 유형명이 한 행 안에서 과하게 늘어지거나 표시 균형을 깨는 문제를 줄인다.

## 변경 기준

- 대상 컬럼은 상대적으로 넓게 유지한다.
- 작업지시서 컬럼은 긴 제목을 말줄임 처리한다.
- 유형 컬럼은 좁은 배지 형태로 표시한다.
- 크기 컬럼은 더 좁게 유지한다.
- 휴지통 테이블에만 행/헤더 여백 조정을 적용한다.
- 복원, 선택 삭제, 비우기, 새로고침, R2 purge, 감사 로그 흐름은 변경하지 않는다.

## 확인 항목

- 긴 파일명이 한 줄에서 말줄임 처리되는지 확인한다.
- 긴 작업지시서명이 작업지시서 컬럼을 밀어내지 않는지 확인한다.
- 유형명이 길어도 배지 안에서 말줄임 처리되는지 확인한다.
- 휴지통 목록 행 클릭, 복원, 선택 삭제, 비우기 기능이 기존처럼 동작하는지 확인한다.
