# 0.10.3 고객관리자 운영 대시보드 상단 영역 축소

## 목적

고객관리자 `/admin` 첫 화면에서 업무 요약 영역이 너무 크게 차지해 주요 메뉴 카드까지 스크롤해야 하는 문제를 줄인다.

## 반영 내용

- 운영 대시보드 내부의 설명 문구와 기간 선택 버튼을 제거했다.
- 상태 흐름, 상태 분포 카드를 고객관리자 첫 화면에서 제거했다.
- 좌측 업무 목록을 검토·발주 대기 중심으로 축소했다.
- 우측 요약 카드를 주요 대기 현황으로 정리했다.
- 좌우 카드 높이를 맞추고, 좌측 목록은 카드 내부 스크롤로 처리했다.
- 상단 작업지시서 버튼을 제거하고, 작업지시서 업무 화면 진입은 주요 메뉴 카드로 통합했다.
- 개인 설정 진입은 텍스트 버튼 대신 아이콘 버튼으로 정리했다.

## 데이터 기준

- 검토 대기: `review_requested`
- 발주 대기: `review_completed`
- 검수 대기: `inspection`
- 입고 지연: `inspection` 상태이면서 납기일 기준 24시간이 지난 항목

## 보류

- 고객 화면 문구 전체 다듬기
- 기준정보 카드의 실제 라우트 연결
- 멤버관리 실제 화면 연결
- 루트 단일 AppShell 구조

## 확인 항목

- `/admin` 첫 화면에서 주요 메뉴 카드가 이전보다 빨리 보이는지 확인한다.
- 검토·발주 대기 목록이 내부 스크롤로 동작하는지 확인한다.
- 작업지시서 상단 버튼이 사라지고 주요 메뉴 카드로만 진입하는지 확인한다.
- `/me/settings` 아이콘 버튼이 정상 이동하는지 확인한다.
