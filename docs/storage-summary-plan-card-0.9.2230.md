# 0.9.2230 저장소 관리 상단 요약 보정

## 대상 화면

- `/admin/files`

## 변경 목표

저장소 관리 화면의 상단 요약을 통계정보 화면과 비슷한 카드 밀도로 정리하되, 저장소 관리 화면의 목적에 맞게 요금제/용량/파일 운영 상태/파일 유형을 분리한다.

## 변경 내용

### 1. Hydration 오류 수정

기존 `AdminTable`은 `onRowClick`이 있으면 row를 `<button>`으로 렌더링했다. 휴지통 row 내부에도 선택 체크 버튼이 있어 `<button>` 안에 `<button>`이 들어가는 HTML 구조가 발생했다.

수정 후 row는 `div role="button"`으로 렌더링하고, Enter/Space 키로도 row 클릭을 실행한다.

### 2. 좌측 카드: 요금제와 저장소 사용량

좌측 카드는 다음 정보를 담당한다.

- 현재 요금제 표시: Basic
- 업그레이드 버튼
- 저장소 원통형 사용량 시각화
- 현재 사용량 / 전체 한도
- GB 환산값
- 남은 용량
- 사용률 progress bar

업그레이드 버튼은 현재 화면 이동 없이 후속 버전에서 요금제 업그레이드 화면으로 연결할 수 있게 UI만 둔다.

### 3. 가운데 카드: 파일 운영 요약

가운데 카드는 파일 운영 상태를 요약한다.

- 전체 첨부파일
- 사용중 파일
- 휴지통
- 대용량 파일
- 정리 필요

대용량 파일은 아직 실제 계산 기준이 연결되지 않았으므로 `검토 예정`으로 표시한다.

### 4. 우측 카드: 파일 유형 비율

파일 유형 도넛은 통계정보 화면의 도넛과 유사한 정사각형 비율, 중앙 총합, 개수/비율 범례 구조로 정리했다.
