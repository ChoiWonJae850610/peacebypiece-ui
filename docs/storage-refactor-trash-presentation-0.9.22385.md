# 0.9.22385 저장소 휴지통 화면 표현 컴포넌트 리팩토링

## 목적

고객관리자 저장소 휴지통 화면의 시각 표현 로직을 `FileTrashSection.tsx`에서 분리한다.

## 반영 내용

- 휴지통 파일/작업지시서 시각 배지 렌더링을 `fileTrashSectionPresentation.tsx`로 분리했다.
- 작업지시서 삭제 당시 단계 표시 컴포넌트를 같은 presentation 파일로 분리했다.
- 휴지통 상단 액션 버튼의 반복 className 조합을 공통 함수로 정리했다.
- 휴지통 row 생성 로직은 기존 `fileTrashSectionRows.ts` 구조를 유지했다.
- 삭제/복원/비우기 동작과 R2 Worker 기반 흐름은 변경하지 않았다.

## 검증 항목

1. `/admin/files` 휴지통 목록 표시
2. 작업지시서/첨부파일 시각 배지 표시
3. 이미지 preview 실패 시 fallback 표시
4. 작업지시서 상세 모달의 단계 표시
5. 복원 / 선택 삭제 / 비우기 버튼 표시와 활성화 상태
6. 기존 삭제/복원 동작 회귀 확인
