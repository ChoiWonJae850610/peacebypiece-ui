# 0.11.20 관리자 히스토리/점검 공통 UI 적용

## 목적
관리자 히스토리 화면과 관리자 점검 요약 영역의 버튼, 상태 라벨, empty state를 공통 관리자 UI 컴포넌트 기준으로 정리한다.

## 반영 내용
- 관리자 히스토리 필터 버튼과 새로고침 버튼을 `AdminButton`으로 전환했다.
- 히스토리 목록 empty state를 `AdminEmptyState`로 전환했다.
- 히스토리 액션 라벨, 상세 토글 라벨, 전이 상태 라벨을 `AdminStatusBadge`로 전환했다.
- 관리자 점검 요약의 상태/집계 라벨을 `AdminStatusBadge`로 전환했다.
- 관리자 점검 펼치기/닫기 버튼을 `AdminButton`으로 전환했다.

## 제외 범위
- 히스토리 데이터 조회 로직 변경 없음
- 히스토리 필터/검색/날짜/사용자 조건 변경 없음
- 관리자 점검 패널의 내부 점검 로직 변경 없음
- DB schema 변경 없음
