# 0.20.54 모바일·태블릿 모달 포커스 정책 보정

## 목적

모바일·태블릿에서 분류, 담당자, 재고 모달을 반복적으로 열고 입력한 뒤 다른 필드나 닫기/적용 버튼을 누를 때 포커스가 메모 필드로 강제 이동하거나 버튼 터치가 먹히지 않는 문제를 줄인다.

## 변경 원칙

- 모바일·태블릿 coarse pointer 환경에서는 공통 모달의 자동 focus 이동을 하지 않는다.
- 모바일·태블릿에서는 Tab focus trap과 이전 focus 복원을 하지 않는다.
- 모달 하단 적용 버튼 터치 시 먼저 active element를 blur한다.
- 재고 모달 숫자 입력 완료 후 메모 textarea로 자동 focus가 이동하지 않도록 memo focus guard를 둔다.
- 분류/담당자 선택 버튼은 터치 전 active element를 정리한다.

## 영향 제한

- PC 키보드 접근성용 focus trap과 이전 focus 복원 흐름은 유지한다.
- 상태전환, 권한, API, DB, R2, 첨부, 메모, 휴지통, purge 흐름은 변경하지 않는다.
