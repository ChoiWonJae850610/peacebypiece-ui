# 0.20.12 환경설정 회사 정보 탭 UI 재정리

## 목적

0.20.11에서 환경설정 상단 구조를 탭형 내비게이션으로 줄인 뒤, 0.20.12에서는 회사 정보 탭의 내부 밀도를 낮춘다. 기존 화면은 회사 정보, 요청 흐름, 운영 기준, 회사 파일 안내가 모두 카드형으로 노출되어 실제 운영 화면보다 설명 문서처럼 보이는 문제가 있었다.

## 변경 기준

- 회사 기본 정보는 카드 나열이 아니라 필드형 정보표로 표시한다.
- 회사 정보 변경 요청과 탈퇴 요청은 상단 액션 버튼으로 제공한다.
- 개인설정 이동 카드는 제거한다. 개인설정은 우측 상단 사람 아이콘 진입을 기본으로 본다.
- 대표 이미지와 사업자등록증은 설명 카드가 아니라 현재 등록 상태와 미리보기 영역을 먼저 보여준다.
- 파일이 없으면 등록 버튼을, 파일이 있으면 변경 버튼을 표시한다.
- 사업자등록증 반려 사유가 있으면 파일 카드 내부에 바로 표시한다.
- 요청 이력은 하단에 남겨 접수 상태와 처리 결과를 확인한다.
- 조직 정보와 개인 설정 분리 같은 운영 기준 문구는 고객 화면에서 제거하고 문서 기준으로만 유지한다.

## 회사 정보 탭 구조

```txt
회사 계정 정보

[회사 기본 정보]
- 회사명
- 사업자등록번호
- 회사 주소
- 대표 로그인 이메일
- 계정 상태
- 이용 상태
[회사 정보 변경 요청] [탈퇴 요청]

[대표 이미지·사업자등록증]
- 대표 이미지 현재 상태 / 미리보기 / 등록·변경
- 사업자등록증 현재 상태 / 미리보기 / 등록·변경 / 검토 상태

[요청 작성]
- 요청 버튼을 눌렀을 때만 표시

[요청 이력과 처리 결과]
- 최근 요청 5건
```
