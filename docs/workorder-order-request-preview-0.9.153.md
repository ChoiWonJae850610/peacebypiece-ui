# 0.9.153 작업지시서 발주요청 확인 영역 정렬

## 목적

발주요청 확인 모달의 대표 이미지 영역과 요청사항 영역이 서로 다른 높이와 구조로 보이던 문제를 정리한다.

## 반영 내용

- 대표 이미지 카드와 요청사항 카드를 같은 grid 안의 독립 카드로 구성했다.
- 두 카드의 header 높이, border, padding, 최소 높이를 맞췄다.
- 요청사항 상단에 있던 `대표이미지는 디자인 영역 파일을 우선 사용합니다.` 안내 문구를 제거했다.
- 대표 이미지가 없는 경우에도 요청사항 카드와 같은 높이 기준이 유지되도록 했다.

## 변경하지 않은 것

- 발주요청 상태 변경 로직은 변경하지 않는다.
- PDF 출력 HTML 생성 로직은 변경하지 않는다.
- 첨부/메모 저장, R2 Worker, purge 기능은 변경하지 않는다.
- DB schema, package.json, package-lock.json은 변경하지 않는다.

## 테스트 기준

1. 발주요청 확인 모달을 연다.
2. 대표 이미지 카드와 요청사항 카드의 높이와 상단 header 높이가 맞는지 확인한다.
3. 요청사항 입력 textarea가 대표 이미지 영역과 같은 세로 기준을 갖는지 확인한다.
4. PDF 출력 버튼과 발주요청 진행 버튼이 기존처럼 동작하는지 확인한다.
