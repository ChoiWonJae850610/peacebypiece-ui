# 0.20.13 환경설정 요금제·저장공간 탭 UI 재정리

## 목적

요금제·저장공간 화면에서 설명 카드와 운영 기준 노출을 줄이고, 고객사 관리자가 실제로 확인해야 하는 현재 요금제, 요금제 종료일, 저장공간, 멤버 수, 변경·결제 액션을 우선 표시한다.

## 변경 기준

- 상단에는 현재 요금제와 구독 상태만 badge로 표시한다.
- 본문은 `현재 요금제`, `저장공간 사용량`, `멤버 사용량` 3개 핵심 상태만 큰 요약으로 표시한다.
- `무료체험 종료일` 표현은 `요금제 종료일`로 정리한다.
- 최근 갱신, 데이터 기준 등 보조 정보는 작은 필드형 목록으로 낮춘다.
- 요청 흐름/운영 기준 카드형 설명은 제거하고, 변경·결제 액션 목록으로 정리한다.
- 실제 결제창 연결 전에는 운영팀 검토 흐름으로 접수된다는 짧은 안내만 남긴다.

## 후속 작업

- 0.20.14 약관·정책 탭을 문서 목록 + 모달 열람 구조로 재정리한다.
- 0.20.15 기준정보 탭을 내부 탭 + 목록/테이블 관리 구조로 재정리한다.
