# 0.10.22 저장소 관리 휴지통 표시량 보정

## 목표

저장소 관리 화면에서 상단 현황 카드가 차지하는 세로 공간을 줄이고, 휴지통 목록이 PC 화면에서 최소 5줄 이상 보이도록 보정한다.

## 변경

- 저장소 현황 카드 3개의 최소 높이와 내부 여백을 축소했다.
- 요금제 용량 그래프와 파일 유형 도넛 차트를 소폭 축소했다.
- 휴지통 카드 padding과 목록 상단 간격을 줄였다.
- 공통 AdminTable에 `headerClassName`, `rowBaseClassName` 선택 prop을 추가했다.
- 휴지통 테이블에서만 헤더/행 padding을 소폭 줄였다.
- 기존 복원, 선택 삭제, 비우기, 새로고침, R2 purge, 감사 로그 흐름은 변경하지 않았다.

## 검증 기준

- `/admin/files`에서 저장소 현황 카드 3개가 더 낮게 표시된다.
- 휴지통 목록이 이전보다 더 많이 보인다.
- PC 기준 휴지통 목록 5줄 이상 표시를 목표로 한다.
- 다른 AdminTable 사용 화면은 기본 row/header class를 유지한다.
