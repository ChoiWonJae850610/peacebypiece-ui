# 0.18.42 협력업체관리 상단 영역 정리

## 목적

협력업체관리 목록은 0.18.41에서 컨테이너 폭 기준 wide table / compact card 구조로 정리되었으나, 상단 요약 카드와 검색/유형/상태 필터 영역은 여전히 큰 카드 나열과 분리된 필터 블록처럼 보여 목록 화면의 위계가 어색했다.

이번 버전은 기능 흐름을 변경하지 않고 상단 영역의 정보 밀도와 반응형 흐름을 정리한다.

## 반영 내용

- 협력업체 요약 카드를 compact metric card 형태로 축소
- 요약 카드는 실제 화면 폭에 따라 2열, 3열, 5열 흐름으로 자연스럽게 재배치
- 검색/유형/상태/현재 목록 개수를 하나의 toolbar 흐름으로 정리
- 넓은 화면에서는 검색창, 유형 필터, 상태 필터, 현재 목록 count가 한 줄에 가깝게 정렬
- 좁은 화면에서는 검색, 유형, 상태, count가 세로 흐름으로 자연스럽게 쌓임
- 기존 검색/필터/정렬/등록/수정 흐름은 변경하지 않음
- WorkspaceShell 스크롤 구조는 변경하지 않음
- DB/API/협력업체 저장 흐름은 변경하지 않음

## 테스트 포인트

1. PC 협력업체관리
   - 상단 요약 카드가 과하게 크지 않은지 확인
   - 검색/유형/상태/현재 목록 count가 한 toolbar처럼 보이는지 확인
   - + 업체 추가 버튼 위치와 위계가 기존처럼 유지되는지 확인

2. 태블릿 가로/세로
   - 요약 카드가 화면 폭에 맞게 2~3열로 자연스럽게 재배치되는지 확인
   - 필터 버튼들이 과하게 벌어지거나 잘리지 않는지 확인

3. 모바일/좁은 폭
   - 검색창, 유형 필터, 상태 필터, 현재 목록 count가 세로로 읽히는지 확인

4. 기존 기능
   - 검색/유형 필터/상태 필터 동작 확인
   - 업체 등록/수정 모달 진입 확인
   - 0.18.41 목록 table/card 반응형 유지 확인
