# 0.9.2232 — 저장소 휴지통 목록/상세 모달 정보 구조 정리

## 목적

저장소 관리의 휴지통 목록은 간결하게 유지하고, 삭제된 작업지시서의 단계와 연결 정보는 상세 모달에서 확인하도록 정리한다.

## 변경 사항

- 작업지시서 단위 휴지통 항목의 작업지시서 컬럼은 `-`로 표시한다.
- 휴지통 목록의 대상명/작업지시서명 폰트 굵기를 낮춰 테이블 가독성을 높인다.
- 상세 모달의 `복구정책` 필드를 제거한다.
- 작업지시서 상세 모달에는 삭제 당시 진행 단계를 축약형 진행 단계 막대로 표시한다.
- 복구 범위는 하단 안내 문구로만 설명한다.

## 후속

- 저장소 썸네일/파일 아이콘은 별도 버전에서 적용한다.
- 휴지통 상세 모달의 실제 복구/영구삭제 API 연결 상태는 기존 액션 흐름을 유지한다.
