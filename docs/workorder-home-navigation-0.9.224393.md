# 작업지시서 홈 버튼과 공통 메인화면 1차 구현 — 0.9.224393

## 목적

작업지시서 화면을 좌측 패널 없는 업무 집중형 화면으로 유지하되, 사용자가 언제든 자기 역할에 맞는 메인화면으로 돌아갈 수 있는 홈 진입점을 추가한다.

## 이번 버전 반영 범위

1. 작업지시서 화면 홈 버튼 추가
   - PC/태블릿: 작업지시서 상세 영역 상단에 홈 버튼 표시
   - 모바일: 상단 바에 홈 아이콘 버튼 표시

2. 역할 기반 홈 경로 유틸 추가
   - 관리자 역할: `/admin`
   - 일반 멤버 역할: `/workspace`
   - 시스템관리자 경로는 후속 인증/권한 체계에서 확장

3. 일반 멤버용 `/workspace` 1차 화면 추가
   - 작업지시서 업무 화면 카드
   - 개인 환경설정 후보 카드
   - 실제 권한 DB/멤버 초대/개인 설정 저장은 후속 버전으로 분리

## 구현 판단

이번 버전에서는 권한 DB를 만들지 않는다. 역할과 권한 체계가 아직 확정되지 않은 상태에서 `roles`, `permissions`, `role_permissions`, `user_roles`를 먼저 만들면 작업지시서 안정화 범위를 크게 흔들 수 있다.

따라서 홈 이동 구조만 먼저 고정하고, 권한 기반 카드 필터링은 후속 버전에서 붙일 수 있도록 경로 유틸과 `/workspace` 진입점만 추가한다.

## 후속 작업

- `/workspace` 카드 registry를 권한 코드 기준으로 확장
- `/me/settings` 또는 개인 설정 화면 설계
- 시스템관리자와 고객관리자, 일반 멤버의 홈 경로 판정 기준 분리
- 실제 로그인/초대/권한 DB 도입 후 route guard와 카드 노출 필터 연결
