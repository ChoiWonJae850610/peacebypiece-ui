# 0.9.22428 관리자 통계 기간 요약 카드

## 목표

관리자 통계 화면의 기간별 분석 카드가 기능 대비 가로 폭을 과하게 차지하는 문제를 줄이고, 우측 빈 공간에 선택 기간 기준의 핵심 숫자 요약을 배치한다.

## 변경 내용

- 기간별 분석 영역을 PC에서 좌우 2-column 구조로 정리했다.
- 좌측에는 기존 날짜 선택 popover와 적용/초기화 버튼을 유지했다.
- 우측에는 기간 요약 카드를 추가했다.
- 기간 요약 항목은 그래프가 아니라 목록+숫자 형태로 표시한다.

## 기간 요약 항목

- 완료 작업지시서: 선택 기간 안에 완료된 작업지시서 수
- 리오더: 선택 기간 안의 리오더 작업지시서 수
- 불량 작업지시서: 불량이 1개 이상 기록된 작업지시서 수

## 정책

- 날짜 입력은 키보드 입력이 아니라 DayPicker 클릭 기준만 유지한다.
- PC는 좌우 2-column 레이아웃을 사용한다.
- 모바일/좁은 화면은 세로 스택으로 표시한다.
- DB schema 변경은 없다.
