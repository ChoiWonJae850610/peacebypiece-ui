# 0.10.26 — 통계 화면 섹션 탭 구조 정리

## 목표
고객관리자 통계 화면에서 세로로 길게 이어지던 통계 카드 묶음을 탭 구조로 정리한다.

## 반영 내용
- 상단 누적 카드 4개는 유지한다.
- 하단 통계 영역을 섹션 탭으로 전환한다.
  - 생산 구성
  - 업체 성과
  - 기간 분석
- 탭 전환 시 카드 영역에 가벼운 좌우 슬라이드/페이드 전환을 적용한다.
- 기존 통계 계산 로직, repository, API는 변경하지 않는다.
- 차트 라이브러리 도입은 후속 버전에서 별도 검토한다.

## 제외
- DB schema 변경 없음
- 통계 계산 로직 변경 없음
- 감사 로그 흐름 변경 없음
- 저장소/휴지통/R2 purge 흐름 변경 없음
