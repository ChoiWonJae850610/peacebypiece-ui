# 작업지시서 직접 그리기 tablet 사용성 수습 (0.12.35)

## 목표

0.12.32~0.12.34에서 추가한 확대/축소와 손바닥 이동 기능을 제거하고, native canvas 직접 그리기를 tablet 기준의 안정적인 100% 입력 도구로 되돌린다.

## 반영 내용

- 확대/축소 버튼 제거
- 손바닥 이동 도구 제거
- viewport scale / pan 상태 제거
- zoom/pan stage transform 제거
- canvas 표시 높이 복구
- 펜 / 지우개 / 직선 / 화살표 / 사각형 / 원·타원 입력 안정성 우선
- 기존 색상 / 굵기 / 실선·점선 / undo / redo / 전체 지우기 / 저장 흐름 유지
- backdrop click 닫기 방지 유지
- 브라우저 뒤로가기 방지 유지
- PNG 저장 및 기존 디자인 첨부 업로드 흐름 유지

## 제외한 기능

- 확대/축소
- 손바닥 이동
- 이미지 위에 그리기
- 기본 의류 템플릿 배경
- 텍스트 삽입

## 다음 확인 항목

1. 직접 그리기 모달 최초 진입 시 캔버스가 정상 높이로 보이는지 확인
2. 하단 툴바와 저장/전체 지우기 버튼이 스크롤 없이 보이는지 확인
3. tablet 펜/손가락 입력이 정확히 반영되는지 확인
4. 지우개 크기 변경이 실제 삭제 영역과 cursor 원형에 반영되는지 확인
5. 직선/화살표/사각형/원·타원 입력이 정상인지 확인
6. 저장 후 PNG가 기존 디자인 첨부 목록에 표시되는지 확인
