# 0.9.22412 저장소관리 영어 locale fallback 보정

## 배경

0.9.22411 적용 후 브라우저/저장 설정이 영어 locale로 잡힌 경우 `/admin/files` 저장소관리 화면의 상단 제목, 좌측 메뉴, 휴지통 제목 등이 영어로 표시되었다.

현재 저장소관리 화면은 한국어 운영 화면 기준으로 안정화 중이며, 영어 locale은 아직 완성된 사용자 노출 언어가 아니다. 따라서 영어 locale이 선택되더라도 저장소관리와 관리자 기본 내비게이션은 한국어 기준 문구가 표시되도록 보정한다.

## 변경 내용

- `en/admin.ts`의 `filesPage` 문구를 한국어 기준으로 보정
- `en/admin.ts`의 관리자 내비게이션 문구를 한국어 기준으로 보정
- `en/admin.ts`의 `trashPage.title`을 `휴지통`으로 보정
- `en/admin.ts`의 저장소 topbar summary를 한국어 기준으로 보정

## 범위

이번 버전은 화면 문구 보정만 수행한다.

삭제일시 시간대 차이 문제는 별도 버전에서 다룬다.

## 확인 항목

1. `/admin/files` 진입
2. 좌측 메뉴가 한국어로 표시되는지 확인
3. 상단 제목이 `저장소 관리`로 표시되는지 확인
4. 상단 요약 문구가 `문서/디자인 · 휴지통 · 용량 관리`로 표시되는지 확인
5. 휴지통 영역 문구가 한국어 기준으로 표시되는지 확인
6. 삭제일시 시간대 문제는 다음 버전에서 별도 확인
