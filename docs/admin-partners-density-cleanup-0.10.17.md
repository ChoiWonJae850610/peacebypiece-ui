# 0.10.17 고객관리자 협력업체 관리 화면 밀도 정리

## 목표

`/admin/partners` 화면의 중복 요약과 안내 문구를 줄이고, 협력업체 목록을 화면 안에서 관리하기 쉬운 내부 스크롤 구조로 정리한다.

## 반영 내용

- 상단 요약 카드 5개는 유지한다.
- 요약 카드와 중복되는 `업체 유형 분포` 카드는 제거한다.
- 반복 노출되는 파란색 도움말 안내 카드는 제거한다.
- 본문 상단의 중복 제목/설명은 제거하고 `+ 업체추가` 액션만 유지한다.
- 협력업체 목록은 남은 영역을 채우고 목록 내부에서 스크롤되도록 정리한다.

## 제외 범위

- 협력업체 DB schema 변경 없음.
- 협력업체 저장/수정/필터 로직 변경 없음.
- 감사 로그 쓰기 지점 변경 없음.
