# 0.20.60 작업지시서 구성 탭 편집 시트 1차

## Scope

- 구성 탭의 원단·부자재 입력을 상시 노출 인라인 입력에서 목록 카드 + 편집 모달 흐름으로 전환했습니다.
- 모바일/태블릿/PC 모두 자재 목록에서는 구분, 항목명, 수량, 단위만 요약 표시합니다.
- 자재 추가/수정은 별도 모달에서 구분, 항목명, 수량, 단위를 입력한 뒤 적용합니다.
- 기존 저장/API/DB/R2/권한/상태전환 흐름은 변경하지 않았습니다.

## Policy

- 구성 탭은 보기 중심 화면으로 유지합니다.
- 자재 입력 필드는 필요할 때만 편집 모달에서 표시합니다.
- 원단·부자재 발주처, 단가, 배송 정보는 기존처럼 원단·부자재 발주 화면에서 다룹니다.
