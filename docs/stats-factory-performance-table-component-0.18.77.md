# 0.18.77 통계정보 업체 성과 테이블 컴포넌트 분리

## 변경 목표

통계정보 분석 카드 파일에 남아 있던 업체 성과 테이블 렌더링 책임을 별도 컴포넌트로 분리했습니다.

## 변경 내용

- `AdminStatsFactoryPerformanceTable` 추가
- 업체 성과 테이블의 공통 responsive table shell 사용 구조 유지
- `AdminStatsAnalysisCards`는 분석 카드 shell, 기간 요약, 막대형 카드 중심으로 축소
- 통계 계산, 기간 선택, 탭 전환, DB/API 흐름 변경 없음

## 확인 항목

- 통계정보 > 업체 성과 탭의 업체명/납기 지연율/검수·불량률 row 표시 유지
- 생산 구성/업체 성과/기간 분석 탭 전환 유지
- 기간 선택/적용 흐름 유지
