# 0.9.2235 저장소 preview 실패 fallback 및 목록 시각 밀도 보정

## 목적

저장소 관리 휴지통 목록에서 R2 preview 이미지가 실패해도 목록이 깨지지 않도록 fallback을 적용하고, 휴지통 목록과 상세 모달의 글씨 굵기와 시각 밀도를 낮춘다.

## 변경 내용

- 이미지 preview 로딩 실패 시 `IMG` 배지로 되돌아가도록 처리
- 목록 visual 배지 크기를 줄여 row 높이 부담 완화
- 대상/작업지시서/용량/유형 텍스트를 13px 중심으로 조정
- 상세 모달 정보 카드 값의 글씨 굵기를 낮춰 압박감 완화
- 파일 미리보기 링크의 글씨 굵기를 낮춰 보조 액션처럼 보이도록 정리

## 확인 방법

1. `/admin/files` 접속
2. 이미지 파일 row에서 preview가 보이는지 확인
3. preview URL이 실패하는 경우 `IMG` 배지로 fallback되는지 확인
4. 휴지통 row 높이와 글씨 굵기가 이전보다 차분해졌는지 확인
5. 상세 모달 정보 카드 값이 너무 진하게 보이지 않는지 확인
