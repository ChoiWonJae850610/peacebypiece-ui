# 0.17.20 PC 생산 구성 2컬럼 레이아웃 정리

## 목표

작업지시서 상세의 생산 구성 영역에서 필요 자재와 외주공정을 PC 기준 좌우 2컬럼으로 배치한다.

## 적용 범위

- PC 작업지시서 상세 화면
- 생산 구성 내부의 필요 원단/부자재 섹션
- 생산 구성 내부의 외주공정 섹션

## 유지 기준

- 생산 공장 영역은 이번 변경 대상이 아니다.
- 태블릿/모바일 전용 섹션은 기존 세로 흐름을 유지한다.
- 작업지시서 PDF 양식은 변경하지 않는다.
- DB schema는 변경하지 않는다.
- 원단·부자재/외주공정의 입력 데이터 구조는 변경하지 않는다.

## 레이아웃 기준

PC에서는 생산 구성 내부를 다음처럼 정리한다.

```text
[필요 원단/부자재] [외주공정]
```

각 섹션은 PC 기준으로 최대 높이를 제한하고, 항목이 늘어나면 해당 섹션 내부에서 스크롤되도록 한다.

## 후속 검토

- 실제 PC 화면에서 2컬럼 폭이 충분한지 확인한다.
- 항목이 많은 경우 내부 스크롤이 과도하지 않은지 확인한다.
- 태블릿/모바일은 기능 흐름이 안정된 뒤 별도 반응형 보정 단계에서 재검토한다.
