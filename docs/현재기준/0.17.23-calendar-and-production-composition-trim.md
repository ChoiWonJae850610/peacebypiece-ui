# 0.17.23 달력 팝업 및 생산 구성 압축

## 기준
- 기준 버전: 0.17.22
- 결과 버전: 0.17.23

## 변경 내용
- 공통 단일 날짜 선택 팝업의 폭을 줄여 우측 빈 여백을 줄였다.
- 날짜 선택 입력 필드의 compact 스타일은 유지했다.
- PC 작업지시서 생산 구성 영역에서 외주공정 블록을 제거하고 필요 원단/부자재만 남겼다.
- 필요 원단/부자재 테이블은 PC 중앙 패널 폭을 더 넓게 사용하도록 컬럼 비율을 조정했다.

## 보류한 내용
- 외주 발주를 발주 정보 영역으로 이동하는 구조는 이번 버전에서 DB/UI 흐름을 바꾸지 않고 설계 후보로만 둔다.
- 외주처/공정 마스터 선택과 생산 발주 수량 자동 복사는 다음 단계에서 별도 설계한다.

## 비변경
- 작업지시서 PDF 양식 변경 없음
- DB schema 변경 없음
- R2/첨부/메모/휴지통/purge 흐름 변경 없음
