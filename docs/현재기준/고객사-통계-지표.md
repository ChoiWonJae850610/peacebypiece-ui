Version: 0.16.49
Status: metric skeleton
Scope: 고객관리자 통계 항목 확장
Non-goal: 실제 DB 통계 쿼리, 차트 UI, AI 분류 통계

## 1. 이번 버전의 목적

0.9.69에서 만든 공통 stats 구조 위에 고객관리자 통계 항목을 보강한다.
화면에서 직접 통계를 계산하지 않고 repository/selector가 만든 metric summary만 표시하는 구조를 유지한다.

## 2. 고객관리자 통계 항목

### 월별 작업지시서 수

- 이번 달 작업지시서
- 지난 달 작업지시서
- 기간 필터 기준 작업지시서

### 상태별 작업지시서 수

- 작성중
- 검토요청
- 검토완료
- 생산/검수
- 완료

주의:
- 문자열 직접 비교가 아니라 상태 policy/상수 기준으로 매핑한다.

### 디자이너별 작업량

- 담당자별 작업지시서 수
- 미지정 작업지시서 수

### 공장별 생산량

- 공장별 생산 진행 수량
- 공장 미지정 항목

### 카테고리별 생산량

- 카테고리별 생산량
- 미분류 항목

### 첨부파일 / 용량 사용량

- 첨부파일 수
- 저장공간 사용량
- 저장공간 사용률
