# 0.17.18 작업지시서 러프 생산구성 입력 기준

## 목표

작업지시서 화면은 생산 지시와 러프한 필요 항목 입력에 집중한다. 실제 거래처, 단가, 확정 수량, 금액, 배송/전달 정보는 원단·부자재 발주 화면과 이후 외주공정 화면에서 확정한다.

## 유지하는 것

- 작업지시서 PDF 양식은 유지한다.
- 작업지시서 PDF 안의 원단 내역, 부자재 내역, 외주 내역 테이블 구조는 유지한다.
- 작업지시서 화면의 생산 공장, 생산 수량, 납기, 담당자, 디자인 이미지, 메모 흐름은 유지한다.
- DB schema는 이번 버전에서 변경하지 않는다.

## 이번 버전에서 바꾼 것

### 작업지시서 화면

- 원단/부자재 섹션명을 `필요 원단/부자재`로 변경한다.
- 외주 공정 섹션명을 `예상 외주공정`으로 변경한다.
- 원단/부자재 테이블에서는 러프 입력에 필요한 항목만 노출한다.
  - 구분
  - 필요 항목
  - 대략 수량
  - 대략 단위
- 외주공정 테이블에서는 러프 입력에 필요한 항목만 노출한다.
  - 예상 공정
  - 대략 수량
  - 대략 기준
- 실제 거래처, 외주처, 단가, 금액 컬럼은 작업지시서 화면에서 노출하지 않는다.

### 원단·부자재 발주 화면으로 넘길 책임

- 실제 거래처
- 실제 품목명
- 실제 발주 수량
- 단위
- 단가
- 금액
- 연결 작업지시서
- 배송/전달 정보
- 자재 발주 PDF 생성

### 외주공정 화면으로 넘길 책임

- 실제 외주처
- 공정명
- 단가
- 수량
- 금액
- 납기
- 외주 발주/완료 처리
