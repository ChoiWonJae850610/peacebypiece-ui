# 0.17.22 외주공정 공정명 단순화 및 입력 필드 compact 기준

## 기준

작업지시서 화면의 생산 구성은 실제 발주 상세 입력 화면이 아니라 필요한 구성 항목을 정리하는 영역이다.
따라서 외주공정은 작업지시서 단계에서 수량/단위까지 입력하지 않고 공정명만 등록한다.

## 변경 내용

- 외주공정 입력 테이블을 `공정` 중심으로 단순화한다.
- 외주공정의 수량/단위 컬럼은 작업지시서 PC 화면에서 제거한다.
- 실제 외주처, 단가, 금액, 납기, 완료 처리는 이후 외주공정 화면에서 확정한다.
- 입력/선택/날짜 필드의 기본 시각 크기를 전역 compact 방향으로 조정한다.
- 작업지시서 테이블 안의 편집 필드도 같은 compact 기준을 따른다.

## 유지 사항

- 필요 원단/부자재는 `구분 / 필요 항목 / 수량 / 단위`를 유지한다.
- 기존 외주공정 데이터 구조는 유지한다.
- DB schema는 변경하지 않는다.
- 작업지시서 PDF 양식은 변경하지 않는다.
- 태블릿/모바일 전용 생산 구성 구조는 이번 버전에서 직접 변경하지 않는다.

## 후속 검토

- 전역 compact 필드가 너무 작아 보이는 화면은 후속 버전에서 보정한다.
- 외주공정은 향후 기준정보에 등록된 외주공정 마스터에서 선택하는 구조를 검토한다.
