# 0.17.21 작업지시서 PC 카드 헤더 평탄화

## 목적
작업지시서 상세 PC 화면에서 발주 정보와 생산 구성 카드 내부의 중복 접힘 헤더를 제거해 가운데 패널 높이를 줄인다.

## 적용 기준
- 상위 섹션 제목과 설명은 기존 화면 구조를 유지한다.
- 카드 내부에서 같은 제목과 요약을 반복하지 않는다.
- 발주 정보와 생산 구성은 PC 상세 화면에서 항상 펼쳐진 상태로 표시한다.
- 발주 정보의 검수 상태 배지는 테이블 위의 얇은 상태 줄로 유지한다.
- 생산 구성의 필요 원단/부자재와 외주공정은 0.17.20의 좌우 2컬럼 구조를 유지한다.

## 변경하지 않은 것
- 작업지시서 PDF 양식은 변경하지 않는다.
- DB schema는 변경하지 않는다.
- 태블릿/모바일 전용 섹션은 변경하지 않는다.
- 원단·부자재 발주 화면 변환 기능은 다음 단계에서 다룬다.

## 다음 단계
0.17.22에서 원단·부자재 화면이 작업지시서의 필요 자재 항목을 실제 발주 라인으로 가져오는 흐름을 정리한다.
