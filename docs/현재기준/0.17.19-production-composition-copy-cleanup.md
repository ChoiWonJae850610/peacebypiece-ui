# 0.17.19 생산 구성 문구 정리

## 목적
작업지시서의 생산 구성 영역을 실제 발주 상세 입력 화면처럼 보이지 않도록 문구를 정리한다.

## 적용 기준
- 작업지시서 화면은 필요 자재와 외주공정 항목을 간단히 정리하는 위치다.
- 실제 거래처, 단가, 확정 수량, 금액, 배송/전달 정보는 원단·부자재 발주 화면 또는 외주공정 화면에서 확정한다.
- 작업지시서 PDF 양식과 DB schema는 변경하지 않는다.

## 변경 내용
- 생산 구성 요약에서 금액 합계를 제거한다.
- 요약 문구는 `필요 자재 n개 · 외주공정 n개` 형식으로 표시한다.
- 원단·부자재 컬럼은 `구분 / 필요 항목 / 수량 / 단위` 기준으로 표시한다.
- 외주공정 컬럼은 `공정 / 수량 / 단위` 기준으로 표시한다.
- `러프`, `대략`, `예상` 표현을 작업지시서 생산 구성 UI 문구에서 제거한다.

## 유지 사항
- 작업지시서 PDF 양식은 유지한다.
- 기존 원단/부자재/외주 데이터 구조는 유지한다.
- 원단·부자재 발주 화면과 외주공정 화면에서 확정 상세를 다루는 방향은 유지한다.
