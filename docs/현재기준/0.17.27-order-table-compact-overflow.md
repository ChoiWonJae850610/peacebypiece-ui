# 0.17.27 발주 정보 테이블 compact 보정

## 목적
- 발주 정보 단일 테이블에서 우측 삭제 버튼이 기본 화면에서 보이도록 컬럼 폭을 조정한다.
- 수량/공임비/로스비/금액은 입력값을 제한하지 않고, 표시 폭만 줄여 긴 값은 말줄임 처리한다.
- 발주 정보와 생산 구성 하단의 헬프 메시지를 제거해 중앙 패널 높이를 줄인다.
- 필요 원단/부자재 빈 상태 행 높이를 일반 행 수준으로 낮춘다.

## 범위
- PC 작업지시서 상세의 발주 정보/생산 구성 표시 보정
- DB schema 변경 없음
- 작업지시서 PDF 양식 변경 없음

## 주의
- 숫자 입력 자릿수는 제한하지 않는다.
- 셀 표시 폭만 제한하고, 전체값은 title로 확인 가능하게 둔다.
