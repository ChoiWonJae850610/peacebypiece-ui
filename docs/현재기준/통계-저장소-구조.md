# PeaceByPiece 통계 selector / repository 구조

Version: 0.16.49
Status: 구조 skeleton
Scope: admin stats, system stats, 공통 metric 타입과 repository 구조
Non-goal: 실제 DB 통계 쿼리, 차트 UI, AI 분류 통계

## 1. 원칙

통계 계산은 화면에서 직접 하지 않는다.
화면은 repository 또는 selector가 만든 metric summary만 표시한다.

## 2. 계층

- `lib/stats/statsTypes.ts`: 통계 타입
- `lib/stats/statsSelectors.ts`: count, ratio, series metric 생성
- `lib/stats/statsRepository.ts`: admin/system 통계 repository skeleton
- `lib/stats/api/statsRouteHandlers.ts`: 얇은 API handler
- `app/api/admin/stats/route.ts`: 고객관리자 통계 route
- `app/api/system/stats/route.ts`: 시스템관리자 통계 route
