# 0.11.64 관리자 멤버관리 IA 재구성 1차

## 목적

멤버관리 화면의 중복 타이틀 카드와 긴 스크롤 구조를 줄이고, 요약 카드와 탭 중심으로 멤버 초대/승인/전체 멤버/권한 관리를 분리한다.

## 반영 내용

- 본문 상단의 중복 멤버관리 타이틀 카드 제거
- 조직 설정 보기 버튼 제거
- 전체 멤버, 초대 대기, 승인 대기, 권한 템플릿 요약 카드를 화면 상단으로 이동
- 탭 추가
  - 멤버 초대
  - 승인
  - 전체 멤버
  - 권한 관리
- 각 탭 선택 시 해당 영역만 표시되도록 1차 분리
- 기존 초대 생성, 가입 신청 승인/거절, 멤버 권한 저장 API 호출 흐름은 유지

## 유지한 정책

- join_requests 승인/거절 API 변경 없음
- 권한 저장 API 변경 없음
- 초대 링크 생성 API 변경 없음
- DB schema 변경 없음

## 확인 항목

1. `/admin/members` 진입 시 요약 카드가 최상단에 표시되는지 확인
2. `조직 설정 보기` 버튼이 사라졌는지 확인
3. 멤버 초대 탭에서 초대 링크 생성 UI가 표시되는지 확인
4. 승인 탭에서 가입 신청 승인/거절 영역이 표시되는지 확인
5. 전체 멤버 탭에서 멤버 목록과 역할 기본값이 표시되는지 확인
6. 권한 관리 탭에서 홈 카드 권한, 권한 카탈로그, 권한 매트릭스가 표시되는지 확인
7. 모바일 폭에서 탭 카드가 1열 또는 2열로 자연스럽게 배치되는지 확인
