# 0.9.161 작업지시서 삭제 확인 모달 정리

## 목적

브라우저 기본 `window.confirm`으로 처리하던 작업지시서 삭제 확인을 앱 내부 모달로 교체한다.

## 반영 내용

- 작업지시서 삭제 버튼 클릭 시 바로 삭제하지 않고 확인 모달을 연다.
- 모달에는 삭제 대상 작업지시서명을 표시한다.
- 삭제 후 목록에서 숨겨진다는 안내를 표시한다.
- 첨부파일 전체 휴지통 이동 정책은 이번 버전에서 구현하지 않는다.
- 기존 삭제 API와 repository 삭제 흐름은 변경하지 않는다.

## 다음 단계

- 작업지시서 삭제 시 연결된 디자인/첨부/메모 첨부를 함께 휴지통으로 이동하는 정책을 별도 버전에서 확정한다.
- 고객관리자 저장소의 작업지시서/첨부파일목록/휴지통 탭 구조를 별도 버전에서 설계한다.
