# UI Admin Select Application 0.18.02

## 목적

관리자/시스템 화면의 native select 사용처 중 영향 범위가 작고 상태 변경 로직이 단순한 필터 UI부터 WAFL 공통 `AppSelect`로 전환한다.

## 적용 범위

- 저장소 파일 목록 정렬 select
- 멤버관리 목록 상태 필터 select
- 멤버관리 목록 권한 필터 select
- 작업지시서 변경이력 날짜 필터 select
- 작업지시서 변경이력 사용자 필터 select

## 유지한 범위

- 검색 input 동작
- 저장소 파일 선택/삭제 흐름
- 멤버 승인/권한/역할 저장 흐름
- 이력 필터링 계산 로직
- 관리자/시스템 화면 라우팅 구조
- DB/API/R2/첨부/메모 흐름

## 다음 후보

0.18.03 이후에는 다음 select 사용처를 점검한다.

- 멤버 초대 방식 선택
- 멤버 권한 상세 역할 선택
- 시스템 감사 로그 필터
- 시스템 회사 승인/상태 선택
- 기준정보 관리 화면의 template/category/unit/process select

단, 복잡한 폼 상태와 DB 저장이 직접 연결된 select는 한 번에 대량 전환하지 않고 화면별로 분리해 적용한다.
