# 0.9.2226 — 통계 화면 UI 정리와 날짜 선택 정책 보강

## 목적

고객관리자 통계정보 화면에서 중복 제목, 불필요한 안내 문구, 하단 대시보드형 요약 카드를 제거하고 직접 기간 선택 정책을 명확히 한다.

## 반영 내용

- 본문 내부 `ANALYTICS / 통계정보` 헤더 제거
- 상단 4개 현재/누적 요약 카드 위의 보조 제목과 `기간 필터 미적용` 문구 제거
- 기간 선택 카드의 `현재 범위` 문구와 일반 안내 문구 제거
- 잘못된 직접 선택 기간일 때만 오류 문구 표시
- 하단 `검토 대기 / 검수 대기 / 납기 후보 / 검수·불량 후보` 카드 제거
- 시작일은 오늘 이후 선택 불가
- 종료일은 시작일보다 이전 선택 불가, 오늘 이후 선택 불가
- 날짜 input 키 입력과 붙여넣기 차단
- 업체별 납기·검수 지표에 hover title 추가

## 비고

업체별 hover 상세는 현재 DB에서 계산 가능한 범위로 제공한다. 실제 불량 수량 필드가 아직 없으므로 `검수/불량 후보` 기준은 반려 상태 또는 재작업/리오더 후보로 표시한다.
