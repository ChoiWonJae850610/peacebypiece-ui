# 0.11.59 통계 기간분석 Top5 선택 유지 보정

## 목적

기간분석 탭에서 `완료 작업지시서`, `리오더`, `불량 작업지시서` 중 하나를 선택한 뒤 7일, 30일, custom 기간, 초기화 버튼을 눌러도 사용자가 보고 있던 Top5 종류를 유지한다.

## 수정 내용

- `/admin/dashboard` query string에 `topMode`를 허용했다.
- `topMode` 값은 `completed`, `reorder`, `defect`만 허용하고 그 외 값은 `reorder`로 fallback한다.
- 기간 버튼 href 생성 시 현재 선택 중인 `topMode`를 함께 붙인다.
- 서버에서 읽은 `topMode`를 `AdminStatsDashboard`의 초기 선택값으로 전달한다.
- 기간 변경 후에도 기간분석 탭과 선택 Top5 카드가 유지되도록 했다.

## 확인 항목

1. `/admin/dashboard`에서 기간분석 탭으로 이동한다.
2. 완료 작업지시서 Top5를 선택한다.
3. 7일 또는 30일 버튼을 누른다.
4. 기간분석 탭이 유지되고 완료 작업지시서 Top5가 계속 선택되어 있는지 확인한다.
5. 불량 작업지시서 Top5도 같은 방식으로 확인한다.
6. 초기화 버튼을 눌러도 현재 선택 중인 Top5 종류가 유지되는지 확인한다.

- Recharts 데이터 구조
- 생산 구성 누적 기준 정책
- 업체 성과 누적 기준 정책
