# WAFL List Control Set 0.21.46

## 목적

작업지시서 목록과 원단·부자재 발주서 목록의 검색 입력필드, 필터 select, 생성 버튼, 검색 결과 없음 안내를 같은 목록 control set 기준으로 맞춘다.

## 기준 결정

- 기본 목록 패널 구조는 작업지시서 목록을 기준으로 한다.
- 검색 입력필드의 글자 크기는 발주서 목록 기준인 `text-xs`를 적용한다.
- 검색어 입력 시 지우기 버튼은 작업지시서 목록 기준을 발주서 목록에도 적용한다.
- 검색 결과 없음 안내 박스는 발주서 목록 기준을 작업지시서 목록에도 적용한다.

## 적용 범위

- 작업지시서 목록 검색 input: `text-sm`에서 `text-xs`로 조정
- 작업지시서 목록 empty 상태: 단순 empty card에서 WAFL state block으로 변경
- 발주서 목록 검색 input: clear button 추가
- 발주서 목록 select: `triggerClassName="text-sm"` 명시
- 발주서 목록 생성 버튼: 작업지시서 생성 버튼과 같은 `size="md"`, `pbp-touch-target`, `mt-2.5` 기준으로 조정

## 제외 범위

- DB 변경 없음
- 발주서 생성 로직 변경 없음
- 작업지시서 목록 필터링 로직 변경 없음
- 모바일 전용 목록 패널은 이번 1차 기준에서 제외
