# 0.19.19 멤버 초대 작업 아이콘 정렬과 초대 카드 폭 보정

## 목적

0.19.18 적용 후 멤버 초대 목록의 작업 아이콘이 테이블 우측 라인 밖으로 밀려 보이는 문제를 좁게 보정한다.

## 변경 범위

- 멤버 초대 목록 작업 버튼을 icon-only로 유지한다.
- 복사/취소 아이콘이 active/disabled 상태 모두에서 식별되도록 명시적 크기와 색상을 유지한다.
- 작업 컬럼 폭과 버튼 간격을 줄여 테이블 라인 안에 들어오도록 보정한다.
- 초대 생성 카드와 초대 목록 카드의 비율을 재조정해 좌측 초대 생성 영역을 넓게 유지하면서도 우측 작업 컬럼이 잘리지 않게 한다.

## 유지 기준

- 초대 링크 URL 또는 suffix 컬럼은 다시 표시하지 않는다.
- 버튼 텍스트는 다시 표시하지 않는다.
- disabled 상태에서도 아이콘은 흐리게 보이도록 남긴다.
- 멤버 초대 생성/복사/취소 API 흐름은 변경하지 않는다.

## 후속 방향

멤버 초대 탭은 별도 탭으로 유지하기보다 멤버 관리 단일 화면 안의 섹션으로 합치는 방향이 적합하다. 다만 이번 버전에서는 레이아웃 구조 통합은 하지 않고, 테이블 작업 컬럼 회귀만 보정한다.
