# WAFL Empty Message 단순화 및 중앙 정렬 - 0.21.48

## 목표

작업지시서 목록, 발주서 목록, 작업지시서 상세/우측 패널의 empty message를 짧고 동일한 기준으로 정리한다.

## 기준

- 검색 결과 없음 상태는 제목만 `검색 결과 없음`으로 표시한다.
- 검색 결과 없음의 보조 설명 문구는 제거한다.
- WaflEmptyCard는 내부 텍스트를 가로/세로 중앙 정렬한다.
- 원단·부자재 empty 문구는 `원단·부자재가 없습니다.`로 단순화한다.
- 대표 이미지/첨부파일 empty 문구는 기존 짧은 문구를 유지하되 중앙 정렬 기준을 따른다.

## 적용 범위

- 작업지시서 목록 검색 결과 없음
- 발주서 목록 검색 결과 없음
- 발주서 자재 선택 패널 검색 결과 없음
- 작업지시서 상세 원단·부자재 empty 문구
- 작업지시서 우측 디자인/첨부파일 empty box 중앙 정렬

- 기능 로직 변경 없음
- 표시 문구와 WAFL empty primitive 정렬 기준만 조정
