# 0.10.20 저장소 관리 내부 스크롤 구조 보정

## 목표

`/admin/files` 저장소 관리 화면을 PC 기준 고정형 관리 화면으로 보정한다. 저장소 사용 현황은 상단에서 유지하고, 휴지통 목록만 내부 스크롤되도록 구성한다.

## 변경 기준

- 저장소 사용 현황 카드는 유지한다.
- 저장소 용량 / 파일 운영 요약 / 파일 유형 카드의 높이와 여백을 줄인다.
- `/admin/files`에 `AdminShell`의 `fixed-md` 콘텐츠 모드를 적용한다.
- 휴지통 영역은 남은 높이를 차지하고, 목록 내부만 스크롤되게 한다.
- 모바일은 기존 자연 스크롤 흐름을 크게 해치지 않는다.

## 변경하지 않은 것

- 저장소 snapshot 조회 로직
- 휴지통 복원/삭제/비우기 로직
- R2 purge 흐름
- 감사 로그 흐름
- DB schema
