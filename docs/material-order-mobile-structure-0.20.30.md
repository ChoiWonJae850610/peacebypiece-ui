# 0.20.30 원단·부자재 발주 모바일 구조 1차

## 목적

원단·부자재 발주 화면의 모바일 구조를 PC 3패널 축소 방식이 아니라 목록 → 상세 흐름으로 정리한다.

## 반영 범위

- 모바일에서는 `WaflMobileShell`을 사용한다.
- 모바일 첫 화면은 발주서 목록이다.
- 발주서 선택 시 상세 입력 화면으로 이동한다.
- 상세 화면 하단 고정 액션바에는 목록 복귀와 자재 선택 진입을 배치한다.
- 작업지시서 자재 선택은 기존 `AppSheet` full-size bottom sheet 흐름을 유지한다.
- 모바일 발주 품목 입력은 테이블 대신 카드형 입력 UI를 사용한다.
- PC는 기존 3패널 구조를 유지한다.
- 태블릿은 기존 2패널 + 자재 선택 sheet 구조를 유지한다.

## build fail 보정

0.20.29 build fail 원인인 `mobileDrawer.detailTabs` i18n 타입 누락을 보정했다. `detailTabs`와 `relatedSections`를 실제 접근 경로인 `mobileDrawer` 하위로 이동했다.
