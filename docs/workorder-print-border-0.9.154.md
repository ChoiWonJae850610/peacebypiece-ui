# 0.9.154 작업지시서 출력/PDF 테두리 보정

## 목적

작업지시서 발주요청 출력/PDF 문서에서 마지막 표 영역의 하단 테두리가 보이지 않는 문제를 보완한다.

## 기준

- 화면용 작업지시서 UI는 변경하지 않는다.
- 출력/PDF HTML의 print CSS만 최소 수정한다.
- 빈 페이지 방지 로직과 자동 인쇄 흐름은 유지한다.
- 상태 변경, 첨부, 메모, R2 purge 기능은 변경하지 않는다.

## 변경 내용

### 마지막 section border 제거 방지

이전 출력 CSS에는 `.print-page > *:last-child`에 `border-bottom-width: 0 !important`가 포함되어 있었다. 마지막 출력 section이 `table-section`인 경우 외곽 하단 테두리가 제거될 수 있어, 해당 border 제거 규칙을 삭제했다.

### 마지막 table-section 외곽선 보장

마지막 표 section에도 `border-bottom: 1px solid #78716c`가 유지되도록 별도 규칙을 추가했다.

### footer row 기준 정리

표 footer 영역은 합계 행을 표시하는 영역이므로, section 외곽선과 충돌하지 않는 범위에서 하단 border 기준을 명확히 했다.

## 테스트 기준

1. 발주요청 확인 모달에서 PDF 출력 또는 인쇄 미리보기를 실행한다.
2. 원단/부자재/외주 내역이 있는 작업지시서를 사용한다.
3. 마지막 외주 내역 또는 마지막 table-section 하단 테두리가 보이는지 확인한다.
4. 불필요한 빈 페이지가 추가되지 않는지 확인한다.
5. 대표 이미지/요청사항 영역 높이 정렬이 0.9.153 기준으로 유지되는지 확인한다.
