# 작업지시서 모바일 생산구성 semantic token 적용 — 0.11.96

## 목적

모바일 작업지시서 생산구성 카드에서 입력 가능한 항목, 선택 가능한 항목, 계산되는 항목을 같은 화면 규칙으로 구분한다.

## 적용 기준

- 입력 가능 항목: 수량, 단가 등 사용자가 직접 입력하는 값
- 선택 가능 항목: 구분, 거래처, 단위, 공정, 외주처, 단가 기준 등 후보 중 선택하는 값
- 계산 항목: 금액, 총 금액 등 직접 입력하지 않고 계산되는 값
- 항목 카드: 모바일에서 원단/부자재 또는 외주공정 한 줄을 감싸는 카드

## 변경 범위

- 모바일 원단/부자재 카드
- 모바일 외주공정 카드
- 공통 detail editor shared class
- semantic theme token 설명
- CSS semantic class

## 제외 범위

- 실제 테마 선택 UI 변경 없음
- DB schema 변경 없음
- full_reset.sql 변경 없음
- PC/tablet table 구조 변경 없음
- R2/첨부/메모/휴지통/purge 흐름 변경 없음
