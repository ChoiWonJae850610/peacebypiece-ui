# 0.18.68 통계정보 기간 프리셋/탭 정렬 보정

## 변경
- 7일/30일 버튼은 즉시 조회하지 않고 시작일/종료일 필드만 세팅하도록 변경했습니다.
- 7일은 종료일을 오늘, 시작일을 오늘 기준 7일 전으로 설정합니다.
- 30일은 종료일을 오늘, 시작일을 오늘 기준 30일 전으로 설정합니다.
- 실제 통계 반영은 기존 적용 버튼을 눌렀을 때만 수행합니다.
- AdminSegmentedTabs의 기본 정렬을 태블릿 이상에서 우측 정렬되도록 보정했습니다.

## 유지
- 달력 팝오버 폭/하단 버튼 형태는 0.18.67 기준을 유지했습니다.
- 통계 데이터 계산, DB/API 조회 흐름, WorkspaceShell 구조는 변경하지 않았습니다.
