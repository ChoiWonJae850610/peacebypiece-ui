# 0.11.67 고객관리자 QA 잔여 레이아웃 보정

## 목표

0.11.62~0.11.66에서 재정리한 고객관리자 홈, 환경설정, 멤버관리 흐름 이후 PC 화면에서 남는 시각 밀도 문제를 작게 보정한다.

## 반영 내용

- 고객관리자 홈 운영 관리 카드의 최소 높이와 패딩을 축소했다.
- 4개 운영 카드가 한 줄로 들어오는 구조는 유지했다.
- 카드 설명 문구의 줄 높이와 badge 크기를 줄여 PC 화면에서 과하게 커 보이지 않도록 했다.
- 검토·발주 대기 제목 옆 작업지시서 바로가기 버튼에 PC/tablet 기준 짧은 텍스트를 추가했다.
- 모바일에서는 기존처럼 아이콘 중심의 compact 버튼을 유지했다.

## 변경하지 않은 내용

- 운영 관리 카드 4개 구성은 유지한다.
- 작업지시서 업무 화면 카드는 복구하지 않는다.
- 환경설정 카드는 복구하지 않는다.
- 작업지시서 바로가기 링크는 `/worker`를 유지한다.
- DB/API/권한 로직은 변경하지 않는다.

## 확인 항목

- `/admin` PC 화면에서 운영 관리 카드가 한 줄로 보이는지 확인한다.
- 운영 관리 카드가 0.11.61처럼 과하게 커 보이지 않는지 확인한다.
- 검토·발주 대기 영역의 작업지시서 바로가기 버튼이 눈에 보이는지 확인한다.
- 모바일에서는 버튼 텍스트가 숨겨지고 아이콘만 표시되는지 확인한다.
