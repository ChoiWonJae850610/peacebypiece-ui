# 0.9.22273 — 생산품 유형 세부 분석 카드 구조 보정

## 목적

통계정보 화면의 생산품 유형 분석에서 세분류 전체 TOP5를 별도 카드로 반복 표시하던 구조를 제거하고, 선택한 항목의 하위 항목을 보여주는 구조로 정리한다.

## 화면 기준

- `/admin/dashboard`
- 고객관리자 > 통계정보

## 변경 전 문제

- 왼쪽 생산품 유형 비율 카드에서 이미 선택 depth의 TOP 목록을 보여준다.
- 오른쪽 카드도 `세분류 생산품유형 TOP5`를 다시 보여주면서 정보가 중복된다.
- 세분류 전체 비교는 `반팔`, `오버핏`, `린넨`처럼 서로 다른 성격의 항목이 한 차트에 섞일 수 있다.

## 변경 후 구조

### 왼쪽 카드

`생산품 유형 비율`

- 대분류 / 품목 선택
- 도넛 차트
- 선택 기준 TOP 목록
- 항목 클릭 시 오른쪽 세부 분석 카드가 갱신됨

### 오른쪽 카드

`선택 항목 세부 분석`

- 대분류 선택 시: 선택한 대분류 안의 품목 TOP5
- 품목 선택 시: 선택한 품목 안의 세부형태 TOP5

## 용어 기준

- 대분류: 상의 / 하의 / 아우터 / 원피스
- 품목: 티셔츠 / 셔츠 / 블라우스 / 슬랙스 / 자켓 등
- 세부형태: 반팔 / 긴팔 / 오버핏 / 테일러드 / 플리츠 등

## DB 조회 기준

추가 조회는 기존 `spec_sheets`의 카테고리 연결을 사용한다.

- `category1_id` → `category2_id`: 대분류별 품목 TOP5
- `category2_id` → `category3_id`: 품목별 세부형태 TOP5
