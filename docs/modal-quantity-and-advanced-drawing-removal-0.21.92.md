# 0.21.92 모달 수량 조건 및 고급 그리기 제거

## 변경 요약

- 작업지시서 제작공정/추가공정/원단·부자재 모달의 적용 가능 수량 기준을 0 초과, 1 이상으로 통일했다.
- 발주서 발주 품목 추가 모달의 주문수량도 1 이상일 때만 추가 가능하도록 맞췄다.
- 발주서 발주 품목 추가 모달의 주문수량/단가 입력을 `AppNumberInput`으로 교체해 세 자리 콤마 표시와 WAFL 숫자 입력 동작을 공유한다.
- 디자인 영역의 고급 그리기(tldraw PoC) 진입점, 런타임 플래그, i18n 문구, 패키지 의존성, 모달 파일을 제거했다.

## 확인 포인트

- 제작공정 모달에서 업체 선택 후 수량 0이면 적용 버튼이 비활성화된다.
- 추가공정 모달에서 항목/업체 선택 후 수량 0이면 적용 버튼이 비활성화된다.
- 원단·부자재 추가/수정 모달에서 자재명과 구분이 있어도 수량 0이면 적용 버튼이 비활성화된다.
- 발주 품목 추가 모달에서 주문수량 0이면 추가 버튼이 비활성화된다.
- 발주 품목 추가 모달 주문수량/단가 입력값은 1,000처럼 콤마가 표시된다.
- 디자인 메뉴에는 직접 그리기만 남고 고급 그리기는 표시되지 않는다.
