# 0.9.22427 관리자 통계 달력 popover UX 보정

## 목표
관리자 통계 화면의 기간 선택 달력이 페이지 공간을 과도하게 차지하지 않도록 popover 크기와 트리거 영역을 정리한다.

## 변경 사항
- 시작일/종료일 트리거 영역을 더 작은 버튼형 UI로 조정했다.
- DayPicker popover 폭과 내부 여백을 줄였다.
- 날짜 셀 크기를 줄여 통계 화면의 시각적 점유율을 낮췄다.
- popover 내부에 완료 버튼을 추가해 선택 후 명시적으로 닫을 수 있게 했다.
- 날짜 입력은 input/select 없이 DayPicker 날짜 클릭으로만 수행되도록 유지했다.
- 한국어/영어 locale 표시와 미래 날짜 선택 차단 정책을 유지했다.

## 주의
- 키보드로 날짜 문자열을 직접 입력하는 경로는 없다.
- `react-day-picker` 라이브러리는 유지한다.
- package.json/package-lock.json은 수정하지 않았다.

## 테스트
1. `/admin/dashboard` 진입
2. 기간 선택 영역 클릭
3. 작은 popover 달력이 열리는지 확인
4. 날짜가 직접 입력이 아니라 마우스 클릭으로만 선택되는지 확인
5. 완료 버튼으로 popover가 닫히는지 확인
6. 한국어/영어 locale에서 월/요일 표시 확인
