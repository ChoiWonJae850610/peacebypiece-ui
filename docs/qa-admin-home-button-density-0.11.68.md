# 0.11.68 QA — 고객관리자 홈 카드 밀도와 홈 버튼 형태 보정

## 변경 목적

0.11.67에서 고객관리자 홈 운영 관리 카드가 과하게 축소되어 PC 화면의 시원한 느낌이 줄어든 문제를 보정한다. 또한 검토·발주 대기 영역의 작업지시서 바로가기 버튼은 원래 의도대로 문서 아이콘만 노출하고, 작업지시서 화면의 홈 이동 버튼은 관리자 화면 우측 상단 홈 버튼과 같은 원형 아이콘 형태로 통일한다.

## 반영 내용

- 운영 관리 카드 4개 한 줄 구조는 유지한다.
- 0.11.67에서 줄였던 카드 padding, 최소 높이, 제목/설명 크기, badge 크기를 일부 복원한다.
- 검토·발주 대기 제목 옆 `/worker` 바로가기 버튼에서 `업무화면` 텍스트를 제거한다.
- `/worker` 바로가기 버튼은 문서 아이콘만 표시한다.
- 작업지시서 화면의 홈 이동 버튼을 텍스트형 `← 관리자 홈` 버튼에서 원형 홈 아이콘 버튼으로 변경한다.
- 홈 이동 목적지는 기존 `homeNavigation.href`를 그대로 사용한다.

## 변경하지 않은 내용

- 운영 관리 카드 목록 4개 정책은 유지한다.
- 작업지시서 업무 화면 카드를 다시 추가하지 않는다.
- 환경설정 카드를 다시 추가하지 않는다.
- `/worker` 이동 경로는 변경하지 않는다.
- 작업지시서 홈 이동 목적지 계산 로직은 변경하지 않는다.
- DB/API/권한 로직은 변경하지 않는다.

## 확인 항목

1. `/admin`에서 운영 관리 카드 4개가 한 줄로 보이는지 확인한다.
2. 카드가 0.11.67보다 덜 작고 시원하게 보이는지 확인한다.
3. 검토·발주 대기 제목 옆 버튼에 텍스트가 없고 문서 아이콘만 보이는지 확인한다.
4. 해당 아이콘 버튼 클릭 시 `/worker`로 이동하는지 확인한다.
5. 작업지시서 화면의 홈 이동 버튼이 관리자 topbar 홈 버튼과 같은 원형 홈 아이콘 형태인지 확인한다.
6. 작업지시서 홈 이동 버튼의 목적지가 기존과 동일하게 동작하는지 확인한다.
