# 0.10.30 — 고객관리자 환경설정 IA 재구성

## 변경 목표

고객관리자 환경설정 화면을 실제 운영 설정 중심으로 재구성한다.
기존의 임시 조직 설정, 정책 설명, 권한 미리보기 구성은 제거하고, 다음 메뉴 구조로 정리한다.

- 기준정보 설정
- 알림 정책
- 요금제·결제
- 계정 설정
- 개발 건의

권한 관리는 환경설정에서 제외하고 멤버관리 화면에서 다루는 방향으로 분리한다.

## 반영 내용

- `/admin/settings` 화면을 `AdminSettingsHub` 기반 메뉴형 화면으로 변경
- 기준정보 설정 메뉴에서 생산품 유형, 단위 표준, 외주공정 유형 관리를 유지
- 알림 정책은 클릭 시 개발중 안내 모달 표시
- 알림 정책 모달에 `2026년 하반기 이후 순차 적용 예정` 문구 반영
- 요금제·결제, 계정 설정, 개발 건의는 준비중 안내 모달로 분리
- 권한 관리는 환경설정 메뉴에서 제외
- 기존 기준정보 관리 모달 로직은 유지

## 변경하지 않은 내용

- DB schema 변경 없음
- 실제 결제 연동 없음
- 실제 알림 발송 없음
- 실제 회원 탈퇴 처리 없음
- 멤버/권한관리 구현 변경 없음
- 감사 로그 흐름 변경 없음
