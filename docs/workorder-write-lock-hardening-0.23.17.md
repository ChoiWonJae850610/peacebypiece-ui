# 0.23.17 작업지시서 저장 중복 요청 방지 강화

## 목적
React 상태 갱신 전에 동일 저장 또는 작업 버튼이 빠르게 반복 입력되는 경우, 동일 요청이 중복 실행될 수 있는 짧은 구간을 동기식 ref 잠금으로 차단합니다.

## 변경 내용
- 작업지시서 workspace 공통 쓰기 작업에 즉시 반영되는 ref 잠금 추가
- 저장·생성·삭제·재발주 등 공통 write lock을 사용하는 작업의 연속 실행 방지
- 공장 전달사항 저장에 별도 in-flight ref 잠금 추가
- 성공·실패 여부와 관계없이 finally에서 잠금 해제
- 기존 버튼 비활성화, 상태 문구, 토스트 교체 정책은 유지

## 영향 범위
- UI 표시 및 API 규격 변경 없음
- DB Migration 없음
