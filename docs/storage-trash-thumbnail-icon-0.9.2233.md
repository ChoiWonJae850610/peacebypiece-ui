# 0.9.2233 — 저장소 휴지통 썸네일/파일 아이콘 1차 적용

## 목적

저장소 관리의 휴지통 목록과 상세 모달에서 파일 유형을 더 빠르게 구분할 수 있도록 작은 시각 표시를 추가한다.

## 적용 범위

- `/admin/files`
- 휴지통 목록의 대상 컬럼
- 휴지통 상세 모달 상단 요약

## 반영 내용

1. 작업지시서 단위 항목은 `작지` 배지로 표시한다.
2. 디자인 이미지는 `IMG` 배지로 표시한다.
3. PDF 문서는 `PDF` 배지로 표시한다.
4. 기타 파일은 `FILE` 배지로 표시한다.
5. 실제 R2 이미지 썸네일은 아직 연결하지 않는다.
6. row 높이가 크게 흔들리지 않도록 32px 수준의 compact visual만 적용한다.

## 보류한 내용

- 실제 R2 preview URL 기반 이미지 썸네일
- PDF preview thumbnail
- 이미지 로딩 실패 fallback
- 큰 미리보기 모달

위 항목은 R2 preview URL 정책과 fallback 처리를 확정한 뒤 별도 버전에서 진행한다.
