# 0.20.35 모바일 FAB/드로어 UX 오류 보정

## 목적

0.20.34에서 적용한 모바일 FAB 단일 진입 구조를 실제 사용 흐름 기준으로 보정한다.
작업지시서와 원단·부자재 발주 화면의 목록 드로어 패턴을 더 일관되게 맞추고,
모바일 직접그리기 가로모드 전환 후 모달 상태가 남는 문제를 차단한다.

## 변경 범위

- 원단·부자재 발주 목록 드로어를 작업지시서 모바일 드로어와 유사한 폭/형태로 보정
- 발주 도구 sheet에서 본문과 중복되는 `발주서` 탭 제거
- 발주 도구 sheet의 첫 탭을 `작업지시서` 선택 흐름으로 변경
- 작업지시서 모바일 FAB 라벨을 `관련 정보`에서 `추가정보`로 변경
- 모바일 직접그리기 landscape 차단 상태에서 닫기 시 drawing modal/session/draft 상태를 함께 정리

## 변경하지 않은 범위

- PC 3패널 구조
- 태블릿 구조
- 상태전환 로직
- 권한 로직
- API/DB/R2 흐름
- 첨부/메모/휴지통/purge 정상 흐름
