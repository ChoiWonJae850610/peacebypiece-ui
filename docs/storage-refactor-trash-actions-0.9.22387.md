# 0.9.22387 저장소 휴지통 액션 상태 리팩토링

## 목적

고객관리자 저장소 휴지통 화면의 선택 상태, 액션 가능 여부, 작업지시서 처리 미리보기 계산을 `FileTrashSection.tsx`에서 분리했다.

## 변경 내용

- `fileTrashSectionActions.ts` 추가
- 선택된 첨부파일/작업지시서 수 계산 분리
- 복원/선택 삭제/비우기 버튼 활성화 조건 분리
- 작업지시서 묶음 처리 미리보기 집계 분리
- 기존 휴지통 UI, row 생성, 컬럼 렌더링, 삭제/복원 동작은 변경하지 않음

## 확인 항목

1. 휴지통 목록 표시
2. 복원 버튼 활성화 조건
3. 선택 삭제 버튼 활성화 조건
4. 비우기 버튼 활성화 조건
5. 작업지시서 묶음 복원/삭제 미리보기 모달 표시
6. 선삭제 첨부파일과 작업지시서 묶음 첨부파일 구분 유지
