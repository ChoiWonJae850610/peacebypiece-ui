# 0.20.47 workspace header and responsive IA

## Scope
- Align the workorder desktop/tablet-landscape header with the WAFL workspace topbar pattern used by material orders.
- Keep mobile and tablet portrait list navigation drawer-based so the main content has more width.
- Keep mobile progress vertical and tablet progress horizontal.

## Policy
- Mobile: portrait-first, drawer list, single-content focus, vertical progress.
- Tablet portrait: drawer list, wider content, horizontal progress.
- Tablet landscape: PC-like split workspace, horizontal progress.
- Desktop: common WAFL topbar and multi-panel workspace.

## Non-goals
- No workflow, permission, API, DB, R2, attachment, memo, trash, or purge changes.
