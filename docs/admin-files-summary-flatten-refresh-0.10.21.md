# 0.10.21 저장소 관리 현황 카드 축소 및 새로고침 피드백

## 목표

`/admin/files` 저장소 관리 화면에서 상단 현황 영역의 중첩 카드 구조를 줄이고, 휴지통 목록 영역이 더 많은 행을 표시하도록 정리한다.

## 변경 기준

- 외부 `저장소 사용 현황` 카드 제목/설명 영역을 제거한다.
- 현황 카드 3개를 본문 상단에 직접 배치한다.
- 새로고침 아이콘 버튼은 휴지통 액션 라인의 한글 버튼으로 이동한다.
- 새로고침 실행 시 하단 중앙 토스트로 `새로고침 중입니다.` / `새로고침되었습니다.` 메시지를 표시한다.
- 휴지통 목록 내부 스크롤 구조와 기존 복원/삭제/비우기 액션은 유지한다.

## 변경하지 않은 것

- 저장소 snapshot 조회 API
- 휴지통 복원/선택 삭제/비우기 처리 로직
- R2 purge 흐름
- 감사 로그 흐름
- DB schema
