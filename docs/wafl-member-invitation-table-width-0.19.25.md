# 0.19.25 멤버 초대 링크 목록 table width 보정

## 목표

멤버관리 하단 초대 관리 영역에서 초대 링크 목록이 PC, 태블릿, 모바일 가로 화면에서 카드 내부 전체 폭을 사용하도록 보정한다.

## 적용 기준

- 모바일 세로만 카드형 유지
- 모바일 가로, 태블릿 세로, 태블릿 가로, PC는 테이블형 유지
- 테이블 wrapper는 전체 폭을 사용
- 상태, 생성일, 만료일, 작업 컬럼은 전체 폭 안에서 균등/준균등 분배
- 각 셀 내용은 가운데 정렬
- 작업 버튼은 작업 컬럼 중앙에 배치

## 변경하지 않는 항목

- 멤버 초대 생성 API
- 초대 취소 API
- 링크 복사/공유 실제 동작
- 승인/거절/권한 저장
- DB/API/R2 흐름
