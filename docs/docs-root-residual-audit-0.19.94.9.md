# docs 루트 잔여 문서 점검 0.19.94.9

## 목적

0.19.94.8에서 보관 후보 문서 185개를 `docs/보관문서/` 하위로 이동한 뒤, `docs/` 루트에 남은 문서와 README 링크 기준을 재점검한다.

이번 버전은 기능 코드, DB schema, API route, R2 업로드 흐름을 변경하지 않는다.

## 기준 상태

- 기준 버전: 0.19.94.8
- 결과 버전: 0.19.94.9
- docs 루트 md 파일 수: 44개
- 삭제 파일: 없음
- 이동 파일: 없음

## 루트 잔여 문서 판단

`docs/` 루트에는 최근 운영 판단에 가까운 문서를 남긴다.

- 정책/약관 공개 및 동의 흐름 문서
- DB/API smoke test 문서
- Playwright E2E 도입 및 보정 문서
- 회사 계정 요청/승인 흐름 문서
- 멤버 생명주기 문서
- 협력업체 연락처 중복 기준 문서
- 고객사 회사 파일 업로드 설계 문서
- 0.19.94대 문서 정리 기준 문서

## README 링크 보정

0.19.94.8 이후 `docs/README.md`에 남아 있던 과거 0.18~0.19 초반 작업 기록 링크는 실제 위치가 `docs/보관문서/` 하위로 이동했으므로 직접 링크 목록에서 제거한다.

대신 아래 기준을 사용한다.

- 현재 개발 기준: `docs/현재기준/`
- 정책/약관 기준: `docs/정책문서/`
- 과거 작업 기록: `docs/보관문서/`
- 최근 운영 문서: `docs/` 루트

## 보류 사항

- `docs/현재기준/`, `docs/정책문서/`, `docs/보관문서/` 내부 링크 전수 검사는 다음 정리 단계에서 진행한다.
- 보관문서 내부 파일명 변경은 하지 않는다.
- 문서 삭제는 하지 않는다.
