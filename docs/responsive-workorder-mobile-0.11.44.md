# 작업지시서 모바일 보정 1차 (0.11.44)

## 목적

작업지시서 업무 화면의 모바일 폭에서 카드, 섹션, 액션 버튼, 첨부/메모 패널이 화면 밖으로 밀리는 위험을 줄인다.

## 반영 범위

- 작업지시서 모바일 레이아웃 wrapper 여백 축소
- 작업지시서 모바일 상세 카드의 `min-w-0`, `overflow-x-hidden` 보강
- 기본 정보 카드의 긴 제목/요약/담당자 표시 줄바꿈 보정
- workflow 단계 badge와 action button의 모바일 줄바꿈 보정
- 발주정보 카드의 1열/2열 전환 기준 보정
- 생산구성 모바일 섹션 wrapper overflow 보정
- 모바일 메모/첨부 side panel accordion wrapper overflow 보정
- 작업지시서 목록 카드의 `min-w-0` 보강

## 변경하지 않은 범위

- 작업지시서 제목/담당자 즉시 저장 정책
- 발주정보 저장 버튼 정책
- 생산구성 저장 버튼 정책
- workflow 상태 변경 로직
- 첨부/R2 upload/download/delete/restore/purge 흐름
- 메모 저장/삭제/답글 로직
- DB schema
- API 응답 포맷

## 확인 항목

1. 모바일 폭에서 작업지시서 목록 카드 제목이 화면 밖으로 밀리지 않는지 확인한다.
2. 작업지시서 상세 기본 정보 카드의 요약/담당자/현재 재고 값이 줄바꿈되는지 확인한다.
3. workflow 단계 badge가 모바일 폭에서 자연스럽게 줄바꿈되는지 확인한다.
4. workflow action button의 텍스트가 버튼 밖으로 밀리지 않는지 확인한다.
5. 발주정보 카드의 공장명, 상태 badge, 수량/공임/손실비가 좁은 폭에서 접근 가능한지 확인한다.
6. 첨부/메모 accordion이 모바일 폭에서 화면 밖으로 밀리지 않는지 확인한다.
