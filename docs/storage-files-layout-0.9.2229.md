# 0.9.2229 — 통계 기간 초기화 + 저장소 상단/휴지통 UX 정리

## 대상 화면

- `/admin/dashboard`
- `/admin/files`

## 변경 요약

1. 통계정보 직접 기간 선택 영역에 `초기화` 버튼을 추가했다.
2. 저장소 관리 상단의 7일/15일/30일 기간 버튼을 제거하고 새로고침 버튼만 유지했다.
3. 저장소 상단 영역을 사용량 요약 / 파일 상태 요약 / 파일 유형 비율 구조로 재정리했다.
4. 휴지통 목록에서 복구정책 컬럼과 row 내 직접 작업 버튼을 제거했다.
5. 휴지통 row 클릭 시 상세 모달에서 복구/영구삭제를 처리하도록 변경했다.
6. 휴지통 목록 글씨 크기를 키우고 긴 값은 말줄임 처리하도록 정리했다.

## 보류

- 썸네일은 이번 버전에서 적용하지 않는다. R2 preview URL 및 fallback 정책을 먼저 안정화한 뒤 별도 버전에서 진행한다.
