# 0.10.32 환경설정 기준관리 모달 공통 프레임 정리

## 목적

환경설정의 기준정보 3종은 모두 기준값 관리 기능이지만 데이터 구조가 다르다.

- 생산품 유형: 1차 → 2차 → 3차 계층 구조
- 단위 표준: 한글명 + 영문 코드/약어 세트
- 외주공정 유형: 단순 공정 목록

따라서 본 버전에서는 세 기능을 하나의 동일 폼으로 억지 통합하지 않고, 공통 모달 프레임과 공통 스타일만 분리했다.

## 반영 내용

- `StandardManagementModalFrame` 추가
- 기준관리 모달의 제목/설명/분류 배지/본문 간격 통일
- 추가 버튼, 목록 박스, 목록 스크롤, 목록 행 스타일 공통화
- 생산품 유형 모달은 계층형 3단 구조 유지
- 단위 표준 모달은 한글명 + 영문 코드 입력 구조 유지
- 외주공정 유형 모달은 사용/미사용 단순 목록 구조 유지

## 변경하지 않은 것

- 기준정보 저장 API 변경 없음
- DB schema 변경 없음
- 생산품 유형 계층 구조 변경 없음
- 단위 표준 데이터 구조 변경 없음
- 외주공정 유형 저장 흐름 변경 없음
- 감사 로그 흐름 변경 없음
