# UI Tooltip 적용 기록 - 0.18.15

## 목적

긴 안내 문구를 화면에 계속 노출하지 않고, 사용자가 필요한 지점에서만 확인할 수 있도록 `AppTooltip` 적용 범위를 넓힌다.

## 이번 적용 범위

- 관리자 통계 화면의 공장별 성과 테이블
- 공장명/생산 수량 셀
- 납기 지연율 배지
- 검수/불량률 배지

## 적용 기준

- 기존 `title` 기반 브라우저 기본 툴팁을 `AppTooltip`으로 전환한다.
- 화면에 항상 보이는 긴 보조 문구를 늘리지 않는다.
- 통계 계산, 필터, API, DB 흐름은 변경하지 않는다.

## 후속 후보

- 시스템 기준정보의 긴 category help text
- 초대 링크 생성 화면의 긴 helper 문구
- 작업지시서/원단·부자재 화면의 반복 안내 문구
