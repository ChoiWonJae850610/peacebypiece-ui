# 고객사 메인·작업지시서 목록 compact 정리 — 0.23.90

- 고객사 관리자 메인의 대형 hero와 대형 queue 카드를 제거했다.
- 검토 대기, 발주 대기, 검수 대기, 입고 지연을 compact queue 카드로 통합했다.
- 작업지시서와 원단·부자재 발주 화면을 같은 업무 헤더에서 바로 열 수 있게 했다.
- 선택 queue는 최대 8개의 실제 DB 업무 항목을 compact 목록으로 표시한다.
- 정상·사용 가능 배지는 숨기고 미완성 기능만 표시한다.
- 고객사 관리 바로가기 카드와 작업지시서 좌측 목록 높이를 축소했다.
