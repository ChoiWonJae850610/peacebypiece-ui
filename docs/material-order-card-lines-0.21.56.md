# 0.21.56 작업지시서/발주서 카드 액션 및 발주 품목 카드형 정리

- 작업지시서 목록 카드는 같은 항목 재클릭 시 선택을 해제한다.
- 작업지시서 목록 더보기 메뉴의 삭제 항목에 휴지통 아이콘을 추가한다.
- 발주서 금액 표시는 `0원`, `100원` 형식으로 통일한다.
- 발주서 기본정보의 총 비용 카드는 작업지시서 비용 요약 총 비용 카드와 같은 info tone을 사용한다.
- 발주 품목 영역의 설명 문구와 테이블 헤더를 제거하고 카드형 목록으로 표시한다.
- 발주 품목 수정은 카드 더보기 메뉴의 수정 액션으로 모달을 열어 처리한다.
- 발주 품목 삭제는 카드 더보기 메뉴의 삭제 액션으로 처리한다.
- 제작공정 샘플/재작업/불량 표시 로직은 `lib/workorder/reorder/helpers.ts`에 유지되어 있으며 이번 패치에서는 UI 표시 회귀를 만들지 않는다.
