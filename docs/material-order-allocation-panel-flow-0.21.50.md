# 0.21.50 원단·부자재 발주 우측 패널 흐름 정리

## 목적

원단·부자재 발주 화면의 우측 작업지시서 패널을 독립 검색 패널이 아니라, 좌측 발주서 선택과 가운데 발주 기본정보의 자재 종류에 종속되는 자재 선택 패널로 정리한다.

## 기준

- 발주서가 선택되지 않은 상태에서는 우측 패널을 빈 상태로 둔다.
- 발주서가 선택된 뒤에만 우측 패널에 작업지시서 자재 카드가 표시된다.
- 가운데 패널에서 자재 종류가 원단이면 우측 패널에는 원단 항목이 있는 작업지시서 카드만 표시한다.
- 가운데 패널에서 자재 종류가 부자재이면 우측 패널에는 부자재 항목이 있는 작업지시서 카드만 표시한다.
- 우측 패널의 제목, 건수 뱃지, 검색 필드는 제거한다.
- 우측 패널은 첫 카드부터 바로 보이도록 구성한다.
- 발주서 목록에서 이미 선택된 발주서를 한 번 더 클릭하면 선택을 해제한다.

## 작업지시서 목록 선택 기준

작업지시서 화면의 목록 재클릭 선택 해제 동작은 기존 `useWorkOrder`의 `handleSelectWorkOrder`에 이미 반영되어 있다. 같은 작업지시서 id를 다시 선택하면 `selectedId`를 빈 값으로 되돌리고 저장 상태를 안정 상태로 갱신한다.

## 모바일/태블릿 기준

- 모바일에서는 발주서가 선택된 경우에만 작지·자재 하단 액션을 열 수 있다.
- 하단 액션 안의 작업지시서 카드도 현재 자재 종류 기준을 그대로 따른다.
- PC/태블릿 3패널에서는 우측 고정 패널을 유지하되, 발주서 미선택 시 빈 상태로 둔다.
