# 0.18.63 통계정보 달력 팝오버 공통 보정

## 목적

통계정보 기간 분석에서 사용하는 공통 `AdminDateRangePicker`의 달력 팝오버 하단 영역이 잘리는 문제를 줄이고, 선택 기간 안내 문구로 인한 불필요한 공간을 제거했습니다.

## 반영 내용

- `AdminDateRangePicker` 팝오버 폭을 달력 콘텐츠 기준으로 축소
- 팝오버 내부 최대 높이와 내부 스크롤 기준 추가
- `선택 기간` 안내 문구 제거
- 팝오버 하단 `비우기` / `완료` 버튼을 icon-only compact 버튼으로 변경
- `비우기` / `완료` 버튼의 `aria-label`, `title`, `sr-only` 텍스트 유지
- 통계정보 기간 선택/적용 로직 변경 없음

## 영향 범위

- 공통 `AdminDateRangePicker` UI
- 통계정보 기간 분석 날짜 선택 팝오버

## 비변경

- 통계 데이터 계산 변경 없음
- 기간 적용 href 흐름 변경 없음
- WorkspaceShell 스크롤 구조 변경 없음
- DB/API 통계 조회 흐름 변경 없음
