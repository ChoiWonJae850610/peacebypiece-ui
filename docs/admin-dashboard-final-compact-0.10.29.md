# 0.10.29 통계 화면 최종 압축 보정

## 목표
PC 기준 통계 화면에서 불필요한 세로 여백을 줄이고, 상단 요약 카드와 탭 콘텐츠가 한 화면에 더 안정적으로 들어오도록 미세 조정한다.

## 반영
- 상단 누적 카드 높이와 내부 여백을 추가 축소했다.
- 통계 섹션 외부 카드와 탭 영역의 위아래 간격을 줄였다.
- 생산 구성, 업체 성과, 기간 분석 탭 콘텐츠의 최소 높이를 낮췄다.
- 도넛/막대/기간분석 카드의 내부 간격을 조금 더 줄였다.
- 업체별 납기·검수 지표 테이블의 헤더/행 여백을 줄였다.

## 유지
- 통계 계산 로직은 변경하지 않았다.
- 통계 API와 repository는 변경하지 않았다.
- DB schema는 변경하지 않았다.
- 감사 로그 흐름은 변경하지 않았다.
