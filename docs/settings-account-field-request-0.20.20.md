# 0.20.20 환경설정 회사 정보 필드별 수정 요청 UI

## 목적

환경설정 > 회사 정보 탭의 회사 정보 변경 흐름을 하단 일괄 작성 영역에서 필드별 수정 요청 모달로 전환한다.

## 변경 기준

- 회사명, 사업자등록번호, 회사 주소는 각 행의 `수정 요청` 버튼으로 요청을 시작한다.
- 버튼을 누르면 현재 값을 확인할 수 있는 모달을 열고, 변경 내용과 사유를 입력한다.
- 요청은 기존 `company_info_change` 요청 흐름을 재사용한다.
- 요청 메시지에는 변경 대상 필드와 현재 값을 포함해 시스템관리자가 검토할 수 있게 한다.
- 탈퇴 요청은 회사 정보 변경과 분리된 단일 위험 액션으로 유지한다.
- 요청 이력은 회사 정보 탭 하단에 계속 표시한다.

## UI 원칙

- 상단의 일괄 `회사 정보 변경 요청` 버튼은 제거한다.
- 사용자는 변경할 필드 옆의 액션을 눌러야 한다.
- 모달은 현재 값, 입력 영역, 요청 접수 버튼만 포함한다.
- 회사 정보 행의 설명은 짧게 유지한다.

## 범위 제외

- DB schema 변경 없음
- 시스템관리자 검토 API 변경 없음
- 요청 승인 후 실제 회사 정보 자동 반영은 후속 버전에서 다룬다.
- R2/첨부/메모/휴지통/purge 흐름 변경 없음
