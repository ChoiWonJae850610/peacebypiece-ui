# /worker 화면 밀도 축소 — 0.23.53

## 목적

`/worker` 작업지시서 화면의 기능 구조는 유지하면서 PC 화면에서 과도했던 패널 간격, 목록 카드 높이, 내부 여백과 정보 밀도를 줄인다.

## 적용 내용

- 실제 경로 `/worker`의 데스크톱 workspace에 전용 compact class 적용
- 3패널 간격 축소
- 좌측 목록 패널 폭을 줄이고 중앙 상세 비중 확대
- 목록 패널 padding과 검색·필터 영역 간격 축소
- 작업지시서 목록 카드 세로 padding 축소
- 목록 제목, 상태 배지, 상태 점, 보조정보 간격 축소
- 목록 카드 사이 간격과 하단 여백 축소
- 모바일·태블릿 공통 목록 카드도 같은 밀도 기준을 적용하되 터치 가능한 메뉴 버튼 크기는 유지

## 유지한 동작

- 작업지시서 선택, 생성, 삭제, 리오더 기능
- PC 3패널 구조
- 태블릿·모바일 반응형 정책
- 좌·우 독립 스크롤
- 저장 lifecycle과 mutation 계약

## 위험 요소

- 매우 좁은 데스크톱 viewport에서 패널 최소 폭 때문에 가로 공간이 부족할 수 있음
- 목록 카드 정보량이 많은 경우 줄바꿈 대신 기존처럼 말줄임 표시됨

## 직접 확인 항목

- PC 1440px 이상에서 목록·상세·우측 패널 비율
- PC 1280px 부근에서 패널 잘림 여부
- 태블릿 가로 2~3패널 전환
- 모바일 목록 드로어 카드 높이와 메뉴 터치 영역
- 긴 작업지시서명·업체명·카테고리 말줄임
