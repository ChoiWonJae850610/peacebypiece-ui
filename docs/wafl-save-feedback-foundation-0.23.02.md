# WAFL 저장 피드백 및 수량 표시 정리 0.23.02

## 목적

작업지시서와 발주서의 저장 상태 문구, 자동 저장의 dirty 표시, 발주서 선택 동기화 시 저장 완료 문구 초기화, 재고 수량 표시 형식을 공통 기준으로 정리한다.

## 공통 정책

- 자동 저장 화면은 dirty 상태를 사용자에게 표시하지 않는다.
- saving, saved, error 상태만 노출한다.
- 저장 대상별 문구는 `WAFL_SAVE_TARGET`과 `getWaflSaveFeedbackMessage`에서 생성한다.
- 저장 완료 시각은 저장 상태 UI에 노출하지 않는다.
- 발주서 저장 상태 초기화는 선택한 발주서 ID가 바뀔 때만 수행한다.
- 수량 표시는 공통 숫자 포맷터를 사용한다.
