# 0.10.27 통계 화면 탭 전환감 보정

## 목적
0.10.26에서 도입한 통계 섹션 탭 구조의 높이와 전환감을 미세 조정한다.

## 변경
- 통계 섹션 외부 카드의 padding을 소폭 축소했다.
- 탭 콘텐츠 영역에 `overflow-hidden`과 최소 높이를 적용해 좌우 슬라이드 전환 시 화면 흔들림을 줄였다.
- 탭 전환 거리를 줄여 과한 움직임을 완화했다.
- 생산 구성, 업체 성과, 기간 분석 카드의 최소 높이와 내부 간격을 소폭 정리했다.
- `motion-reduce` 환경에서는 전환 효과를 줄이도록 처리했다.

## 비변경
- 통계 계산 로직 변경 없음
- 통계 API/repository 변경 없음
- DB schema 변경 없음
- 감사 로그 흐름 변경 없음
