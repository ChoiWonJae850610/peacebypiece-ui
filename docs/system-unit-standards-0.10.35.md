# 0.10.35 시스템관리자 단위 표준 관리 1차 화면

## 목적

고객관리자 환경설정에서 단위 표준을 직접 추가하지 않고 시스템 표준 중 사용 여부만 선택하도록 정책을 정리했기 때문에, 시스템관리자가 관리할 단위 표준 원장의 1차 화면을 추가한다.

## 반영 내용

- `/system/standards/units` 경로 추가
- 시스템 기준정보 설계 화면에서 단위 표준 관리 화면으로 이동하는 링크 추가
- 시스템 콘솔 빠른 링크에 단위 표준 관리 추가
- 단위 표준 원장의 기준 필드 화면화
  - 한글명
  - 영문 코드/약어
  - 설명
  - 사용 예시
  - 정렬 순서
  - 상태
- 단위 추가 버튼은 비활성 상태로 표시
- 실제 CRUD, DB schema, 고객사별 사용 여부 저장은 후속 버전에서 분리

## 변경하지 않은 것

- DB schema 변경 없음
- 단위 표준 저장 로직 연결 없음
- 고객관리자 기준정보 저장 로직 변경 없음
- 감사 로그 흐름 변경 없음
