# 관리자 통계 화면 레이아웃 UX 보정 — 0.9.22431

## 목적
0.9.22429에서 통계 화면 구조를 재배치한 뒤 카드 높이, 간격, grid 비율이 불균형해 보일 수 있는 지점을 1차로 보정한다.

## 반영 내용
- 상단 현재 요약 카드 4개의 최소 높이를 통일했다.
- 생산품 유형 비율, 선택항목 품목 TOP5, 업체 성과, 업체별 납기/검수 지표 카드를 2열 균등 grid로 정리했다.
- 주요 분석 카드에 `auto-rows-fr`, `h-full`, `min-h` 기준을 적용해 같은 행의 카드 높이가 맞도록 보정했다.
- 기간별 분석의 카드 상단 간격을 줄이고, 기간 선택 컨트롤이 PC와 모바일에서 자연스럽게 줄바꿈되도록 정렬을 보정했다.
- 기간 요약 카드와 동적 TOP5 카드를 같은 2열 균등 grid에 배치했다.

## 확인 필요
- npm run build 미실행 — 사용자가 로컬에서 확인
- /admin/dashboard PC 화면에서 오른쪽 빈 공간이 과하게 생기지 않는지 확인
- 모바일/태블릿에서 카드가 세로로 자연스럽게 쌓이는지 확인
- DayPicker popover가 기간 선택 버튼 영역 안에서 정상 표시되는지 확인
