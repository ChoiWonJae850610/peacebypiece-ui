# 0.9.2231 — 저장소 파일 운영 요약과 휴지통 목록 밀도 보정

## 목표

저장소 관리 화면의 상단 파일 운영 요약을 실제 판단에 필요한 3개 항목으로 축소하고, 휴지통 목록 row 높이가 들쭉날쭉해지는 보조 문구를 상세 모달로 이동한다.

## 적용 기준

- 좌측 카드는 요금제/용량 사용량을 유지한다.
- 가운데 카드는 사용중 파일, 휴지통 파일, 대용량 파일만 보여준다.
- 우측 파일 유형 도넛은 유지한다.
- 휴지통 목록에는 최소 정보만 표시한다.
- 작업지시서의 현재 단계, 첨부파일 수, 메모 수는 row가 아니라 상세 모달에서 확인한다.

## 변경 사항

### 파일 운영 요약

기존의 `첨부파일`, `정리 필요` 항목을 제거했다.

남긴 항목:

- 사용중 파일: 개수와 용량
- 휴지통 파일: 개수와 용량
- 대용량 파일: 후속 연결 예정 상태

### 휴지통 목록

작업지시서 row 아래에 나오던 현재 단계와 메모 개수를 제거했다. grouped attachment 안내 문구도 row에서 제거했다.

상세 모달에는 다음 정보를 유지/추가한다.

- 작업지시서 항목: 현재 단계, 첨부파일 수, 메모 수, 삭제일시, 복구정책
- 파일 항목: 유형, 용량, 연결 작업지시서, 삭제일시, 복구정책

## 검증

- `/admin/files` 접속
- 상단 가운데 카드가 3개 항목만 표시되는지 확인
- 휴지통 목록 row 높이가 일정한지 확인
- row 클릭 시 상세 모달에서 현재 단계/첨부파일/메모 정보를 확인
