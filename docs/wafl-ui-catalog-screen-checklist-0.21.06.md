# WAFL UI Catalog Screen Checklist 0.21.06

## 목적

`/ui` 카탈로그와 기존 업무 화면을 연결하는 화면별 점검표를 추가했다.
이번 단계는 실제 업무 화면 리팩토링이 아니라, 각 화면에서 어떤 WAFL 공통 컴포넌트를 기준으로 점검해야 하는지 고정하는 작업이다.

## 추가된 화면별 점검 대상

- 작업지시서
- 원단·부자재 발주
- 운영 대시보드
- 협력업체
- 저장소
- 통계
- 멤버관리
- 개인설정

## `/ui` 반영 내용

- `Screen checklist · 기존 화면별 점검표` 섹션 추가
- 화면별 필수 WAFL 컴포넌트 badge 표시
- 화면별 점검 항목 3개씩 정리
- 누락 시 위험 요소 정리
- 향후 직접 `rounded / shadow / border / bg` 검색 결과를 어떤 컴포넌트로 치환해야 하는지 기준 연결

## 유지한 조건

- `/ui` 접근 제한 임시 해제 상태 유지
- WAFL debug outline 비활성 상태 유지
- 기존 업무 화면 로직 변경 없음

## 다음 단계

0.21.07에서는 이 점검표를 기준으로 실제 화면의 잔여 WAFL 누락 요소를 목록화하고, 우선순위 높은 부분만 소규모 보정한다.
