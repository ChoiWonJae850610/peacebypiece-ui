# 0.23.23 태블릿 패널 최소 너비 정책 정리

## 목적

태블릿 가로·분할 화면·브라우저 확대 상태에서 3패널 전환이 너무 이르게 발생해 중앙 상세와 우측 패널이 좁아지는 문제를 방지한다.

## 정책

- 3패널 전환 최소 폭은 1280px로 통일한다.
- 2패널 우측 패널 최소 너비는 300px로 유지한다.
- 3패널 최소 너비는 다음 공통 값으로 계산한다.
  - 목록: 300px
  - 상세: 420px
  - 우측: 286px
  - 패널 간격: 12px × 2
- 합산 최소 너비는 1030px이며, 화면이 더 좁으면 가로 스크롤로 구조를 보존한다.
- 레이아웃 결정 breakpoint와 내부 스타일 breakpoint를 분리한다.

## 영향 범위

- 공통 반응형 정책
- 공통 2패널·3패널 Workspace
- 작업지시서와 자재 발주 화면이 동일한 최소 너비 정책을 사용한다.

## 기능 영향

저장, 선택, API, DB, 권한 로직은 변경하지 않는다.
