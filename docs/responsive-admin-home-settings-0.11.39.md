# 0.11.39 모바일 관리자 홈/설정 화면 보정

## 목적

0.11.38에서 정리한 모바일/태블릿 점검 계획에 따라 고객관리자 홈과 환경설정 화면의 작은 폭 표시를 먼저 보정한다.

## 적용 범위

- 관리자 공통 Shell의 모바일 외곽 여백과 content gap 축소
- 관리자 상단 Topbar의 모바일 padding, radius, title 크기, summary badge 줄바꿈 보정
- `AdminSectionHeader` action 영역이 모바일에서 전체 폭을 사용할 수 있도록 보정
- 고객관리자 홈 카드 grid를 `sm`부터 2열로 전환
- 환경설정 메뉴 카드의 모바일 padding/radius/min-height 보정

## 변경하지 않은 범위

- 작업지시서 저장 정책
- 저장소/휴지통/R2 purge 흐름
- 회사 설정 저장 API
- 기준정보 관리 로직
- 멤버/권한/초대 로직
- DB schema

## 확인 기준

### `/admin`

- iPhone 폭에서 좌우 여백이 과도하지 않아야 한다.
- Topbar 제목과 summary badge가 화면 밖으로 밀리지 않아야 한다.
- 운영 관리 카드가 작은 폭에서는 1열, 태블릿 폭부터 2열 이상으로 자연스럽게 전환되어야 한다.

### `/admin/settings`

- 환경설정 안내 문구가 action 영역에서 화면 밖으로 밀리지 않아야 한다.
- 설정 메뉴 카드가 작은 폭에서는 1열, `sm` 폭부터 2열로 전환되어야 한다.
- 회사 설정 저장, 기준정보 진입, 요금제/계정/피드백 안내 모달 동작은 기존과 같아야 한다.

## 후속 작업

0.11.40에서는 `/admin/files` 저장소 화면의 table overflow, 휴지통 모달, 버튼 줄바꿈을 모바일 기준으로 점검한다.
