# 저장소관리 i18n / 하드코딩 정리 2차 (0.9.22410)

## 목적

저장소관리 화면과 시스템 저장소 실제 삭제 흐름에 남아 있던 사용자 노출 문구를 i18n dictionary 또는 presentation formatter 쪽으로 이동한다.

## 반영 내용

- 저장소 요약 카드의 요금제/파일 운영 문구를 `admin.filesSummary` i18n key로 정리했다.
- 휴지통 화면과 작업지시서 처리 모달에서 쓰는 문구 key를 `admin.filesList`에 보강했다.
- 작업지시서 휴지통 결과 메시지 조립을 `adminFiles.presentation.ts` formatter로 이동했다.
- `adminFiles.serverActions.ts`의 작업지시서 휴지통 성공/실패 문구 조립 책임을 축소했다.
- 권한 정책 설명의 고객 노출 용어를 `삭제 요청` 기준으로 정리했다.
- 공통 한국어 i18n에서 `작지`, `복구` 표현을 정리했다.
- 작업지시서 리오더 배지 문구에서 `작지` 표현을 제거했다.

없음.

4. 작업지시서 복원/선택 삭제 결과 메시지 확인
