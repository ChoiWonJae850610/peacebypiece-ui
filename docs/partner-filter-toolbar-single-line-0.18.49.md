# 0.18.49 협력업체관리 필터 toolbar 한 줄 배치 보정

## 목적

협력업체관리 검색/필터 영역이 PC와 태블릿 가로에서 두 줄로 내려가거나 왼쪽에 몰려 보이는 문제를 줄인다.

## 변경 기준

- 720px 이상에서는 검색창, 유형 select, 상태 select를 한 줄 grid로 배치한다.
- 검색창은 남는 폭을 차지한다.
- 유형/상태 select는 140~180px 범위의 안정적인 폭을 사용한다.
- 720px 미만에서는 검색창 전체 폭 + 유형/상태 2열 구조를 유지한다.

## 유지 사항

- 현재 목록 count 제거 상태 유지
- 협력업체 목록 wide table / compact card 구조 유지
- 검색/필터/정렬/등록/수정 모달 흐름 유지
- WorkspaceShell 변경 없음
- DB/API/협력업체 저장 흐름 변경 없음
