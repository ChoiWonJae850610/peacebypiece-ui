# 0.12.32 작업지시서 native 직접 그리기 확대/이동 1차

## 목적

직접 그리기 모달에서 캔버스의 일부 영역을 더 크게 보고 표시할 수 있도록 확대/축소와 손바닥 이동 도구를 추가한다.

## 반영 내용

- 손바닥 이동 도구 추가
- 확대 버튼 추가
- 축소 버튼 추가
- 화면 맞춤 버튼 추가
- 현재 확대율 표시
- 손바닥 이동 선택 시 색상/굵기/선 스타일 도구 비활성화
- 캔버스 변환은 화면 표시용 CSS transform으로 처리
- 저장되는 PNG는 기존 캔버스 원본 기준을 유지

## 정책

- 직접 그린 선/도형은 선택/이동하지 않는다.
- 손바닥 이동은 화면 보기 위치만 조정한다.
- 확대/축소도 화면 보기 상태만 바꾼다.
- 저장 시 기존 PNG 디자인 첨부 흐름을 유지한다.
- DB/R2/API/휴지통/purge 흐름은 변경하지 않는다.
- tldraw 고급 그리기 개발 플래그 정책은 변경하지 않는다.

## 테스트 포인트

1. 직접 그리기 모달을 연다.
2. 확대/축소 버튼으로 캔버스 표시 배율이 바뀌는지 확인한다.
3. 화면 맞춤을 누르면 100%와 중앙 위치로 돌아오는지 확인한다.
4. 손바닥 이동 도구를 선택한 뒤 드래그하면 보기 위치가 이동하는지 확인한다.
5. 손바닥 이동 후 펜/도형으로 다시 그릴 수 있는지 확인한다.
6. 확대 상태에서 펜/도형 좌표가 실제 pointer 위치와 크게 어긋나지 않는지 확인한다.
7. 저장 후 PNG가 기존 디자인 첨부 목록에 추가되는지 확인한다.
