# 0.19.70 개인 프로필 본인 탈퇴 요청 UI

## 목표
- 일반 멤버가 개인 프로필에서 탈퇴 요청을 보낼 수 있는 1차 UI/API를 추가한다.
- 고객사 관리자의 최종 처리 흐름은 기존 멤버관리 상태 변경 UI를 사용한다.

## 반영 내용
- `POST /api/me/profile/withdrawal` 추가.
- 개인 프로필 응답에 `memberStatus`를 포함한다.
- 개인 프로필에 탈퇴 요청 섹션을 추가한다.
- 일반 멤버는 `탈퇴 요청`을 보낼 수 있다.
- 고객사 관리자는 개인 프로필에서 탈퇴 요청을 보낼 수 없도록 안내만 표시한다.
- 이미 탈퇴 요청/탈퇴 완료 상태인 경우 중복 요청을 막는다.

## 변경하지 않은 범위
- 멤버관리 승인/초대/권한 저장 흐름.
- 담당자 후보 조회 로직.
- 작업지시서/R2/PDF/첨부/메모 흐름.
- DB schema.

## 테스트
1. 일반 멤버로 로그인한다.
2. 개인 설정 화면에서 탈퇴 요청 섹션을 확인한다.
3. 탈퇴 요청 버튼 클릭 후 확인창에서 승인한다.
4. 멤버관리 화면에서 해당 멤버 상태가 탈퇴 요청으로 바뀌는지 확인한다.
5. 고객사 관리자 계정에서는 탈퇴 요청 버튼이 비활성 또는 차단 안내로 표시되는지 확인한다.
