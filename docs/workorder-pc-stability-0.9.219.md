# 0.9.219 — 작업지시서 PC 화면 안정화 1차

## 목적

작업지시서 PC 화면의 3패널 구조를 유지하면서 좌측 목록, 중앙 상세, 우측 첨부/메모 패널의 스크롤과 여백을 안정화한다.

이번 버전은 원단/부자재 발주 화면을 구현하지 않는다. 현재 작업지시서 핵심 화면을 먼저 제품 화면처럼 테스트 가능한 상태로 정리하는 것이 목표다.

## 적용 범위

- PC 전용 작업지시서 레이아웃
- 좌측 작업지시서 목록 패널
- 중앙 작업지시서 상세 패널
- 우측 디자인/첨부/메모 패널

## 변경 기준

### 1. PC 3패널 균형

- 전체 여백을 줄여 실제 작업 공간을 확보한다.
- 좌측 목록은 기존보다 약간 넓혀 카드 텍스트 잘림을 줄인다.
- 우측 패널은 과도하게 넓지 않게 조정한다.
- 중앙 상세 패널은 최대 폭을 조금 넓혀 비용/발주/생산구성 섹션의 답답함을 줄인다.

### 2. 스크롤 안정화

- 중앙 상세와 우측 패널에 `scrollbar-gutter: stable`을 적용한다.
- 하단 padding을 추가해 마지막 섹션이 패널 하단에 붙거나 잘려 보이지 않게 한다.
- 좌측 목록 하단 padding을 유지·확대해 마지막 작업지시서 카드 확인성을 보장한다.

### 3. 섹션 간격 정리

- 중앙 상세 하단 섹션 간격을 소폭 정리한다.
- 우측 디자인/첨부/메모 카드 간격을 소폭 줄여 한 화면에서 확인 가능한 정보량을 늘린다.

2. 좌측 목록 마지막 카드까지 스크롤 가능한지 확인

4. 우측 디자인/첨부/메모 카드가 과하게 벌어지지 않는지 확인
