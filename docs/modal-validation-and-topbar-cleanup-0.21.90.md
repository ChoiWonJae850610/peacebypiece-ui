# 0.21.90 모달 입력 조건과 상단 헤더 정리

## 적용 범위
- 작업지시서 제작공정/추가공정 수정 모달
- 작업지시서 원단/부자재 추가·수정 모달
- 발주서 우측 발주 대상 패널 빈 상태
- 공통 AdminTopbar

## 변경 내용
- 제작공정 모달은 업체 선택 전 수량, 단가, 로스비 입력을 비활성화한다.
- 제작공정 적용 버튼은 업체 선택과 수량 0 이상 조건을 만족할 때만 활성화한다.
- 추가공정 모달은 항목과 업체 선택 전 적용 버튼을 비활성화한다.
- 추가공정 모달은 업체 선택 전 수량, 단가, 로스비 입력을 비활성화한다.
- 원단/부자재 모달은 자재명 입력 전 수량 입력을 비활성화한다.
- 원단/부자재 적용 버튼은 구분, 자재명, 수량 0 이상 조건을 만족할 때만 활성화한다.
- 발주서 우측 패널은 발주서 또는 자재 종류 미선택 시 작업지시서와 같은 빈 상태 안내를 표시한다.
- 공통 상단 헤더에서 WAFL 배지와 설명형 헬프 메시지 배지를 제거하고 고객사/버전/화면 제목만 유지한다.

## QA 포인트
- 제작공정/추가공정 모달에서 업체 선택 전 숫자 입력 필드가 비활성화되는지 확인한다.
- 추가공정 모달에서 항목 선택 후 업체 목록과 적용 버튼 조건이 맞는지 확인한다.
- 원단/부자재 모달에서 자재명 입력 전 수량 입력이 막히는지 확인한다.
- 발주서에서 아무 발주서도 선택하지 않았을 때 우측 패널이 비어 보이지 않는지 확인한다.
- 작업지시서, 발주서, 저장소, 협력업체 등 공통 상단 헤더에서 WAFL 배지와 헬프 메시지가 제거되었는지 확인한다.
