# 0.9.2219 — 성능/캐시/summary 기준 시스템관리자 이동

## 목표

고객관리자 통계정보 화면(`/admin/dashboard`)에서 개발/운영 기준 정보를 제거하고, 시스템관리자 콘솔(`/system`)에서 운영 기준으로 확인하도록 이동한다.

## 이동 대상

- 통계 API 캐싱 기준
- TanStack Query 도입 판단
- summary table / materialized view 검토 기준
- 성능 측정 기준

## 유지 기준

고객관리자 통계 화면은 생산 흐름, 생산 분석, 협력업체 성과, 리오더, 품질/납기 준비 상태 중심으로 유지한다.

시스템관리자 화면은 고객사 사용량, 저장소 purge, 성능/캐시/집계 전략을 함께 보는 운영 화면으로 확장한다.

## SQL DDL 필요 여부

불필요.

## 전체 리셋 필요 여부

불필요.

## 테스트 기준

1. `/admin/dashboard`에서 운영/개발 기준 영역이 보이지 않는지 확인한다.
2. `/system`에서 통계 운영 기준 영역이 표시되는지 확인한다.
3. 캐싱, summary table, 성능 측정 기준이 시스템관리자 콘솔에서 확인되는지 확인한다.
4. `APP_VERSION`이 `0.9.2219`인지 확인한다.
