# 0.19.61 작업지시서 첨부·디자인·메모 패널 UI 보정 1차

## 목표

작업지시서 우측 패널의 첨부 파일, 디자인, 작업메모 영역을 테스트 불가 상태에서도 안전하게 정리하기 위해 표시 밀도와 보조 문구만 보정했다.

## 적용 범위

- 첨부/디자인 패널 제목 옆 등록 개수 배지 표시
- 첨부/디자인 파일 항목에 파일 유형 보조 정보 추가
- 첨부/디자인 미리보기 버튼의 title/aria-label 보강
- 작업메모 입력 영역의 작성자 라벨 표시
- 작업메모 입력 글자 수 표시의 aria-live 보강
- 메모 댓글 토글 버튼의 잠금 상태 title 보강

## 제외 범위

- R2 업로드 흐름 변경 없음
- 첨부/디자인 삭제·미리보기·대표 지정 로직 변경 없음
- 메모 작성·수정·삭제 repository/API 흐름 변경 없음
- DB schema 변경 없음
- PDF 생성 로직 변경 없음

## 테스트 포인트

1. 작업지시서 상세 우측 패널 진입
2. 첨부 파일 패널 제목 옆 개수 배지 확인
3. 디자인 패널 제목 옆 개수 배지 확인
4. 첨부/디자인 파일명 아래 작성자와 파일 유형 표시 확인
5. 이미지/PDF/일반 파일 미리보기 버튼 동작 기존과 동일한지 확인
6. 대표 디자인 지정/삭제 버튼 동작 기존과 동일한지 확인
7. 작업메모 작성자 라벨과 글자 수 표시 확인
8. 메모 댓글 열기/닫기, 수정/삭제 동작 기존과 동일한지 확인
