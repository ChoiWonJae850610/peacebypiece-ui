# 0.11.60 통계 그래프 시각 크기 보정

## 목적

관리자 통계 화면에서 생산 구성 도너츠 차트와 막대형 Top 리스트가 화면 크기 대비 작게 보이는 문제를 보정한다.

## 반영 범위

- `/admin/dashboard` 생산 구성 탭
- `/admin/dashboard` 업체 성과 탭
- 공통 차트 컴포넌트 `AdminBasicDonutChart`, `AdminBasicBarChart`

## 변경 내용

- 도너츠 차트 영역 높이와 최대 폭을 확대했다.
- 도너츠 내부/외부 반지름을 키워 실제 그래프가 더 크게 보이도록 조정했다.
- 도너츠 중앙 총합 숫자의 크기를 키웠다.
- 비 compact 도너츠의 차트/범례 grid 폭을 확대했다.
- 막대 차트 영역의 최소 높이와 chart height를 키웠다.
- 막대 최대 너비를 늘려 Top 리스트가 더 또렷하게 보이도록 했다.
- 통계 카드의 최소 높이와 padding을 소폭 확대했다.

## 유지한 것

- 통계 API 구조 변경 없음
- 통계 집계 SQL 변경 없음
- 기간분석 topMode 유지 정책 변경 없음
- 생산 구성 누적 기준 정책 변경 없음
- Recharts 데이터 구조 변경 없음

## 확인 항목

1. `/admin/dashboard` 진입
2. 생산 구성 탭의 도너츠 차트 크기 확인
3. 생산 구성 탭의 품목 Top 리스트 막대 크기 확인
4. 업체 성과 탭의 생산량 Top 리스트와 납기/검수 table 표시 확인
5. 모바일 폭에서 차트가 화면 밖으로 밀리지 않는지 확인
