# 0.10.97 시스템관리자 홈 IA 정리

## 목적

`/system` 홈 화면을 운영자가 이해하기 쉬운 카드형 진입 화면으로 정리한다.

기존 홈은 통계 preview, API 준비 카드, placeholder, 샘플 고객사, 샘플 카테고리 규칙, 스토리지 버튼이 한 화면에 길게 섞여 있어 실제 운영 메뉴와 개발/점검 요소를 구분하기 어려웠다.

## 변경 기준

- `/system`은 시스템관리자 홈/메뉴 화면으로 유지한다.
- 실제 기능 화면은 개별 route로 이동한다.
- 샘플 고객사/샘플 카테고리 규칙 preview는 홈에서 제거한다.
- API path 나열과 placeholder 카드도 홈에서 제거한다.
- 통계/저장소/감사로그/기준정보는 각 전용 화면 링크로 이동한다.
- 홈은 `운영 메뉴`와 `관리·점검 메뉴` 두 구역만 가진다.

## 홈 카드 구역

### 운영 메뉴

- 고객사 관리
- 고객사 초대
- 저장소 관리
- 감사 로그
- 요금제·용량
- 기준정보 관리

### 관리·점검 메뉴

- 기준정보 seed 상태
- 기준정보 회귀점검
- 고객사 초기 기준정보
- 초대·권한 체크포인트
- 카테고리 규칙

## 후속 작업

- 시스템 통계가 필요하면 `/system/stats` 독립 route로 분리한다.
- 개발/진단성 화면은 `/system/dev` 또는 각 기준정보 하위 route로 분리한다.
- 공통 카드/배지/버튼 컴포넌트 표준화 단계에서 홈 카드도 공통 컴포넌트로 흡수한다.
