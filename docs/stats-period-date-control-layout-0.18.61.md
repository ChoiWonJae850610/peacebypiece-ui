# 0.18.61 통계정보 기간 컨트롤 날짜 선택 UI 보정

## 목표

통계정보 기간 분석 영역에서 시작일/종료일 날짜 선택 필드가 과하게 넓어지고, 달력 팝오버 내부 오른쪽 여백이 커 보이는 문제를 줄였습니다.

## 반영 내용

- 기간 분석 컨트롤을 날짜 선택 영역과 빠른 기간/액션 버튼 영역으로 분리했습니다.
- PC/태블릿 가로에서는 날짜 선택 영역을 좌측 compact width로 제한하고, 7일/30일/초기화/적용 버튼은 우측 정렬합니다.
- 모바일/좁은 폭에서는 날짜 선택 영역이 전체 폭을 쓰고, 버튼은 다음 줄에서 자연스럽게 감쌉니다.
- 초기화/적용 버튼은 icon-only compact action button으로 변경하되 `aria-label`, `title`, `sr-only` 텍스트를 유지했습니다.
- 달력 팝오버 폭을 calendar content에 맞게 축소하고 내부 calendar spacing을 더 compact하게 조정했습니다.

## 유지 사항

- 기간 선택 값 처리 흐름은 변경하지 않았습니다.
- 7일/30일/초기화/적용 링크 흐름은 유지했습니다.
- 통계 데이터 계산, 탭 전환, DB/API 흐름은 변경하지 않았습니다.
- WorkspaceShell 스크롤 구조는 변경하지 않았습니다.
