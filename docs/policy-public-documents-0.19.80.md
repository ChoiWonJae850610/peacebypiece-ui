# 0.19.80 정책/약관 고객 공개 화면 1차

## 목적

정책 문서 설계 기준을 고객사 관리자와 일반 멤버가 실제 화면에서 확인할 수 있는 조회 화면으로 연결한다.

## 적용 범위

- `/workspace/legal` 고객 공개 정책 화면 보강
- 환경설정의 약관·정책 카드에서 `/workspace/legal`로 이동할 수 있는 액션 추가
- 고객 공개 문서와 내부 운영 문서 분리 기준 유지
- 정책 버전/동의 이력 DB 구현은 후속 단계로 분리

## 고객 공개 문서 1차 목록

1. 이용약관
2. 개인정보처리방침
3. 요금·환불정책
4. 데이터 보관·삭제정책
5. 서비스 운영정책

## 구현 내용

- `lib/policies/customerPolicyDocuments.ts`에 고객 공개 정책 문서 메타데이터와 1차 본문 요약을 정의했다.
- `/workspace/legal`에서 문서 수, 필수 동의 문서 수, 초기 버전 상태를 요약 카드로 표시한다.
- 각 문서는 카테고리, 버전, 필수 동의 여부, 시행 준비 상태, 주요 조항 요약을 표시한다.
- 고객사 관리자 환경설정 약관·정책 패널에서 공개 문서 화면으로 이동하는 버튼을 제공한다.

## 제외 범위

- `policy_documents` DB 생성 없음
- `policy_versions` DB 생성 없음
- `policy_agreements` DB 생성 없음
- 고객사 승인 요청 전 필수 동의 저장 없음
- 시스템관리자 정책 문서 편집 UI 없음
- 비로그인 공개 정책 route 없음
