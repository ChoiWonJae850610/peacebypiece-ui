# 0.11.65 멤버관리 탭 콘텐츠 1차 보정

## 목적

0.11.64에서 멤버관리 화면을 요약 카드와 탭 구조로 1차 전환한 뒤, 각 탭의 실제 콘텐츠 길이와 업무 성격을 정리한다.

## 변경 요약

- 멤버관리 탭 네비게이션이 모바일/태블릿에서 잘리지 않도록 가로 스크롤 기준을 보강했다.
- 멤버 초대 탭 설명에서 승인 대기 흐름 표현을 제거하고, 초대 링크/QR 생성 업무에 집중하도록 정리했다.
- 승인 탭에서 기존 안내성 승인 workbench preview 영역을 제거하고, 실제 가입 신청/승인 대기 목록 중심으로 정리했다.
- 승인 탭에는 저장 전제 안내만 남겨 승인/거절 정책을 짧게 확인할 수 있게 했다.
- 전체 멤버 탭에서 역할 기본값 카드를 분리해 화면 길이를 줄였다.
- 역할 기본값 카드는 권한 관리 탭 상단으로 이동해 권한 카탈로그/매트릭스와 같은 맥락에서 확인하게 했다.

## 변경하지 않은 것

- 초대 생성 API는 변경하지 않았다.
- 가입 신청 승인/거절 API는 변경하지 않았다.
- 멤버 권한 저장 API는 변경하지 않았다.
- member_permissions 저장 구조는 변경하지 않았다.
- DB schema는 변경하지 않았다.

## 확인 항목

1. `/admin/members` 진입
2. 상단 요약 카드 확인
3. 탭 네비게이션이 PC/태블릿/모바일에서 잘리지 않는지 확인
4. `멤버 초대` 탭에서 초대 링크 생성과 초대 대기 목록 확인
5. `승인` 탭에서 가입 신청 목록과 승인/거절 버튼 확인
6. `전체 멤버` 탭에서 멤버 목록과 권한 저장 확인
7. `권한 관리` 탭에서 역할 기본값, 메인화면 카드 권한, 권한 카탈로그, 권한 매트릭스 확인
