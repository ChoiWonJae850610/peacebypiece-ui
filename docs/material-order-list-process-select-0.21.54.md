# 0.21.54 발주서 목록/공정 모달 보정

## 발주서 목록

- 새 발주서 생성 버튼 아래 divider 간격을 작업지시서 목록 기준에 가깝게 축소했다.
- 발주서 상태 뱃지는 작업지시서 카드와 같은 dot 포함 상태 뱃지 구조를 사용한다.
- 발주서 상태값은 작업지시서 상태 색상 토큰에 매핑한다.

## 작업지시서 공정 모달

- 제작 공정 모달의 업체 select는 업체 없음 상태를 빈칸이 아니라 `선택 가능한 업체 없음`으로 표시한다.
- 제작 항목을 변경하면 업체 선택값을 초기화한다.
- 추가 공정 모달의 항목 select 첫 옵션은 `선택 안함`이다.
- 추가 공정 항목을 선택하지 않은 경우 업체 select는 `항목을 먼저 선택` 상태로 비활성화된다.
- 추가 공정 항목을 변경하면 업체 선택값을 초기화하고 해당 공정 기준 업체 목록을 표시한다.
- 해당 공정에 가능한 업체가 없으면 `선택 가능한 업체 없음`으로 표시한다.
