# 0.18.65 통계정보 달력 footer 폭 보정

## 목적

통계정보 기간 분석에서 사용하는 공통 `AdminDateRangePicker`의 달력 팝오버 우측 빈 여백을 줄이고, 하단 비우기/완료 버튼 폭을 실제 날짜 grid 폭과 맞춥니다.

## 변경 내용

- 달력 grid 폭 기준 상수를 추가했습니다.
- DayPicker root/month/nav 폭과 날짜 grid 폭을 분리했습니다.
- 날짜 grid는 7열 x 20px 기준으로 고정했습니다.
- 하단 비우기/완료 버튼 영역을 날짜 grid 폭과 동일하게 맞췄습니다.
- 완료 버튼이 달력의 토요일 column 오른쪽 끝 기준 안쪽에 배치되도록 조정했습니다.
- 팝오버 padding과 footer gap을 축소했습니다.

## 유지 사항

- 시작일/종료일 선택 로직 유지
- 비우기/완료 동작 유지
- 바깥쪽 7일/30일/초기화/적용 동작 유지
- WorkspaceShell 스크롤 구조 변경 없음
- DB/API 통계 조회 흐름 변경 없음
