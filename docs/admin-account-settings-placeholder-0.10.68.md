# 0.10.68 고객관리자 계정 설정 placeholder 보정

## 목표

고객관리자 환경설정의 `계정 설정`을 단순 준비중 상태에서 읽기 전용 확인 화면으로 보정한다.

## 적용 내용

- `/admin/settings`의 `계정 설정` 카드 상태를 `읽기 전용`으로 변경했다.
- 계정 설정 모달에 회사 정보, 대표 로그인 이메일, 계정 상태, 개인 설정 분리 안내를 추가했다.
- 개인 설정은 조직 설정과 분리해 `/me/settings`로 이동하는 구조를 유지했다.
- 회사 정보 변경, 계정 비활성화 요청은 후속 연결 항목으로 남겼다.
- 표시 데이터와 정책 문구는 `lib/admin/settings/adminAccountSettingsPlaceholder.ts`로 분리했다.

## 정책

- 고객관리자 환경설정은 조직/회사 기준을 다룬다.
- 개인 표시명, 언어, 화면 색상 같은 개인 기준은 `/me/settings`에서 다룬다.
- 로그인 이메일은 OAuth provider 기준 식별값이므로 고객관리자가 임의 수정하지 않는다.
- 회사 정보 변경과 계정 비활성화는 시스템관리자 검토 흐름으로 확장한다.
- 실제 OAuth/session/company_members 조회는 후속 버전에서 연결한다.

## 제외한 내용

- 실제 Google OAuth 연결
- 계정 정보 DB 수정 API
- 회사 정보 변경 요청 저장
- 탈퇴/비활성화 요청 저장
- 시스템관리자 승인 처리
