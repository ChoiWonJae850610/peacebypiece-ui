# 0.18.60 통계정보 기간 컨트롤/달력 여백 정리

## 목적

통계정보 기간 분석 영역에서 달력 선택 UI와 기간 버튼이 차지하는 여백을 줄이고, 기간 컨트롤 영역의 책임을 별도 컴포넌트로 분리한다.

## 반영 내용

- `AdminStatsPeriodControls`를 추가해 기간 분석 제목, 날짜 범위 선택, 7일/30일/초기화/적용 버튼 조합을 분리했다.
- `AdminStatsDashboard`는 기간 분석 컨트롤의 직접 렌더링 대신 새 컴포넌트를 조합한다.
- `AdminDateRangePicker`의 닫힌 버튼과 열린 달력 패널의 padding, gap, 날짜 버튼 크기를 줄였다.
- 달력 UI의 직접 stone 색상 의존을 줄이고 theme semantic token 기반 색상으로 보정했다.
- 통계 데이터 계산, 기간 적용 href, 탭 전환, 그래프/카드 흐름은 변경하지 않았다.

## 확인 범위

- 통계정보 > 기간 분석 탭
- 사용자 지정 날짜 선택
- 7일 / 30일 / 초기화 / 적용 버튼
- PC / 태블릿 / 모바일에서 달력 버튼 높이와 여백
