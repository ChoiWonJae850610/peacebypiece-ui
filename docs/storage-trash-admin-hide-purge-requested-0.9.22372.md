# 0.9.22372 — 고객관리자 휴지통 purge_requested 항목 숨김 처리

## 목적

고객관리자 저장소 휴지통은 복구 가능한 항목을 관리하는 화면으로 유지한다. 작업지시서 영구삭제 요청이 완료되어 `purge_requested` 상태가 된 항목은 고객관리자 휴지통 기본 목록에서 숨기고, 시스템관리자 실제 삭제 후보 화면에서 최종 처리하도록 한다.

## 변경 기준

- `/admin/files` 휴지통 목록
  - 복구 가능한 작업지시서 항목만 표시
  - `delete_status = 'purge_requested'` 또는 `purge_status = 'purge_requested'` 작업지시서는 숨김
- `/system/storage-usage`
  - 기존대로 `purge_requested` 작업지시서 후보 표시
  - 시스템관리자가 최종 삭제 처리

## 검증

1. 작업지시서를 삭제한다.
2. 고객관리자 저장소 휴지통에서 작업지시서 항목이 보이는지 확인한다.
3. 고객관리자 저장소 휴지통에서 영구삭제 요청을 실행한다.
4. 해당 작업지시서가 `/admin/files` 휴지통 목록에서 사라지는지 확인한다.
5. 해당 작업지시서가 `/system/storage-usage` 실제 삭제 후보에 남는지 확인한다.
6. 시스템관리자에서 최종 처리 후 `purged` 상태로 전환되는지 확인한다.
