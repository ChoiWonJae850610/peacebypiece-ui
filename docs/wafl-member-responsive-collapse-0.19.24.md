# WAFL 멤버관리 responsive/접기 보정 0.19.24

## 목적

멤버관리 화면에서 멤버 목록과 초대 링크 목록이 모바일·태블릿에서 과하게 길어지고, 초대 링크 목록의 정렬과 카드 밀도가 어색한 문제를 보정한다.

## 적용 기준

- 초대 링크 목록은 상태, 생성일, 만료일, 작업만 표시한다.
- PC/태블릿/모바일 가로는 compact table 기준을 유지한다.
- 모바일 세로는 상태와 작업을 한 줄에 두고, 생성일과 만료일을 다음 줄에 배치한다.
- 멤버 목록은 compact/card 모드에서 기본 일부만 표시하고 더 보기/접기 버튼으로 확장한다.
- 초대 링크 목록도 기본 일부만 표시하고 더 보기/접기 버튼으로 확장한다.
- DB/API/R2/권한/초대 생성/초대 취소 흐름은 변경하지 않는다.

## 테스트 포인트

- `/workspace/members` PC에서 초대 링크 목록의 상태/생성일/만료일/작업이 가운데 정렬되는지 확인한다.
- 태블릿 세로에서 멤버 목록과 초대 링크 목록이 오른쪽으로 잘리지 않는지 확인한다.
- 모바일 세로에서 초대 링크 카드가 상태+작업 / 생성일+만료일의 2줄 구조로 보이는지 확인한다.
- 멤버 목록 더 보기/접기, 초대 링크 더 보기/접기가 정상 동작하는지 확인한다.
