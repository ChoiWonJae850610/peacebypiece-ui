# 0.18.30 저장소관리 휴지통 표시 밀도 재정리

## 목적

0.18.29에서 태블릿 스크롤은 안정화되었지만, 저장소관리 휴지통 목록이 PC에서는 너무 낮고 작게 보이고 태블릿에서는 항목 하나가 과하게 큰 카드처럼 보이는 문제가 있었다.

## 적용

- 저장소 요약 카드 높이를 줄여 휴지통 목록 노출 영역을 확보했다.
- 휴지통 목록 영역의 PC 최소 높이를 확보했다.
- 태블릿 이하 휴지통 row를 1열 풀 카드가 아니라 compact list-card에 가까운 다중 컬럼 흐름으로 조정했다.
- 태블릿 이하에서도 삭제 대상, 삭제 일시, 작업지시서, 유형, 크기 라벨이 row 내부에 표시되도록 보정했다.
- WorkspaceShell 스크롤 구조와 DB/API/R2/휴지통 처리 흐름은 변경하지 않았다.

## 유지 원칙

0.18.28에서 복구된 태블릿 스크롤 안정성은 건드리지 않는다. 저장소관리의 이번 변경은 FileStorageSummary, FileTrashSection, fileTrashSectionColumns, AdminTable의 표시 밀도 조정에 한정한다.
