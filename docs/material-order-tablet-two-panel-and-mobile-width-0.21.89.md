# 0.21.89 발주서 태블릿 2패널 및 모바일 폭 정렬

## 범위
- 원단·부자재 발주 화면 아이패드 미니급 가로 2패널 보정
- 모바일·태블릿 세로 공통 모바일 shell 상단/본문 폭 정렬
- 발주서 모바일 보조 버튼명 변경

## 변경 내용
- 발주서 태블릿 가로 2패널 조건을 viewport width/height 기준까지 함께 확인하도록 보강했다.
- 아이패드 미니급 가로에서는 발주서 목록을 드로어로 유지하고, 본문은 발주 상세와 발주 대상 패널 2개를 grid로 표시한다.
- 2패널 분기에서 이미 자체 shell을 가진 MaterialOrderDetailPanel/MaterialOrderAllocationPanel을 다시 WorkspacePanel로 감싸던 중첩 구조를 제거했다.
- WaflMobileShell의 topBar 영역에 본문과 동일한 좌우 padding을 적용해 작업지시서·발주서 모바일/태블릿 세로에서 헤더 카드와 본문 카드의 너비 기준을 맞췄다.
- 모바일 보조 FAB 및 sheet title의 `작지·자재` 문구를 `발주 대상`으로 변경했다.

## QA 포인트
- 아이패드 미니 가로: 발주서 목록은 드로어, 본문은 가운데 발주 상세 + 우측 발주 대상 2패널인지 확인한다.
- 태블릿 세로/모바일: 목록은 드로어, 본문은 단일 패널인지 확인한다.
- 작업지시서/발주서 모바일·태블릿 세로: 상단 헤더와 본문 카드 좌우 너비가 맞는지 확인한다.
