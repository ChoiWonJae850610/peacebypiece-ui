# 0.18.34 저장소관리 휴지통 액션/유형 UI 정리

## 목표

0.18.32~0.18.33에서 안정화한 컨테이너 폭 기준 반응형 구조와 WorkspaceShell 스크롤 구조를 유지하면서, 휴지통 영역의 액션 버튼과 유형 표시를 정리한다.

## 변경 기준

- 휴지통 액션 버튼은 제목 영역 우측에 정렬한다.
- 새로고침, 복원, 삭제, 비우기 버튼은 compact한 `icon + text` 형태로 표시한다.
- 유형 텍스트는 AppBadge 기반으로 표시한다.
- 유형 badge는 직접 색상값이 아니라 AppBadge semantic tone을 사용해 테마 CSS variable의 영향을 받게 한다.
- WorkspaceShell, DB/API/R2, 복원/삭제/비우기 기능 흐름은 변경하지 않는다.

## 유형 tone 매핑

- 작업지시서: `workorder`
- 디자인: `design`
- 문서/PDF: `document`
- 메모: `memo`
- 기타 파일: `file`

## 확인 포인트

- PC와 갤럭시탭 가로에서 휴지통 액션 버튼이 우측에 정렬되는지 확인한다.
- 좁은 화면에서 버튼이 우측 기준으로 자연스럽게 줄바꿈되는지 확인한다.
- 유형 badge가 테마 톤과 어긋나지 않는지 확인한다.
- 복원/삭제/비우기 동작은 기존과 동일해야 한다.
