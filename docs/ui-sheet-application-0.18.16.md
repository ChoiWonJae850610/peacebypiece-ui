# 0.18.16 AppSheet 적용 범위 확대

## 적용 위치

- `components/workorder/layout/TabletSplitLayout.tsx`

## 변경 내용

- 작업지시서 태블릿 화면에서 첨부·메모 보조 패널을 본문 아래 고정 영역으로 노출하던 구조를 AppSheet 기반 보조 패널로 전환했다.
- 태블릿 본문 하단에는 `첨부 · 메모` 열기 버튼만 고정한다.
- 작업지시서 선택이 바뀌면 본문 스크롤을 초기화하고 보조 패널 sheet를 닫는다.
- 모바일의 첨부·메모 sheet 흐름과 태블릿 보조 패널 흐름을 같은 AppSheet 계열로 맞췄다.

## 유지한 범위

- 작업지시서 상세 데이터, 첨부, 메모, R2, DB, 삭제/복원/purge 흐름은 변경하지 않았다.
- PC 화면 레이아웃은 변경하지 않았다.
- 모바일 화면의 기존 AppSheet 동작은 변경하지 않았다.

## 후속 점검

- 태블릿 실기기에서 하단 열기 버튼이 상세 입력 영역을 가리지 않는지 확인한다.
- Sheet가 열린 상태에서 첨부 업로드, 메모 저장, 이미지 미리보기 스크롤이 충돌하지 않는지 확인한다.
- 상세 화면 전환 시 sheet가 닫히는지 확인한다.
