# 0.9.22386 저장소 휴지통 컬럼 렌더링 리팩토링

## 목적

고객관리자 저장소 휴지통 화면의 테이블 컬럼 렌더링 책임을 `FileTrashSection.tsx`에서 분리했다.

## 반영 내용

- 휴지통 테이블 컬럼 정의를 `fileTrashSectionColumns.tsx`로 분리했다.
- 선택 컬럼 렌더링을 `TrashSelectionControl`로 분리했다.
- 대상 컬럼 렌더링을 `TrashTargetCell`로 분리했다.
- 삭제일시, 작업지시서, 유형, 크기 텍스트 셀 렌더링을 `CenterTextCell`로 정리했다.
- 기존 휴지통 row 생성, 버튼 동작, 복원/선택 삭제/비우기 로직은 변경하지 않았다.

## 확인 기준

- `/admin/files` 휴지통 목록이 기존과 동일하게 표시되어야 한다.
- 작업지시서 row와 첨부파일 row의 컬럼 정렬이 유지되어야 한다.
- 복원, 선택 삭제, 비우기 버튼 동작이 기존과 동일해야 한다.
- 작업지시서 묶음 row와 선삭제 첨부파일 row 표시가 유지되어야 한다.
