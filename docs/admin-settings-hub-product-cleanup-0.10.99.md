# 고객관리자 환경설정 화면 정리 — 0.10.99

## 목표

`/admin/settings` 화면에서 개발중 카드와 실제 저장 화면이 섞여 보이던 구조를 줄이고, 실제 저장되는 회사·화면 설정과 기준정보 설정을 우선 노출한다.

## 변경

- 기본 선택 메뉴를 `기준정보 설정`에서 `회사·화면 설정`으로 변경했다.
- 회사·화면 설정 메뉴에서는 `/api/admin/companies/current`를 조회해 실제 `CompanySettings`를 불러오고 기존 `AdminCompanySettingsForm`을 표시한다.
- 알림 정책 개발중 카드를 환경설정 메인 메뉴에서 제거했다.
- 요금제·저장공간, 계정 정보, 개발 건의는 모달 확인 영역으로 유지하되, 실제 저장 영역과 분리했다.
- 권한 관리는 멤버관리, 요금제·용량은 시스템관리자 기준으로 분리한다는 안내를 상단에 명시했다.

## 범위 제외

- DB schema 변경 없음
- package.json 변경 없음
- 기준정보 편집 UI 변경 없음
- 요금제/계정 실제 수정 API 연결 없음

## 확인

1. `/admin/settings` 진입 시 회사·화면 설정이 먼저 보이는지 확인한다.
2. 색상/언어 변경이 기존처럼 저장되는지 확인한다.
3. 기준정보 설정 카드를 누르면 기준정보 편집 화면이 보이는지 확인한다.
4. 요금제·저장공간, 계정 정보, 개발 건의 카드는 모달로 열리는지 확인한다.
