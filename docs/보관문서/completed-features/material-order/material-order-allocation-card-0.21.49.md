# 0.21.49 원단·부자재 발주 자재 선택 카드 정리

## 목표

원단·부자재 발주 화면의 우측 작업지시서 카드가 작업지시서 설명 카드가 아니라 발주할 자재를 고르는 카드로 보이도록 단순화한다.

## 기준

- 가운데 패널의 자재 종류가 원단이면 우측 패널에는 원단 항목만 표시한다.
- 가운데 패널의 자재 종류가 부자재이면 우측 패널에는 부자재 항목만 표시한다.
- 작업지시서 카드의 우측 상단 상태/남은 자재 뱃지는 제거한다.
- 구성/상태/발주 대기 자재 같은 라벨형 요약은 제거한다.
- 자재 항목은 자재명과 `발주수량/필요수량 단위` 중심으로 표시한다.

## 수량 표시

- `0/10 yd`: 아직 선택 또는 진행된 수량이 없는 상태이다.
- `5/10 yd`: 일부 수량이 진행 또는 선택된 상태이다.
- `10/10 yd`: 현재 작성 중 발주서에서 필요 수량을 채운 상태이다.

`10/10`은 작성 중 상태에서는 계속 표시한다. 사용자가 선택한 자재를 다시 뺄 수 있어야 하기 때문이다.
검토요청 이상으로 진행된 다른 발주서의 할당 수량은 우측 발주 대상에서 숨긴다. 반려/취소된 발주서는 활성 수량에 포함하지 않으므로 다시 발주 대상에 나타날 수 있다.

## 권한/목록 필터 확인

발주 화면의 우측 작업지시서 목록은 `/api/workorders/summary?status=active`를 통해 조회한다. 이 API는 workspace guard의 scope를 사용하며, 일반 멤버 scope에서는 workorder summary repository의 `assigned` visibility가 적용된다. 현재 repository 기준 assigned visibility는 `manager_id` 기준으로 필터링된다.
