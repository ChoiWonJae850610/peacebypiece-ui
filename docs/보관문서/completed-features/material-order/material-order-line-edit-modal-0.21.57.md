# 0.21.57 발주 품목 수정 모달 폭 보정

## 목적
- 발주 품목 수정 모달이 화면 전체 폭으로 보이지 않도록 WAFL 공통 모달 폭 기준에 맞춘다.
- 발주 품목 수정은 품목명, 단위, 수량, 단가, 금액만 다루므로 작업지시서 공정 수정 모달 수준의 폭이 적합하다.

## 변경 기준
- PC 기준 최대 폭을 520px로 제한한다.
- 기존의 가로로 과하게 긴 입력 레이아웃을 줄이고, 단위/수량/단가 입력은 좁은 모달 안에서 자연스럽게 줄바꿈되도록 한다.
- 금액은 읽기 전용 요약 박스로 유지한다.
- 모바일에서는 화면 폭에 맞춰 `w-full` 동작을 유지한다.
