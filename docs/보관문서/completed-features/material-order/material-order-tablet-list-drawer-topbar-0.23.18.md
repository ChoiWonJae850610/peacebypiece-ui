# 0.23.18 발주서 태블릿 목록 드로어 진입점 보완

## 목적

아이패드 미니 가로 등 `tabletTwoPanel` 레이아웃에서 발주서 목록 드로어는 존재하지만 상단 목록 버튼이 노출되지 않던 문제를 수정한다.

## 변경 내용

- 발주서 상단 메뉴 버튼 노출 조건을 `drawer` 전용에서 `drawer || tabletTwoPanel`로 확장했다.
- 모바일 세로와 태블릿 2패널에서 같은 발주서 목록 드로어 진입점을 사용한다.
- 데스크톱 3패널에서는 좌측 목록이 항상 보이므로 메뉴 버튼을 노출하지 않는다.
- 작업지시서 태블릿 화면과 동일하게 아이패드 미니 가로에서도 홈 옆 발주서 목록 버튼이 표시된다.

## 영향 범위

- UI 진입점과 반응형 노출 조건만 변경한다.
- 발주서 목록, 선택, 저장, API, DB 구조는 변경하지 않는다.

## 확인 항목

- 아이패드 미니 가로에서 홈 옆 발주서 목록 버튼 표시
- 버튼 클릭 시 발주서 목록 드로어 열림
- 발주서 선택 후 드로어 닫힘
- 태블릿 세로 및 모바일에서 기존 목록 버튼 유지
- 3패널 데스크톱에서 불필요한 목록 버튼 미노출
