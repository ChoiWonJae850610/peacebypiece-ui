# 0.20.34 모바일 FAB 액션 패턴 보정

## 범위
- 작업지시서 모바일 하단 액션을 첨부/디자인/메모 개별 버튼에서 단일 floating action button으로 전환한다.
- FAB를 누르면 기존 bottom sheet가 열리고 내부 segmented tab에서 첨부/디자인/메모를 전환한다.
- 원단·부자재 발주 모바일도 단일 FAB로 발주 도구 sheet를 연다.
- 발주서 목록은 하단 버튼이 아니라 상단 모바일 메뉴 버튼에서 left drawer로 연다.
- 원단·부자재 발주 도움말 문구를 모바일에서 줄인다.
- 직접 그리기 모바일 landscape 차단 안내가 화면 중앙에 가로 방향으로 정상 표시되도록 보정한다.

## 유지
- PC 3패널 구조 유지
- 태블릿 구조 유지
- 상태전환 로직 변경 없음
- 권한 로직 변경 없음
- API/DB/R2/첨부/메모/휴지통/purge 흐름 변경 없음

## 확인 포인트
- 작업지시서 모바일 상세에서 우하단 관련 정보 FAB가 보이는지 확인한다.
- 관련 정보 FAB를 누르면 첨부/디자인/메모 탭 sheet가 열리는지 확인한다.
- 원단·부자재 발주 모바일 상단 메뉴 버튼에서 발주서 목록 drawer가 열리는지 확인한다.
- 원단·부자재 발주 모바일 우하단 발주 도구 FAB에서 발주서/자재/PDF·납기 탭 sheet가 열리는지 확인한다.
- 모바일 직접 그리기 상태에서 가로로 돌렸을 때 안내 메시지가 세로로 잘리지 않는지 확인한다.
