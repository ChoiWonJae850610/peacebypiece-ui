# 0.18.70 통계정보 선택 기간 표시 범위 보정

## 변경 목적

통계정보 화면에서 선택 기간 badge가 생산 구성/업체 성과 탭과 상단 누적 지표까지 기간 기준으로 오해되게 보이는 문제를 줄였습니다.

## 반영 내용

- 운영 누적 지표 섹션의 선택 기간 badge 제거
- 작업흐름분석 섹션의 선택 기간 badge는 기간 분석 탭에서만 표시
- 생산 구성/업체 성과 탭은 전체 누적 기준이라는 화면 의미를 유지
- 기간 분석 탭의 기간 선택/적용 흐름은 유지
- 통계 계산, DB/API 조회, WorkspaceShell 구조 변경 없음
