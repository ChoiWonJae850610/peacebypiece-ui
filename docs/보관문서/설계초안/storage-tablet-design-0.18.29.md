# 0.18.29 저장소관리 태블릿 디자인 재정리

## 기준
- 0.18.28은 Galaxy Tab/iPad 세로·가로 스크롤이 정상화된 기준점이다.
- 이번 버전은 WorkspaceShell의 스크롤 구조를 다시 건드리지 않고 저장소관리 화면의 디자인 밀도만 조정한다.

## 변경
- 저장소 요약 영역을 태블릿에서 2열 카드 흐름으로 정리했다.
- PC 2xl 이상에서는 기존 3열 요약 구성을 유지한다.
- 저장소 상단 타이틀 크기와 요약 카드 높이를 낮춰 태블릿에서 과하게 길어지는 느낌을 줄였다.
- 휴지통 action 버튼은 태블릿 이하에서 2열 그리드로 배치해 터치 영역을 명확하게 했다.
- 휴지통 row는 태블릿 이하에서 카드형 터치 row처럼 보이도록 border/radius/shadow를 부여했다.
- PC 2xl 이상에서는 기존 표형 row에 가깝게 유지한다.

## 금지/주의
- 태블릿 이하에서 내부 스크롤 컨테이너를 다시 도입하지 않는다.
- h-dvh, overflow-hidden, overflow-auto 기반 고정 패널은 2xl 이상 PC 전용으로 유지한다.
- DB/API/R2/첨부/메모/휴지통/purge 흐름은 변경하지 않았다.
