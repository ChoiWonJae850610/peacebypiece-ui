# 0.22.05 모바일 발주 품목 추가 모달 레이어 안정화

## 변경 요약
- 발주 품목 추가 모달에서 모바일 입력 후 닫기/추가 버튼이 응답하지 않는 현상을 줄이기 위해 해당 모달의 body fixed scroll lock을 해제했습니다.
- 발주 품목 추가 모달의 z-index와 touch-action을 모달 자체에서 보강했습니다.
- 터치 pointerdown 단계에서는 blur만 수행하고 confirm은 click 단계에서 처리하도록 되돌려 터치 이벤트 소실 가능성을 줄였습니다.

## 확인 필요
- 모바일 세로에서 수량/단가 입력 후 추가 버튼 동작
- 모바일 세로에서 수량/단가 입력 후 닫기 버튼 동작
- 아이패드 미니 가로/세로에서 추가/닫기 버튼 동작
- 갤럭시탭 모달 위치와 추가/닫기 버튼 동작
- npm run build
