# 0.20.53 태블릿 우측 패널 및 모달 포커스 보정

## 목적

- iPad mini 가로처럼 3패널을 안정적으로 표시하기 어려운 태블릿 폭에서 목록은 드로어로 숨기고, 본문과 우측 보조 패널은 유지한다.
- 작업지시서 우측 보조 패널은 PC와 같은 첨부/디자인/메모 세로 섹션 흐름을 사용한다.
- 모바일/태블릿 모달에서 숫자 입력 후 메모 필드로 포커스가 강제로 이동하거나 닫기/적용 버튼 터치가 느려지는 증상을 줄인다.
- 담당자 변경 모달은 중복 open 요청을 무시한다.

## 변경 요약

- `WorkOrderSidePanelTabletView`를 PC 우측 패널과 같은 세로 섹션 스택으로 정리했다.
- 태블릿 2패널 작업지시서의 본문/우측 보조 패널 스크롤 컨테이너에 `overscroll-contain`과 stable scrollbar 기준을 적용했다.
- 모바일/태블릿 coarse pointer 환경에서는 모달 open 시 첫 focus 강제 이동과 close 시 이전 버튼 focus 복원을 하지 않도록 조정했다.
- 재고 수정 모달의 숫자 입력 필드는 Enter/Done 입력 후 현재 필드를 blur하고 다른 필드로 강제 이동하지 않는다.
- 분류 모달은 상위/중분류를 펼침/접힘으로만 제어하고 하위 값을 자동 선택하지 않는다.
- 담당자 변경 모달 open 요청은 이전 open 요청이 처리 중이면 무시한다.

## 영향 제한

- 상태전환/권한/API/DB/R2/첨부/메모/휴지통/purge 흐름은 변경하지 않았다.
- 직접그리기 최상단 레이어 보정 흐름은 유지한다.
