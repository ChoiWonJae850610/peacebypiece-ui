# WAFL 0.19.30 — FilterBar 공통화와 협력업체 스크롤 보정

## 목적

0.19.30은 본문 섹션 공통화 이후 반복되는 검색/필터 영역을 `WaflFilterBar` 기준으로 올리고, 협력업체관리 화면에서 업체 목록 영역에 마우스 휠 스크롤이 막히는 문제를 보정한다.

## 적용 원칙

- 검색/필터 영역의 배경, 테두리, radius, padding은 WAFL theme token을 사용한다.
- 화면별 필터바 색상/간격을 직접 지정하지 않고 `WaflFilterBar`에 위임한다.
- 협력업체관리 화면은 고정 레이아웃보다 페이지 스크롤 레이아웃을 우선 사용한다.
- 데이터/API/DB 흐름은 변경하지 않는다.

## 확인 지점

- `/workspace/partners`에서 업체 목록 위에 마우스를 올려도 페이지 스크롤이 동작해야 한다.
- 협력업체관리 검색/필터 영역이 멤버관리 검색/필터 영역과 같은 계열로 보여야 한다.
- 멤버관리 검색/필터 영역은 기존 기능을 유지하면서 WAFL 공통 필터바 스타일을 따라야 한다.
