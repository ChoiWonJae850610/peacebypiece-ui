# 0.19.17 멤버 초대 목록 icon-only action과 카드 비율 보정

## 목적

0.19.16에서 초대 링크 컬럼을 제거한 뒤 남은 멤버 초대 목록의 작업 버튼과 카드 비율을 보정한다.

## 변경 기준

- 초대 목록 컬럼은 `상태 / 생성일 / 만료일 / 작업`만 유지한다.
- 작업 버튼은 글자 없이 아이콘만 표시한다.
- disabled 상태에서도 아이콘은 흐린 상태로 남긴다.
- 초대 목록 카드는 줄이고, 직원 초대 생성 카드는 넓힌다.
- 초대 링크 전체 URL 또는 suffix는 테이블에 표시하지 않는다.

## 빌드 실패 보정

0.19.16 build fail 로그에서 `MemberInvitationSortKey`에 없는 `"link"` 비교가 남아 TypeScript 오류가 발생했다.

- 제거: `sortState.key === "link"`
- 유지: `status`, `createdAt`, `expires` 정렬

## 테스트 위치

- `/workspace/members`

## 확인할 것

- 작업 버튼이 빈 원형 버튼처럼 보이지 않는지
- 복사/취소 아이콘이 active/disabled 상태 모두에서 보이는지
- 초대 목록 우측 카드 폭이 줄고 좌측 초대 생성 카드가 넓어졌는지
- 생성일은 초 단위까지 표시되는지
- 만료일은 날짜만 표시되는지
- 초대 링크는 더 이상 노출되지 않는지

## 변경 금지

- 멤버 초대 생성 API
- 멤버 초대 취소 API
- 링크 복사 실제 동작
- 멤버 승인/권한 저장
- DB/API/R2 흐름
