# 0.18.43 협력업체관리 필터 select 전환

## 목적
협력업체관리 상단의 요약 카드와 필터 toolbar를 더 단순하게 정리한다.

## 반영 내용
- 요약 카드 5개를 4개로 축소했다.
- `사용중` 요약 카드는 제거하고, `전체 업체` 카드의 보조 문구에 사용/미사용 수를 표시한다.
- 유형 필터 버튼 그룹을 AppSelect 기반 단일 선택 필터로 전환했다.
- 사용 상태 버튼 그룹을 AppSelect 기반 단일 선택 필터로 전환했다.
- 검색/유형/상태/현재 목록 count가 한 toolbar 안에서 정렬되도록 보정했다.
- wide table wrapper에 overflow-hidden을 적용해 table header/row 배경이 rounded border 밖으로 어색하게 보이는 문제를 줄였다.

## 유지 사항
- 협력업체 row 안의 다중 유형 badge 표시는 유지한다.
- 기존 검색/필터/정렬/등록/수정 모달 흐름은 유지한다.
- WorkspaceShell 스크롤 구조는 변경하지 않았다.
- DB/API/협력업체 저장 흐름은 변경하지 않았다.
