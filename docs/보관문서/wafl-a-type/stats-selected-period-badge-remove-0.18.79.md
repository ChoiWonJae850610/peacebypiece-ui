# 0.18.79 통계정보 선택 기간 badge 제거

## 목적
- 기간 분석의 날짜 필드와 적용 버튼만으로 기간 기준을 판단하게 하고, 별도 선택 기간 badge로 인한 혼동을 제거한다.

## 반영
- 작업흐름분석 섹션 actions 영역에서 선택 기간 badge 렌더링을 제거했다.
- 선택 기간 label 및 active label 계산을 제거했다.
- 생산 구성 / 업체 성과 / 기간 분석 탭 전환, 7일/30일 프리셋, 날짜 적용 흐름은 변경하지 않았다.

## 미변경
- 통계 계산 로직
- DB/API 조회 흐름
- WorkspaceShell 스크롤 구조
