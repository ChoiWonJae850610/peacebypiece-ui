# 0.18.47 협력업체관리 필터 toolbar 균형 보정

## 목적

협력업체관리 필터 toolbar가 PC/태블릿 가로에서 왼쪽에 몰려 보이고, 일부 태블릿 폭에서 검색/유형/상태 select 폭 합계가 화면을 넘는 문제를 보정한다.

## 변경 기준

- 1120px 이상에서는 검색창이 남는 폭을 차지하고, 유형/상태 select는 우측 고정 폭 영역에 배치한다.
- 720px 이상 1120px 미만에서는 검색창을 한 줄 전체 폭으로 두고, 유형/상태 select 묶음은 다음 줄 우측 정렬을 유지한다.
- 720px 미만에서는 검색창 전체 폭 + 유형/상태 2열 구조를 유지한다.

## 유지 사항

- 현재 목록 count 제거 상태 유지
- 검색/유형/상태 필터 동작 유지
- 협력업체 목록 wide table / compact card 구조 유지
- WorkspaceShell 변경 없음
- DB/API/협력업체 저장 흐름 변경 없음
