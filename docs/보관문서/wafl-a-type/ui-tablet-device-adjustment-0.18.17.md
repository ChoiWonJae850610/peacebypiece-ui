# UI tablet device adjustment 0.18.17

## 목적

0.18.16에서 작업지시서 태블릿 보조 패널을 AppSheet로 전환한 뒤, 태블릿 실기기 기준으로 하단 버튼과 보조 패널 흐름을 점검하기 위한 보정이다.

## 변경 범위

- 작업지시서 태블릿 레이아웃
  - 하단 첨부·메모 열기 버튼에 safe-area bottom 간격을 반영했다.
  - 상세 본문 하단 padding을 조금 늘려 하단 고정 버튼과 마지막 입력 영역이 겹치지 않도록 했다.
  - AppSheet 내부 하단 padding에도 safe-area 값을 반영했다.

- 원단·부자재 발주 태블릿 레이아웃
  - 태블릿에서 자재 선택 영역을 상세 패널 탭 전환 대신 AppSheet로 열도록 정리했다.
  - 발주서 목록 + 발주 상세의 2열 흐름은 유지하고, 자재 선택은 보조 작업으로 분리했다.
  - 발주서 선택 변경 시 자재 선택 sheet가 닫히도록 했다.

## 비변경 범위

- PC 3분할 원단·부자재 레이아웃은 변경하지 않았다.
- 모바일 원단·부자재 자재 선택 sheet 흐름은 유지했다.
- 작업지시서 저장, 첨부, 메모, R2, 휴지통, purge 흐름은 변경하지 않았다.
