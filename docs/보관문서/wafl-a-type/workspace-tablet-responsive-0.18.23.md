# 0.18.23 일반멤버 홈/태블릿 관리자 화면 보정

## 목적
- 일반멤버 업무 홈에 약관·정책 조회 카드를 추가한다.
- 태블릿 세로/가로 화면에서 저장소관리·협력업체관리·멤버관리 화면이 스크롤되지 않거나 고정 높이에 갇히는 문제를 줄인다.

## 변경 범위
- 일반멤버 홈 카드
  - 약관·정책 카드를 일반멤버 전용 추가 카드로 노출한다.
  - `/workspace/legal` 조회용 정책 허브 화면을 추가한다.
  - 시스템관리자 정책 편집 기능과는 분리한다.

- WorkspaceShell
  - `fixed-md` 모드의 고정 레이아웃 적용 기준을 `md`에서 `lg`로 늦췄다.
  - 태블릿 영역에서는 페이지 스크롤이 가능하도록 유지하고, 데스크톱 폭부터 내부 fixed layout을 유지한다.

- 저장소관리
  - 태블릿에서 전체 화면 스크롤이 막히지 않도록 `md:overflow-hidden`을 `lg:overflow-hidden` 기준으로 조정했다.
  - 휴지통 영역은 태블릿에서 최소 높이를 확보하고 데스크톱 폭에서만 내부 고정 스크롤로 전환한다.

- 협력업체관리
  - 태블릿에서 전체 페이지 스크롤이 가능하도록 fixed/overflow 기준을 `lg` 이상으로 조정했다.
  - 목록 flex 고정도 데스크톱 폭부터 적용한다.

- 멤버관리
  - 멤버관리 대시보드 루트에 overscroll과 하단 여백을 보강했다.
  - 멤버관리 탭 섹션의 최소 너비/overflow를 보정해 태블릿에서 탭과 목록이 밀리는 현상을 줄였다.

## 유지한 범위
- 통계정보 화면은 이번 사용성 확인에서 우선 문제가 낮으므로 변경하지 않았다.
- DB/API/R2/첨부/메모/휴지통/purge 흐름은 변경하지 않았다.
- 멤버 초대/승인/권한 저장 로직은 변경하지 않았다.

## 다음 확인
- 태블릿 세로 저장소관리: 페이지 하단까지 스크롤 가능 여부
- 태블릿 가로 저장소관리: 요약/휴지통 영역이 과도하게 잘리지 않는지
- 태블릿 협력업체관리: 목록 하단까지 접근 가능 여부
- 태블릿 멤버관리: 멤버 관리 탭과 목록이 스크롤 가능한지
- 일반멤버 홈: 약관·정책 카드가 마지막 카드로 노출되는지
