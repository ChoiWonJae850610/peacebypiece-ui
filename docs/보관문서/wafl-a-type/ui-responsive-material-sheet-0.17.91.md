# 0.17.91 원단·부자재 모바일 자재 선택 Sheet 전환

## 목적

모바일 원단·부자재 화면에서 자재 선택 패널이 본문 탭으로 길게 노출되는 문제를 줄이기 위해, 자재 선택 영역을 `AppSheet` 하단 패널로 분리한다.

## 변경 기준

- PC: 기존 3분할 유지
- Tablet: 기존 목록 + 상세/자재 탭 구조 유지
- Mobile: 발주서 / 상세 탭은 유지하되, 자재 선택은 하단 Sheet로 전환

## 적용 내용

- 모바일 상단 전환 탭에서 `자재`를 누르면 본문 탭 전환 대신 Sheet를 연다.
- 모바일 상세 화면 상단에 `자재 선택 열기` 버튼을 추가한다.
- Sheet는 `side="bottom"`, `size="full"` 기준으로 열리며 내부에는 기존 `MaterialOrderAllocationPanel`을 그대로 사용한다.
- 저장, 상태 전환, 자재 계산, API/DB/R2 흐름은 변경하지 않는다.

## 후속 점검

- 실제 태블릿/모바일에서 Sheet 높이와 내부 스크롤 체감 확인
- 하단 버튼 또는 닫기 버튼이 모바일 브라우저 UI에 가리지 않는지 확인
- 자재 선택 후 Sheet를 자동으로 닫을지 여부는 실제 사용감 확인 후 결정
