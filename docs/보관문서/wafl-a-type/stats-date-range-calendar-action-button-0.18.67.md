# 0.18.67 통계정보 달력 하단 버튼 형태 복구

## 목적
- 공통 `AdminDateRangePicker` 달력 팝오버의 비우기/완료 하단 버튼 형태를 복구합니다.
- 0.18.66에서 줄어든 달력 우측 여백 구조는 유지합니다.

## 반영
- 달력 하단 액션 버튼 전용 공통 class 상수 추가
- 비우기/완료 버튼에 `h-8`, `min-h-8`, `w-8`, `min-w-8`, `rounded-full`, `p-0`을 명시
- `AdminButton`의 기본 `min-height`가 남아 버튼이 캡슐형으로 깨지는 문제를 줄임
- 완료 버튼 체크 아이콘 stroke 기준 보정

## 유지
- 기간 선택/비우기/완료/적용 로직 변경 없음
- WorkspaceShell 변경 없음
- DB/API 통계 조회 흐름 변경 없음
