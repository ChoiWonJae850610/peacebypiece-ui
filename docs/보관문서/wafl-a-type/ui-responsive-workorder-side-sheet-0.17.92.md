# 0.17.92 작업지시서 모바일 첨부·메모 Sheet 전환

## 목적

모바일 작업지시서 화면에서 상세 본문 아래에 첨부·메모 패널이 길게 이어져 보이는 구조를 줄이고, 상세 확인과 보조 정보 확인을 분리한다.

## 반영 범위

- 모바일 작업지시서 레이아웃만 변경
- PC 작업지시서 레이아웃 변경 없음
- 태블릿 작업지시서 레이아웃 변경 없음
- 첨부/메모/R2/업로드/미리보기/삭제 로직 변경 없음

## 변경 내용

- 모바일 하단에 `첨부 파일 · 작업메모` 고정 버튼 추가
- 버튼 클릭 시 `AppSheet` bottom 패널로 첨부·메모 패널 표시
- 작업지시서 선택이 변경되면 Sheet를 자동으로 닫음
- 기존 `WorkOrderSidePanel`과 내부 첨부/메모 컴포넌트는 그대로 사용

## 확인 항목

- 모바일 작업지시서 상세에서 첨부·메모가 본문 아래에 바로 길게 노출되지 않는지 확인
- 하단 버튼 클릭 시 Sheet가 열리는지 확인
- Sheet 안에서 메모 등록/수정/삭제가 기존과 동일하게 동작하는지 확인
- Sheet 안에서 첨부 업로드/미리보기/삭제가 기존과 동일하게 동작하는지 확인
- 작업지시서를 다른 항목으로 바꾸면 Sheet가 닫히는지 확인
