# 0.18.74 통계정보 기간 버튼 패밀리 정리

## 목적

통계정보 기간 분석 영역의 7일/30일 프리셋 버튼과 초기화/적용 아이콘 버튼이 서로 다른 버튼 계열처럼 보이는 문제를 줄인다.

## 반영 내용

- 관리자 공통 아이콘 액션 버튼의 base class를 분리했다.
- 짧은 텍스트 프리셋용 `AdminCompactActionButton`을 추가했다.
- 7일/30일 버튼을 `AdminCompactActionButton`으로 연결했다.
- 7일/30일 버튼은 텍스트를 유지하되 초기화/적용 버튼과 같은 높이, radius, border, surface tone을 공유한다.
- 7일/30일 active 상태와 프리셋 토글 동작은 유지한다.

## 유지 사항

- 기간 프리셋/적용 로직 변경 없음
- 달력 팝오버 구조 변경 없음
- 통계 계산/DB/API 흐름 변경 없음
- WorkspaceShell 변경 없음
