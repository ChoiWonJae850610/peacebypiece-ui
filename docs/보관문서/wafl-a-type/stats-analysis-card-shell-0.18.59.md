# 0.18.59 통계정보 분석 카드 shell 정리

## 목적

통계정보 분석 영역의 카드 렌더링 기준을 정리하기 위해 공통 분석 카드 shell을 도입했습니다.

## 반영 내용

- `AdminStatsAnalysisCardShell` 추가
- 기간 상위 카드와 bar list 카드가 같은 카드 shell을 사용하도록 정리
- 생산 구성 카드와 업체 지연/불량 카드도 같은 shell 기준으로 연결
- 통계 데이터 계산, 탭 전환, 기간 선택, 그래프 데이터 흐름은 변경하지 않음

## 유지한 범위

- WorkspaceShell 스크롤 구조 변경 없음
- DB/API 통계 조회 흐름 변경 없음
- 저장소관리/협력업체관리/멤버관리 화면 변경 없음
