# 0.18.69 통계정보 업체 성과 테이블 공통화

## 목표

통계정보의 업체 성과 탭에서 업체별 납기·검수 지표가 테이블 컬럼 안에 정렬되지 않고 아래쪽 badge처럼 떨어져 보이는 문제를 보정했습니다.

## 반영 내용

- 업체 성과 지표 표시를 `FactoryPerformanceTable`로 분리했습니다.
- 저장소관리/협력업체관리에서 사용하는 `AdminResponsiveTableShell`과 `adminResponsiveTableStyles` 기준을 재사용했습니다.
- 업체명/생산건수, 납기 지연율, 검수/불량률을 같은 row 안의 3개 column으로 배치했습니다.
- 기존 tooltip과 badge tone은 유지했습니다.
- 통계 계산, 탭 전환, 기간 선택, DB/API 조회 흐름은 변경하지 않았습니다.

## 확인 포인트

- 통계정보 > 업체 성과 탭에서 업체명, 납기 지연율, 검수/불량률이 같은 행에 정렬되는지 확인합니다.
- badge가 하단으로 떨어져 보이지 않는지 확인합니다.
- 저장소관리/협력업체관리 테이블과 외곽선, 헤더, row 밀도가 같은 계열로 보이는지 확인합니다.
