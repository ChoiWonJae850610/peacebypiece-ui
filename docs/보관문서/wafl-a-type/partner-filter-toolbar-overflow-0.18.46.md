# 0.18.46 협력업체관리 필터 toolbar overflow 보정

## 목적

협력업체관리 검색/필터 toolbar가 태블릿 일부 폭에서 가로로 넘치거나, PC/태블릿 가로에서 세 컴포넌트가 왼쪽에 붙어 보이는 문제를 보정한다.

## 변경 기준

- 1120px 이상: 검색 영역과 필터 영역을 좌우 2구역으로 분리한다.
- 1120px 미만: 검색 영역은 한 줄 전체 폭을 사용하고, 유형/상태 select는 다음 줄에서 2열로 배치한다.
- 720px 이상 1120px 미만: 유형/상태 select 묶음은 오른쪽 정렬을 유지하되 화면 폭을 넘지 않도록 max-width를 둔다.
- 모바일/좁은 폭: 검색 전체폭 + 유형/상태 2열 구조를 유지한다.

## 유지 사항

- 현재 목록 count 제거 상태 유지
- 요약 카드 4개 구조 유지
- 협력업체 목록 wide table / compact card 구조 유지
- 검색/필터/정렬/등록/수정 모달 흐름 유지
- WorkspaceShell 변경 없음
- DB/API/협력업체 저장 흐름 변경 없음
