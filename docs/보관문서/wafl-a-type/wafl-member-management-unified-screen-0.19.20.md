# 0.19.20 멤버관리 단일 화면 구조 정리

## 목적

멤버관리 화면의 `멤버 초대` / `멤버 관리` 탭을 제거하고, 초대 생성·초대 링크 목록·가입 승인 대기·멤버 목록을 하나의 화면 흐름으로 통합한다.

## 변경 범위

- 멤버관리 상단 segmented tab 제거
- 초대 생성/초대 링크 목록 섹션 상시 표시
- 승인 대기/멤버 목록 섹션 상시 표시
- 기존 초대 생성, 링크 복사, 초대 취소, 가입 승인/거절, 멤버 권한 상세 모달 동작 유지

## 유지 기준

- 초대 링크 URL/suffix 컬럼은 노출하지 않는다.
- 초대 목록 작업 버튼은 icon-only 구조를 유지한다.
- 멤버 초대 생성과 초대 목록은 상단 업무 흐름으로 둔다.
- 가입 신청/멤버 목록은 같은 화면 하단에서 이어서 관리한다.

## 변경 금지

- 멤버 초대 생성 API
- 멤버 초대 취소 API
- 링크 복사 실제 동작
- 가입 신청 승인/거절 API
- 멤버 권한 저장 API
- DB schema / R2 / 권한 로직

## 테스트 기준

- `/workspace/members` 진입 시 탭이 없어야 한다.
- 초대 생성 카드와 초대 링크 목록이 바로 보여야 한다.
- 아래에 가입 승인/멤버 목록이 이어서 보여야 한다.
- 초대 생성, 링크 복사, 초대 취소, 가입 승인/거절, 멤버 상세 모달 기능이 기존과 동일해야 한다.
