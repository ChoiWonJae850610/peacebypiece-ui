# 0.19.64 원단·부자재 발주 버튼/상태 표시 보정 1차

## 범위

- 원단·부자재 발주 화면의 버튼 title / aria-label / disabled 안내를 보정했다.
- 상태 변경 중 진행 단계 버튼에 처리 중 안내와 spinner 상태를 연결했다.
- 발주 품목 삭제, 공급처 재조회, 새 발주서 생성, 자재 선택 버튼의 의미를 명확히 했다.
- 모바일/태블릿 자재 선택 버튼 문구를 짧게 정리했다.

## 변경하지 않은 것

- 발주 생성 API
- 발주 상태 변경 로직
- 자재 할당 계산식
- PDF 생성/저장 흐름
- R2/DB schema
- package.json / package-lock.json

## 테스트 포인트

1. 발주서 생성 버튼의 생성 중 표시가 기존 동작과 동일한지 확인한다.
2. 발주 진행 단계 버튼 클릭 중 처리 중 안내가 표시되는지 확인한다.
3. 작성중이 아닌 발주서에서 품목 수정/삭제가 잠기고 안내 문구가 표시되는지 확인한다.
4. 모바일/태블릿에서 자재 선택 버튼이 패널을 기존처럼 여는지 확인한다.
5. 이미 선택됨/발주 완료/진행중 자재 버튼이 기존처럼 비활성 유지되는지 확인한다.
