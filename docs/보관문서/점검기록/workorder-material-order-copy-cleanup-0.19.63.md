# 0.19.63 원단·부자재 발주 표시/문구 정리 1차

## 범위

- 원단·부자재 발주 화면의 목록/상세/자재 선택 패널에 남아 있던 모호한 문구를 정리했다.
- 발주서 목록, 공급처 선택, 발주 품목 empty state, 작업지시서 자재 선택 안내 문구를 사용자 관점의 표현으로 보정했다.
- 상태/버튼/검색 placeholder 문구를 발주 업무 흐름에 맞게 정리했다.

## 변경하지 않은 범위

- 발주서 생성 로직
- 발주 상태 변경 로직
- 자재 할당 계산식
- 공급처 조회 API
- 작업지시서 후보 조회 API
- DB schema
- PDF 생성/첨부 흐름

## 테스트 포인트

1. 원단·부자재 발주 화면 진입 시 페이지 설명 문구 확인
2. 발주서 목록의 검색 placeholder와 empty/search/loading 문구 확인
3. 발주 상세 패널에서 발주 기본정보/자재 종류/공급처 문구 확인
4. 발주 품목이 없을 때 empty state 문구 확인
5. 작업지시서 자재 선택 패널에서 title, 검색 placeholder, empty/search 문구 확인
6. 기존 발주서 생성, 상태 변경, 자재 선택, 라인 삭제 동작이 그대로 유지되는지 확인
