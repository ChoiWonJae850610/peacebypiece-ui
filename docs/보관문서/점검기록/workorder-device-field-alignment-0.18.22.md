# 0.18.22 작업지시서 디바이스 필드 정합성 보정

## 목적

PC 작업지시서의 생산 공장/공정 정보 영역을 기준으로 태블릿·모바일 작업지시서 상세 화면의 필드 구조를 맞춘다.

## 반영 내용

- 태블릿·모바일 생산 공장 카드에 샘플/메인/재작업 구분 선택 필드를 노출한다.
- 태블릿·모바일 생산 공장 카드에 수량, 단가, 장당 로스비 입력 필드를 노출한다.
- 태블릿·모바일 생산 공장 카드에 금액 읽기 값을 노출한다.
- 태블릿·모바일 생산 공장 영역에 외주 공정 row를 함께 표시한다.
- 태블릿·모바일 생산 공장 영역에서 외주 공정 추가 버튼을 노출한다.
- 태블릿·모바일 제품 구성 영역에서는 외주 공정 섹션을 제거하고 제품 구성/자재 구성 중심으로 맞춘다.

## 유지한 흐름

- PC 작업지시서 상세 구조는 변경하지 않는다.
- 기존 EditableValue 저장 흐름을 유지한다.
- 기존 factoryOptions, orderTypeOptions, outsourcingVendorOptionsById, outsourcingProcessOptions를 그대로 사용한다.
- DB/API/R2/첨부/메모/휴지통/purge 흐름은 변경하지 않는다.
- 납기 입력 필드는 태블릿·모바일 작업지시서 상세에 다시 노출하지 않는다.
