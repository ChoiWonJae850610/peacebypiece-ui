# 작업지시서 버튼/상태 표시 보정 1차 (0.19.60)

## 목적
- 작업지시서 진행 단계 버튼의 표시 우선순위와 접근성 문구를 정리한다.
- 상태 변경 처리 중 또는 현재 단계 잠금 상태에서 버튼이 비활성화된 이유를 화면/보조기술에 함께 전달한다.

## 변경 범위
- 진행 단계 공통 패널의 action metadata 확장
- 작업지시서 desktop/tablet/mobile 진행 단계 버튼 title/aria-label/disabled reason 추가
- 반려 버튼이 첫 번째로 노출되는 경우에도 주요 진행 버튼이 primary로 보이도록 tablet/mobile 기준 보정
- ko/en i18n 문구 추가

## 비변경 범위
- 상태 전환 조건 변경 없음
- 권한 판정 변경 없음
- DB/API/R2/PDF/첨부/메모 흐름 변경 없음
