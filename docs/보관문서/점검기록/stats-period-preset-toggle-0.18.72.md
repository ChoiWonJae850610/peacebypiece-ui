# 0.18.72 통계정보 기간 프리셋 토글 보정

## 목적
- 통계정보 기간 분석의 7일/30일 버튼이 눌렸는지 명확히 보이도록 pending preset active 상태를 분리합니다.
- 7일/30일 버튼은 실제 조회를 즉시 실행하지 않고 날짜 필드만 세팅합니다.
- 같은 프리셋 버튼을 다시 누르면 날짜 필드와 active 상태를 초기화합니다.

## 변경
- `AdminStatsDashboard`에 `activePeriodPresetKey` 상태를 추가했습니다.
- 7일/30일 버튼 active 표시는 서버의 현재 적용 기간이 아니라 사용자가 방금 선택한 필드 프리셋 기준으로 표시합니다.
- 날짜를 수동 변경하면 프리셋 active 상태가 해제됩니다.
- 기존 적용 버튼 기준 통계 반영 흐름은 유지합니다.

## 유지
- 달력 UI와 팝오버 구조는 변경하지 않았습니다.
- 통계 계산/DB/API 조회 흐름은 변경하지 않았습니다.
- WorkspaceShell 스크롤 구조는 변경하지 않았습니다.
