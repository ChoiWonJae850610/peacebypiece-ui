# 0.18.80 통계정보 분석 섹션 header depth 정리

## 목적

통계정보 화면의 작업흐름분석 제목 영역이 탭 영역과 분리되어 한 depth 더 깊게 보이는 문제를 줄입니다.

## 변경

- 작업흐름분석 섹션 제목을 “분석 통계”로 변경했습니다.
- 설명 문구를 “생산 구성, 업체 성과, 기간별 흐름을 확인합니다.”로 축약했습니다.
- 제목/설명과 생산 구성·업체 성과·기간 분석 탭을 같은 header row 안에 배치했습니다.
- 기존 nested AdminSection header + 내부 panel 구조를 단일 AdminCard + 내부 header/content 구조로 단순화했습니다.

## 유지

- 생산 구성 / 업체 성과 / 기간 분석 탭 전환 흐름 유지
- 기간 선택 / 7일·30일 프리셋 / 적용 흐름 유지
- 통계 계산 및 DB/API 조회 흐름 변경 없음
- WorkspaceShell 스크롤 구조 변경 없음
