# 작업지시서 PDF 표시/문구 보정 1차 (0.19.62)

## 목적

작업지시서 첨부 패널에서 발주서 PDF의 저장 상태를 사용자가 명확히 구분할 수 있도록 표시 문구와 접근성 문구를 보정했다.

## 변경 범위

- 첨부 패널의 발주서 PDF 상태 표시 문구 정리
- 발주서 PDF가 이미 첨부된 경우의 상태 안내 추가
- 발주서 PDF가 누락된 경우의 재저장 안내 문구 보정
- 발주서 PDF 재저장 버튼의 `title` / `aria-label` 추가
- ko/en i18n 동시 반영

## 제외 범위

- PDF 생성 API 변경 없음
- PDF template 변경 없음
- R2 저장 경로 변경 없음
- 첨부 등록/삭제/미리보기 로직 변경 없음
- 권한/상태 전환 로직 변경 없음

## 확인 포인트

1. 발주서 PDF가 없는 발주요청 완료 작업지시서에서 누락 안내와 재저장 버튼이 보이는지 확인한다.
2. 발주서 PDF가 이미 첨부된 작업지시서에서 저장 완료 안내가 보이는지 확인한다.
3. 첨부파일 목록의 PDF 미리보기/다운로드 동작이 기존과 동일한지 확인한다.
4. 재저장 버튼 클릭 후 기존 PDF 생성/첨부 흐름이 유지되는지 확인한다.
