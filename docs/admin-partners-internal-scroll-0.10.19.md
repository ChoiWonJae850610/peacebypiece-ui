# 0.10.19 — 협력업체 관리 내부 스크롤 구조 재보정

## 목표

`/admin/partners` 화면에서 브라우저/페이지 전체가 스크롤되며 상단 요약 카드와 검색/필터 카드가 사라지던 문제를 보정한다.

## 변경 기준

- PC 화면에서는 고객관리자 헤더 아래 콘텐츠 영역을 고정형으로 사용한다.
- 상단 요약 카드 5개, 안내 문구, 검색/필터는 화면에 유지한다.
- 업체 목록 테이블 카드 내부만 세로 스크롤되도록 한다.
- 모바일에서는 기존처럼 자연 페이지 스크롤을 유지한다.

## 적용 내용

- `AdminShell`에 `contentMode="fixed-md"` 옵션을 추가했다.
- `/admin/partners`에서만 `fixed-md` 모드를 사용한다.
- PC 기준 콘텐츠 프레임은 `overflow-hidden`으로 고정하고, 모바일에서는 `overflow-y-auto`를 유지한다.
- `PartnerMasterSection`은 PC 기준 `h-full`, `max-h-full`, `overflow-hidden` 구조를 유지한다.
- 요약 카드와 검색/필터 카드 간격을 소폭 줄여 목록 영역이 확보되도록 했다.
- 업체 목록 행 높이를 소폭 줄였다.

## 영향 범위

- 협력업체 저장/수정/필터 로직 변경 없음
- DB schema 변경 없음
- 감사 로그 흐름 변경 없음
- 다른 관리자 화면은 기본 `scroll` 모드를 유지함
