# 회사 파일 미리보기 UI 보정 2차 (0.20.24)

## 목적

환경설정의 회사 파일 영역에서 대표 이미지와 사업자등록증의 등록 상태를 더 명확하게 확인할 수 있도록 미리보기 카드와 모달 표시 밀도를 보정한다.

## 적용 범위

- 대표 이미지 미리보기 카드의 표시 높이와 여백을 확대했다.
- 이미지 파일은 작은 아이콘처럼 보이지 않도록 카드 내부에서 더 넓은 프리뷰 영역을 사용한다.
- PDF/문서 파일은 카드 안에서 문서형 미리보기 블록으로 표시한다.
- 기본 카드에서는 긴 원본 파일명을 노출하지 않고, 파일 유형·용량·등록일만 compact badge로 표시한다.
- 원본 파일명은 미리보기 모달 상단에서만 확인하도록 정리했다.
- 각 슬롯 하단에 업로드 가능 파일 유형과 용량 안내를 별도 안내 박스로 표시한다.

## 변경하지 않은 범위

- 회사 파일 R2 업로드 흐름
- 회사 파일 DB 저장 흐름
- 회사 파일 preview API
- 작업지시서 첨부/디자인/메모 흐름
- 휴지통/purge 흐름
- DB schema

## 확인 필요

- 대표 이미지가 카드에서 너무 작게 보이지 않는지 확인
- 사업자등록증 PDF 미리보기 모달이 기존처럼 열리는지 확인
- 원본 파일명이 기본 카드에는 숨겨지고 모달에서만 보이는지 확인
- 파일 유형/용량/등록일 badge가 모바일 폭에서 과도하게 깨지지 않는지 확인
