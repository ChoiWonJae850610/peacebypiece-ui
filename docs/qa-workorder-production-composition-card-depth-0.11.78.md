# 0.11.78 작업지시서 생산구성 표시/카드 depth 보정

## 목적

작업지시서 생산구성에서 `선택 안함` 상태의 거래처/외주처 선택 UX를 명확하게 하고, PC 화면에서 원단/부자재와 외주공정 table 폭이 지나치게 좁아지는 문제를 완화한다.

## 반영 내용

- 원단/부자재 구분이 `선택 안함`이면 거래처 후보에 빈 줄 대신 `등록된 업체 없음`을 표시한다.
- 외주공정 공정이 `선택 안함`이면 외주처 후보에 `등록된 업체 없음`을 표시한다.
- `등록된 업체 없음`은 선택 불가 옵션으로 유지한다.
- 0.11.77 build error 원인이던 원단 기본 단위 문자열 타입을 `MATERIAL_UNIT` 상수 기준으로 보정한다.
- 생산구성 내부의 원단/부자재, 외주공정 section wrapper depth를 줄여 table 좌우 폭을 확보한다.
- 원단/부자재 table의 단위 column 폭을 늘려 `야드`, `개` 표시가 잘리지 않도록 보정한다.
- 외주공정 table의 단가기준 column 폭을 늘려 `장당` 표시가 잘리지 않도록 보정한다.
- 발주정보 outer card padding을 소폭 줄여 중첩 카드 느낌을 완화한다.

## 확인 항목

- `npm run build` 로컬 실행.
- 원단/부자재 구분을 `선택 안함`으로 둔 뒤 거래처 셀을 클릭하면 `등록된 업체 없음`이 표시되는지 확인.
- 외주공정을 `선택 안함`으로 둔 뒤 외주처 셀을 클릭하면 `등록된 업체 없음`이 표시되는지 확인.
- 원단 선택 시 단위가 타입 오류 없이 기본 단위로 설정되는지 확인.
- 부자재 선택 시 단위가 타입 오류 없이 기본 단위로 설정되는지 확인.
- PC 작업지시서 화면에서 원단/부자재 단위가 `야`처럼 잘리지 않는지 확인.
