# 0.9.223791 저장소 휴지통 목록 정렬/버튼/문구 보정

## 목적
고객관리자 저장소 휴지통 화면의 버튼명, 목록 정렬, 선택/상세 안내 문구를 실제 운영자가 이해하기 쉬운 표현으로 정리한다.

## 반영 내용
- 휴지통 상단의 `전체 선택` 버튼을 제거했다.
- `복구` 버튼명을 `복원`으로 변경했다.
- `영구 삭제` 버튼명을 `선택 삭제`로 변경했다.
- `비우기` 버튼을 추가했다.
- `비우기` 클릭 시 확인 모달을 표시한다.
  - 문구: `휴지통의 모든 항목을 삭제 요청하시겠습니까?`
  - 버튼: `예` / `아니오`
- 휴지통 대상 컬럼에서 작업지시서 row는 좌측 정렬하고, 첨부파일 row는 기존 가운데 정렬을 유지했다.
- 작업지시서와 함께 삭제된 첨부/메모 복원 안내 문구를 `작업지시서와 첨부된 파일/메모가 함께 복원됩니다.`로 통일했다.
- 삭제된 작업지시서에 연결되어 파일 단독 복원이 제한되는 경우 `해당 작업지시서를 찾을 수 없습니다.`로 표시한다.

## 테스트 포인트
1. `/admin/files` 휴지통 진입
2. 상단에 `전체 선택` 버튼이 없는지 확인
3. `복원`, `선택 삭제`, `비우기` 버튼명이 보이는지 확인
4. `비우기` 클릭 시 확인 모달이 열리는지 확인
5. 확인 모달에 `휴지통의 모든 항목을 삭제 요청하시겠습니까?` 문구와 `예` / `아니오` 버튼이 보이는지 확인
6. 작업지시서 row 대상명이 좌측 정렬되는지 확인
7. 첨부파일 row는 기존 정렬이 과도하게 틀어지지 않는지 확인
8. 작업지시서 묶음 복원 안내 문구와 파일 단독 복원 제한 문구가 사용자용 표현으로 보이는지 확인
