# 0.19.18 멤버 초대 아이콘 버튼 표시 및 카드 비율 보정

## 목적
멤버 초대 목록의 작업 버튼이 빈 원형 버튼처럼 보이는 회귀를 보정하고, 초대 링크 컬럼 제거 후 남은 여백에 맞춰 좌우 카드 비율을 재조정한다.

## 반영
- 복사/취소 작업 버튼은 글자 없는 icon-only 구조를 유지한다.
- active/disabled 상태 모두에서 Copy/Ban 아이콘이 식별 가능하도록 아이콘 크기, stroke, 색상, disabled opacity를 보정한다.
- disabled 상태는 클릭만 차단하고 아이콘은 흐리게 남겨 행별 작업 위치를 유지한다.
- 초대 목록 카드는 더 좁게, 직원 초대 생성 카드는 더 넓게 보이도록 2xl 레이아웃 비율을 조정한다.
- 초대 링크 URL/suffix 컬럼은 다시 추가하지 않는다.

## 확인
- `/workspace/members` 멤버 초대 탭에서 작업 버튼이 빈 원형으로 보이지 않아야 한다.
- 취소됨/만료됨/사용됨 row에서도 disabled 아이콘이 흐리게 보여야 한다.
- 좌측 직원 초대 생성 카드가 우측 초대 목록 카드보다 넓게 보여야 한다.
