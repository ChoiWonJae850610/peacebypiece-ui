# 환경설정 기준정보 헤더·토글·테마 정렬 보정 0.20.21.4

## 목적

환경설정 기준정보 화면에서 헤더 높이, 선택 색상, 사용 여부 표시가 탭마다 다르게 보이던 문제를 보정한다.

## 반영 기준

- 기준정보 본문 헤더는 환경설정의 다른 탭과 같은 `WaflSectionPanel` standard density를 사용한다.
- 생산품 유형의 선택된 1차/2차/3차 항목은 상단 선택 탭과 같은 `--pbp-surface-selected`, `--pbp-brand-muted`, `--pbp-brand-primary` 계열을 사용한다.
- 생산품 유형, 단위 표준, 외주공정 유형의 사용 여부 표시는 같은 `AdminUsageToggle` 시각 체계를 사용한다.
- 단위 표준과 외주공정 유형은 현재 직접 수정 흐름이 아니므로 read-only 토글로 표시한다.
- 단위 표준과 외주공정 유형 목록에는 `표기명` 컬럼을 추가해 화면 밀도를 보완한다.
- 요청 패널의 `요청하기` 버튼은 비활성 상태에서도 글자를 읽을 수 있도록 대비를 보정한다.

## 화면 구조

### 생산품 유형

- 1차 유형 / 2차 유형 / 3차 유형 3열 구조 유지
- 선택 항목은 WAFL selected token 기반으로 표시
- 사용/미사용은 조작 가능한 `AdminUsageToggle` 사용

### 단위 표준

- 이름
- 표기명
- 사용처
- 사용 여부
- 우측 유형 추가 요청 패널

### 외주공정 유형

- 이름
- 표기명
- 사용처
- 사용 여부
- 우측 유형 추가 요청 패널

## 제외 범위

- DB schema 변경 없음
- 기준정보 저장 API 변경 없음
- R2/첨부/메모/휴지통/purge 변경 없음
