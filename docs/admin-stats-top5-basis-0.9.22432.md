# 0.9.22432 관리자 통계 TOP5 데이터 기준 정교화

## 목적

기간 요약 카드 클릭에 따라 바뀌는 TOP5 카드의 제목, 수치 단위, 데이터 기준을 일치시킨다.

## 적용 기준

1. 완료 작업지시서
   - 제목: 발주수량 상위 TOP5
   - 기준: 선택 기간 안에 완료된 작업지시서의 발주수량 합계
   - 단위: pcs
   - orders.quantity 합계가 0인 항목은 제외한다.

2. 리오더
   - 제목: 리오더 TOP5
   - 기준: 선택 기간 안의 2차 이상 리오더 작업지시서 수
   - 단위: 건
   - reorder_round > 1 또는 parent_spec_sheet_id가 있는 항목을 리오더 후보로 본다.
   - work_order_kind = 'rework' 항목은 리오더 기준에서 제외한다.

3. 불량 작업지시서
   - 제목: 불량 작업지시서 TOP5
   - 기준: 불량이 1개 이상 기록된 작업지시서 수
   - 단위: 건
   - 현재 DB에는 별도의 불량 수량 전용 필드가 명확하지 않으므로 status = 'rejected', is_rework = true, work_order_kind = 'rework' 중 하나라도 해당하면 불량 작업지시서로 집계한다.

## DB 변경

없음.

## 후속 정교화 후보

불량 수량을 실제 수량 기준으로 분석하려면 검수 결과 또는 불량 수량 전용 컬럼/테이블 기준을 0.9.22453에서 확정해야 한다.
