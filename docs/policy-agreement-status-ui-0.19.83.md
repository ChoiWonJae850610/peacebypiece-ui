# 0.19.83 정책 동의 상태 표시/동의 버튼 UI 1차

## 목표

`/workspace/legal` 화면에서 현재 로그인 사용자의 필수 정책 동의 상태를 확인하고, 필수 약관·정책 전체 동의를 저장할 수 있게 한다.

## 적용 범위

- 현재 정책 상태 조회 API: `GET /api/policies/current`
- 필수 정책 동의 저장 API: `POST /api/policies/agreements`
- 고객 공개 정책 화면: `/workspace/legal`

## 반영 내용

- 필수 약관·정책 동의 상태 패널 추가
- 필수 문서 수, 동의 완료 수, 남은 필수 동의 수 표시
- 문서별 동의 완료/동의 필요 상태 표시
- 필수 약관·정책 전체 동의 버튼 추가
- 동의 완료 후 버튼 비활성화 및 상태 즉시 갱신

## 제외 범위

- 로그인 후 강제 차단 UI
- 고객사 승인 요청 전 필수 동의 차단
- 정책 변경 재동의 강제 흐름
- 비로그인 공개 약관 페이지

## 테스트 기준

1. `/workspace/legal` 진입 시 정책 문서와 동의 상태 패널이 표시된다.
2. 필수 동의 전에는 남은 필수 동의 수가 표시된다.
3. 필수 약관·정책 전체 동의 버튼을 누르면 동의 이력이 저장된다.
4. 저장 후 모든 필수 문서가 동의 완료 상태로 표시된다.
5. 새로고침 후에도 동의 완료 상태가 유지된다.
