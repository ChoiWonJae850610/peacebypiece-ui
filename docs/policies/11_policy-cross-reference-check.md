## 6. 현재 확정된 핵심 정책값

### 6.1 도메인/이메일
- wafl.co.kr을 서비스 도메인으로 가정한다.
- 대표/운영 이메일은 hello@wafl.co.kr 예정으로 한다.
- 도메인 구매 및 Google Workspace 개설은 아직 완료 전 상태로 본다.
- 초기 이메일 발송은 Google Workspace 기반으로 설계한다.

### 6.2 요금제
- Trial: 7일 무료, 100MB, 멤버 3명, 데이터 내보내기 제한
- Lite: 월 9,900원, 500MB, 멤버 3명, 데이터 내보내기 월 1회
- Flow: 월 19,900원, 1.5GB, 멤버 10명, 데이터 내보내기 월 3회
- Studio: 월 39,900원, 5GB, 멤버 30명, 데이터 내보내기 월 10회
- 추가 저장소: 1GB당 월 7,000원
- 가격은 VAT 포함으로 표시한다.

### 6.3 해지/결제 실패/장기 미납
- 해지는 다음 결제일부터 정기결제를 중단한다는 의미다.
- 해지 취소는 다음 결제일 전날 23:59까지 가능하다.
- 결제 실패 즉시 유료 업무 기능을 제한한다.
- 결제 실패 상태만으로 고객사 데이터를 즉시 삭제하지 않는다.
- 미납 30일: 장기 미납 안내
- 미납 60일: 데이터 삭제 예정 안내
- 미납 90일: 시스템관리자 삭제 가능 대상으로 표시

### 6.4 저장소/삭제
- 삭제된 데이터는 고객사 휴지통에서 원칙적으로 30일 보관한다.
- 고객사 휴지통 비우기는 허용한다.
- 30일 경과 항목은 시스템관리자 삭제관리의 영구 삭제 가능 대상으로 표시한다.
- 초기에는 자동 영구 삭제하지 않고 시스템관리자가 최종 확인 후 삭제한다.
- export zip 다운로드 가능 기간은 7일이다.

### 6.5 재동의
재동의 필요 변경:
- 요금
- 환불
- 해지
- 데이터 삭제
- 개인정보 처리
- 데이터 내보내기 제한
- 결제 실패 또는 미납에 따른 서비스 제한
- 고객사 데이터 보관기간
- 파일 영구 삭제 기준

재동의 불필요 변경:
- 오탈자
- 문장 표현 정리
- 조항 번호 정리
- 연락처 변경
- 서비스명 표기 정리
- 법령명 또는 링크 정리

고객 공개 문서는 v1.0부터 시작한다.
