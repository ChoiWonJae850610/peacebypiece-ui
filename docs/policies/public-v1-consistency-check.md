## 5. 정합성 판단

### 5.1 서비스명
고객 공개 기준 서비스명은 WAFL이다.

고객 공개 문서에서는 PeaceByPiece와 WAFLOW를 제거해야 한다.

현재 공개 초안에서 PeaceByPiece 또는 WAFLOW가 발견되는 경우 고객 공개 v1.0 확정 전 반드시 WAFL로 치환한다.

### 5.2 이메일
고객 공개 이메일 역할은 다음 기준으로 통일한다.

- 고객지원: support@wafl.co.kr
- 대표/일반 문의: hello@wafl.co.kr
- 자동 발신/알림: no-reply@wafl.co.kr
- 개인정보/정책 문의: privacy@wafl.co.kr
- 결제 문의: billing@wafl.co.kr

### 5.3 핵심 수치
다음 수치는 고객 공개 문서 전체에서 동일하게 유지한다.

- Trial: 7일, 100MB, 멤버 3명, 데이터 내보내기 제한
- Lite: 9,900원, 500MB, 멤버 3명, 데이터 내보내기 월 1회
- Flow: 19,900원, 1.5GB, 멤버 10명, 데이터 내보내기 월 3회
- Studio: 39,900원, 5GB, 멤버 30명, 데이터 내보내기 월 10회
- 추가 저장소: 1GB당 월 7,000원
- 저장소 80% 도달: 안내
- 저장소 100% 초과: 새 파일 업로드 제한
- 고객사 휴지통: 30일
- export zip 다운로드 가능 기간: 7일
- 장기 미납: 30일 안내, 60일 삭제 예정 안내, 90일 삭제 가능
- 서비스 전체 종료: 원칙적으로 30일 전 사전 고지
- 해지 취소: 다음 결제일 전날 23:59까지

현재 고객 공개 초안은 변환 연습용이므로 일부 문서에 다음 표현이 남아 있을 수 있다.

- 공개 버전: v1.0 초안
- 내부 기준 버전

- 사업자명 확정 필요
- 대표자명 확정 필요
- 사업장 주소 확정 필요
- 사업자등록번호 확정 필요
- 고객 공개 전 검토용 초안
- 법무 검토 여부
- PG사 확정 필요
- 처리위탁사 확정 필요

실제 고객 공개 v1.0 확정본에서는 위 표현을 제거하고 실제 확정값으로 치환한다.

### 7.5 파일 및 저장소 보관 정책

### 7.6 데이터 내보내기 정책
- 포함/제외 대상 재검토
- 멤버정보 범위 이름/전화번호/역할 재확인
- 로그인 필요 다운로드 링크 방식 확인

### 7.7 서비스 운영 정책
- 고객지원 응답 시간 미보장 문구 확인
- 서비스 전체 종료 30일 전 고지 문구 확인
- 장애 보상 원칙적 미제공 문구 법무 검토

### 7.8 정책 변경 고지 및 동의 정책
- 재동의 필요 항목 확인
- 미동의 시 기능 제한 범위 확인
