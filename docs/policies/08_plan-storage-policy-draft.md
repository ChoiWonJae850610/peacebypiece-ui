# 요금제 및 저장소 정책 초안

## 문서 상태
- Policy Version: P-0.91.8
- Based App Version: 0.15.73
- 문서 성격: 법무 검토 전 초안
- 적용 대상: PeaceByPiece/WAFLOW 요금제, 저장소, 멤버 수, 데이터 내보내기 제한

## 1. 목적
이 문서는 WAFLOW 서비스의 무료 체험, 유료 요금제, 저장소 용량, 멤버 수, 데이터 내보내기 횟수, 저장소 초과 및 요금제 변경 기준을 정리한다.

## 2. 요금제 가격 표시
요금제 가격은 VAT 포함 금액으로 표시한다.

## 3. 요금제 구성 초안

| 구분 | 월 요금 | 저장소 | 멤버 수 | 데이터 내보내기 |
| --- | ---: | ---: | ---: | ---: |
| Free Trial | 7일 무료 | 100MB | 추후 확정 | 제한 |
| Starter | 9,900원 | 500MB | 3명 | 월 1회 |
| Basic | 19,900원 | 1.5GB | 10명 | 월 3회 |
| Pro | 39,900원 | 5GB | 30명 | 월 10회 |

## 4. 추가 저장소
- 추가 저장소는 1GB 단위로 제공한다.
- 추가 저장소 요금은 1GB당 월 7,000원으로 한다.
- 추가 저장소 축소는 다음 결제일부터 적용한다.
- 현재 사용량이 축소 후 용량을 초과하면 추가 저장소 축소를 제한한다.

## 5. 무료 체험
- 모든 고객사는 최초 승인 후 7일 무료 체험을 이용할 수 있다.
- 무료 체험 저장소는 100MB로 한다.
- 무료 체험 중 데이터 내보내기는 제한한다.
- 무료 체험 종료 3일 전부터 상단 배너를 표시한다.
- 무료 체험 종료 후 유료 결제가 완료되지 않으면 유료 업무 기능을 제한한다.

## 6. 저장소 사용량 안내 및 제한
- 저장소 사용량이 80%에 도달하면 안내한다.
- 저장소 사용량이 100%를 초과하면 새 파일 업로드를 제한한다.
- 저장소 초과 상태에서도 기존 데이터 조회, 파일 삭제, 휴지통 비우기, 요금제 업그레이드는 허용한다.
- 작업지시서 텍스트 생성까지 제한할지는 추후 검토한다.

## 7. 데이터 내보내기 횟수 제한
- 요금제별 데이터 내보내기 횟수는 월 단위로 제한한다.
- 월 횟수를 초과하면 다음 결제주기까지 추가 요청을 제한한다.
- 상위 요금제로 변경하면 데이터 내보내기 가능 횟수도 즉시 상향 적용한다.

## 8. 멤버 수 제한
- 멤버 수 한도를 초과하면 신규 멤버 초대를 제한한다.
- 기존 멤버는 유지한다.
- 현재 멤버 수가 하위 요금제 한도를 초과하면 하위 요금제 변경을 제한한다.

## 9. 요금제 변경
- 상위 요금제 변경은 즉시 적용한다.
- 하위 요금제 변경은 다음 결제일부터 적용한다.
- 현재 사용량이 하위 요금제 용량을 초과하면 하위 요금제 변경을 제한한다.
- 현재 멤버 수가 하위 요금제 멤버 한도를 초과하면 하위 요금제 변경을 제한한다.

## 10. 요금제별 기능 차이
요금제별 핵심 기능은 동일하게 제공한다.

요금제 차이는 다음 기준 중심으로 둔다.
- 저장소 용량
- 멤버 수
- 데이터 내보내기 횟수

## 11. 미정 사항
- 장기 미납 데이터 보관/삭제 기준
- 무료 체험 멤버 수 제한
- 저장소 초과 시 작업지시서 텍스트 생성 제한 여부
- 실제 PG사
- 실제 요금제명 최종 확정
