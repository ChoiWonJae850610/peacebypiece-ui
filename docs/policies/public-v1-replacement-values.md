# 고객 공개 v1.0 치환값 관리표

## 문서 상태
- Policy Version: P-0.91.30
- Based App Version: 0.15.73
- 문서 성격: 고객 공개 v1.0 전환 전 치환값 추적표
- 적용 대상: docs/policies/public 하위 고객 공개 초안 문서 전체

## 1. 목적
이 문서는 WAFL 고객 공개 v1.0 정책 문서에서 “확정 필요”, “미정”, “초안”, “내부 기준 버전” 등으로 남아 있는 값을 실제 공개값으로 치환하기 위한 관리표다.

고객 공개 전에는 이 문서의 미정값을 모두 확정하거나, 공개 가능한 표현으로 바꿔야 한다.

## 2. 치환값 상태 구분

| 상태 | 의미 |
| --- | --- |
| 확정 | 고객 공개 문서에 바로 사용할 수 있는 값 |
| 예정 | 방향은 정했지만 도메인/계정/계약 등 실제 설정이 아직 완료되지 않은 값 |
| 미정 | 사업자등록, PG 계약, 법무 검토 등 외부 조건 때문에 아직 정할 수 없는 값 |
| 검토 필요 | 법무/세무/운영 검토 후 공개 가능 여부를 판단해야 하는 값 |
| 공개 전 제거 | 고객 공개 문서에는 남기면 안 되는 내부 표현 |
