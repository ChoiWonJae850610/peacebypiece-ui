## 제1조 목적
본 정책은 WAFL 서비스에서 고객사가 서비스 이용기간 중 업무자료를 내보낼 수 있는 데이터 내보내기 기능의 제공 범위, 처리 방식, 다운로드 기간, 보안 기준, 포함 및 제외 대상을 설명합니다.

## 제2조 데이터 내보내기 기능
1. 고객사 관리자는 서비스 이용기간 중 회사가 제공하는 “데이터 내보내기” 기능을 통해 일부 업무자료를 내려받을 수 있습니다.
2. 데이터 내보내기는 고객사가 서비스 이용 중 필요한 자료를 보관하거나 이전할 수 있도록 지원하기 위한 기능입니다.
3. 데이터 내보내기의 제공 범위, 파일 형식, 처리 시간, 포함 대상은 서비스 운영 상황에 따라 달라질 수 있습니다.

## 제3조 이용 가능 조건
1. 데이터 내보내기는 원칙적으로 유료 서비스 정상 이용 상태에서 사용할 수 있습니다.
2. 무료 체험 중에는 데이터 내보내기 기능이 제한됩니다.
3. 무료 체험 종료 후 미결제 상태, 결제 실패 상태, 해지 효력 발생 후, 장기 미납 상태, 이용 제한 상태에서는 데이터 내보내기 기능이 제한될 수 있습니다.
4. 회사는 보안, 부정 이용, 과도한 요청, 장애 대응 등 필요한 경우 데이터 내보내기 요청을 제한하거나 보류할 수 있습니다.

## 제4조 데이터 내보내기 처리 방식
1. 데이터 내보내기는 요청 기반으로 처리됩니다.
2. 고객사 관리자가 “데이터 내보내기”를 요청하면 회사는 서버에서 export zip 파일을 생성할 수 있습니다.
3. export zip 생성에는 데이터 양, 파일 수, 서버 상태에 따라 시간이 소요될 수 있습니다.
4. export zip 생성이 완료되면 회사는 고객사 관리자에게 이메일로 다운로드 링크를 발송할 수 있습니다.
5. 이메일에는 export zip 파일을 직접 첨부하지 않습니다.

## 제5조 다운로드 링크 보안
1. 데이터 내보내기 다운로드 링크는 고객사 관리자 로그인 및 권한 확인 후 사용할 수 있습니다.
2. 이메일 링크 자체만으로는 파일을 다운로드할 수 없습니다.
3. 회사는 다운로드 요청 시 고객사, 사용자 권한, export 요청 상태, 만료일 등을 확인할 수 있습니다.
4. 권한이 없거나 만료된 링크는 사용할 수 없습니다.

## 제6조 다운로드 가능 기간
1. export zip 다운로드 링크는 생성 완료 후 7일간 유효합니다.
2. 7일이 경과하면 export zip은 삭제될 수 있습니다.
3. 다운로드 가능 기간이 지난 경우 고객사 관리자는 데이터 내보내기를 다시 요청해야 할 수 있습니다.
4. 회사는 만료된 export zip의 복구를 보장하지 않습니다.

## 제7조 내보내기 포함 대상
데이터 내보내기에는 다음 항목이 포함될 수 있습니다.

1. 작업지시서
2. 디자인 파일
3. 일반 첨부파일
4. 메모
5. 댓글
6. 업체정보
7. 멤버정보
8. 기타 회사가 정한 고객사 업무자료

## 제8조 멤버정보 제공 범위
멤버정보는 업무 확인에 필요한 최소 범위로 제공됩니다.

포함될 수 있는 항목:
- 이름
- 전화번호
- 역할

제외될 수 있는 항목:
- 이메일
- 권한 상세값
- 계정 상태
- 생년월일
- 로그인 기록
- 내부 사용자 ID
- 인증 provider 정보
- 초대 토큰
- 시스템 로그

## 제9조 내보내기 제외 대상
다음 항목은 데이터 내보내기 대상에서 제외될 수 있습니다.

- 통계정보
- 분석정보
- 시스템 운영 로그
- 감사 로그
- 서비스 내부 분석 데이터
- 결제대행사 내부 데이터
- 인증정보
- 초대 토큰
- 권한 상세값
- 계정 상태
- 로그인 기록
- 내부 시스템 ID
- 회사의 운영상 생성한 내부 데이터

## 제10조 파일 구성
1. 데이터 내보내기 파일은 zip 형식으로 제공될 수 있습니다.
2. 작업지시서별 폴더 또는 파일 구조는 회사가 정한 방식에 따릅니다.
3. 디자인 파일과 일반 첨부파일은 작업지시서 또는 관련 업무자료 단위로 정리될 수 있습니다.
4. 메모와 댓글은 텍스트 파일, 문서 파일, 표 형식 파일 등 회사가 정한 방식으로 제공될 수 있습니다.
5. 업체정보와 멤버정보는 텍스트, CSV, 엑셀 등 회사가 정한 형식으로 제공될 수 있습니다.

## 제11조 요금제별 이용 횟수
데이터 내보내기 횟수는 요금제별로 제한될 수 있습니다.

| 요금제 | 데이터 내보내기 |
| --- | ---: |
| Trial | 제한 |
| Lite | 월 1회 |
| Flow | 월 3회 |
| Studio | 월 10회 |

월 이용 횟수를 초과한 경우 다음 결제주기까지 추가 요청이 제한될 수 있습니다.

상위 요금제로 변경하면 데이터 내보내기 가능 횟수도 즉시 상향 적용될 수 있습니다.

## 제12조 요청 실패 및 재요청
1. 데이터 내보내기 요청은 파일 수, 저장소 상태, 서버 상태, 보안 확인, 권한 문제 등으로 실패할 수 있습니다.
2. 요청이 실패한 경우 고객사 관리자는 서비스 화면 또는 고객지원 문의를 통해 상태를 확인할 수 있습니다.
3. 회사는 데이터 내보내기 실패에 대해 재시도 또는 재요청 기능을 제공할 수 있습니다.
4. 회사는 모든 데이터 내보내기 요청이 항상 즉시 완료되는 것을 보장하지 않습니다.

## 제13조 해지와 데이터 내보내기
1. 고객사는 해지 효력 발생 전까지 데이터 내보내기 기능을 이용할 수 있습니다.
2. 해지 효력 발생 후에는 별도의 읽기 전용 기간이나 다운로드 기간이 제공되지 않을 수 있습니다.
3. 고객사는 해지 신청 후 현재 이용기간 종료 전 필요한 자료를 미리 내보내야 합니다.
4. 해지 신청 상태인 경우 회사는 데이터 내보내기 필요성을 안내할 수 있습니다.

## 제14조 장기 미납과 데이터 내보내기
1. 결제 실패 또는 미납 상태에서는 데이터 내보내기 기능이 제한될 수 있습니다.
2. 결제 실패 또는 미납 상태가 장기간 지속되는 경우 회사는 장기 미납 안내 및 데이터 삭제 예정 안내를 제공할 수 있습니다.
3. 미납 90일 도달 시 회사는 이용계약 해지 및 고객사 업무 데이터 삭제를 처리할 수 있습니다.
4. 장기 미납으로 데이터가 삭제된 이후에는 데이터 내보내기 또는 복구가 제한되거나 불가능할 수 있습니다.

## 제15조 개인정보와 제3자 정보
1. 고객사가 서비스에 입력하거나 업로드한 자료에는 개인정보 또는 제3자 정보가 포함될 수 있습니다.
2. 고객사는 데이터 내보내기 파일을 내려받은 후 해당 파일을 관련 법령과 내부 기준에 따라 안전하게 보관하고 관리해야 합니다.
3. 고객사가 내려받은 export zip 또는 그 안의 자료를 제3자에게 제공하거나 외부에 저장하는 경우 그 이후의 관리 책임은 원칙적으로 고객사에 있습니다.

## 제16조 보안 및 접근 제한
회사는 다음 사유가 있는 경우 데이터 내보내기 요청 또는 다운로드를 제한할 수 있습니다.

- 고객사 관리자 권한 확인 실패
- 다운로드 링크 만료
- 결제 실패 또는 이용 제한 상태
- 과도한 요청
- 보안상 필요한 조치
- 부정 이용 의심
- 법령 또는 기관 요청
- 서비스 운영상 필요한 정당한 사유

## 제17조 문의처

- 고객지원: support@wafl.co.kr
