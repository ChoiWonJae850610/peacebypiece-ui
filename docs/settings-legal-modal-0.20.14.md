# 0.20.14 환경설정 약관·정책 탭 UI 재정리

## 목적

환경설정의 약관·정책 탭을 별도 이동 안내 중심에서 문서 목록과 모달 열람 중심으로 재정리한다.

## 변경 기준

- 약관·정책 탭 진입 시 고객 공개 문서 목록을 바로 표시한다.
- 이용약관, 개인정보처리방침, 요금·환불정책, 데이터 보관·삭제정책, 서비스 운영정책을 같은 목록 구조로 보여준다.
- 각 문서는 `보기` 버튼을 통해 모달에서 전체 내용을 확인한다.
- 문서 카드 내부에 다시 여러 설명 카드를 중첩하지 않는다.
- 필수 동의, 버전, 시행일, 문서 분류는 최소 뱃지로만 표시한다.
- 긴 운영 기준 카드는 제거하고 필수 동의 상태 안내만 짧게 유지한다.

## 적용 범위

- `components/admin/settings/AdminSettingsHub.tsx`
- `tests/e2e/workspace-policy-settings.spec.mjs`

## 제외 범위

- `/workspace/legal` 화면의 전체 재동의 UI 구조 변경
- 정책 DB/API 구조 변경
- 업무 접근 차단 정책 변경
- 신규 정책 문서 추가

## QA 기준

- 환경설정에서 약관·정책 탭 클릭 시 문서 목록이 바로 보여야 한다.
- `보기` 버튼 클릭 시 약관 문서 모달이 열려야 한다.
- 모달에서 문서 섹션 내용이 스크롤 가능해야 한다.
- 닫기 버튼으로 모달을 닫을 수 있어야 한다.
- 기존 `/workspace/legal` 재동의 화면은 계속 동작해야 한다.
