# 0.9.2181 — 작업지시서 목록 스크롤 표시 보정

## 목적

0.9.218 적용 후 PC 작업지시서 화면에서 좌측 작업지시서 목록 하단 카드가 잘려 보이는 문제를 최소 수정한다.

## 변경 범위

- 좌측 목록 패널 루트에 `w-full min-w-0`을 명시한다.
- 목록 헤더를 `shrink-0`으로 고정한다.
- 목록 영역에 `overscroll-contain`, `scrollbar-gutter: stable`, 하단 여유 padding을 추가한다.
- 목록 카드 간격과 카드 내부 세로 여백을 소폭 줄인다.

## 변경하지 않은 것

- 작업지시서 선택/저장/삭제/리오더 로직
- DB schema
- API route
- package 의존성
- 날짜 정책

## 확인 기준

1. `/worker` 접속
2. 좌측 작업지시서 목록에서 마지막 카드까지 스크롤 가능한지 확인
3. 선택된 카드의 하단 정보가 패널 아래에 잘리지 않는지 확인
4. 상세/우측 패널 스크롤 동작이 기존처럼 유지되는지 확인
