# 0.22.03 모바일 발주 품목 추가 모달 입력 후 추가 버튼 안정화

## 변경 요약
- 발주 품목 추가 모달에서 수량/단가 입력 후 모바일/아이패드 미니 일부 방향에서 추가 버튼 터치가 소실되는 문제를 보완했다.
- AppNumberInput blur 시 현재 표시값을 다시 commit하도록 보완했다.
- 발주 품목 추가 confirm 시 현재 모달의 정규화된 수량/단가 값을 직접 넘기도록 변경해 React state 반영 타이밍 의존도를 낮췄다.

## 확인 대상
- 모바일 세로에서 수량/단가 입력 후 추가 버튼 동작
- 모바일 가로 전환 후 다시 세로에서 추가 버튼 동작
- 아이패드 미니 가로/세로에서 추가 버튼 동작
- 갤럭시탭에서 추가 버튼 동작
- 수량 0일 때 추가 버튼 비활성 유지
