# 0.11.79 작업지시서 발주정보 총 금액 요약 보정

## 목적
디자이너 담당 작업지시서 화면에서 발주정보 섹션의 요약 줄에 발주 총 금액을 함께 표시한다.

## 반영 내용
- 발주정보 요약에 `총 n원` 항목을 추가했다.
- 발주 총 금액은 `수량 × 공임비 + 로스비` 기준으로 계산한다.
- 기존 비용 요약 카드의 계산 기준과 동일하게 발주 공임 합계와 로스비 합계를 더한 값을 사용한다.
- PC, 태블릿, 모바일 발주정보 섹션은 공통 `formatOrderSummary`를 사용하므로 동일하게 반영된다.

## 변경 범위
- 발주정보 summary formatter
- 발주정보 총계 계산 helper
- workorder i18n formatting 문구
- APP_VERSION

## 제외 범위
- 발주정보 저장 구조 변경 없음
- 비용 요약 카드 로직 변경 없음
- 생산구성 원단/부자재/외주공정 로직 변경 없음
- DB schema 변경 없음
- SQL 변경 없음
- `lib/runtimeMode.ts` 플래그 정리는 0.11.80에서 별도 분석 예정

## 확인 항목
- `/worker` 진입
- 발주정보 섹션 펼침
- 발주정보 제목 옆 요약 줄에 `총 n원` 표시 확인
- 수량, 공임비, 로스비 수정 시 요약 금액 갱신 확인
- 모바일/태블릿에서도 발주정보 요약 금액 표시 확인
