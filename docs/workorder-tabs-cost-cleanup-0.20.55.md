# 0.20.55 작업지시서 탭·비용 UI 정리

## 범위

- 작업지시서 생성 모달의 모바일 입력 완료 동작 보정
- 작업지시서 상세 탭 명칭 정리
- PC/태블릿 가로 상세 상단 요약 카드 제거
- 공정 입력 영역의 중복 금액 표시 제거
- 비용 요약 카드 전체 권한 표시

## 적용 내용

- 생성 모달 제목 입력에서 Enter/Done 후 현재 입력 focus를 해제합니다.
- 상세 탭은 요약 / 공정 / 구성 기준으로 정리했습니다.
- PC 상세 상단의 생산 / 자재 / 원가 3개 요약 카드는 제거했습니다.
- 공정 정보 입력 카드 하단의 수량·단가·로스비·합계 요약 줄을 제거했습니다.
- 공정 카드 우측 상단의 금액 뱃지를 제거했습니다.
- 비용은 비용 요약 섹션에서만 확인하도록 표시 위치를 단일화했습니다.
- 비용 요약 섹션은 역할/권한과 관계없이 표시합니다.

## 제외

- 상태전환 로직 변경 없음
- 권한 저장/역할 정책 변경 없음
- API/DB/R2/첨부/메모/휴지통/purge 흐름 변경 없음
