# UI Select 적용 1차 (0.17.96)

## 목적

원단·부자재 화면에서 직접 `<select>`를 사용하던 주요 입력/필터를 WAFL 공통 `AppSelect`로 전환한다.

## 적용 범위

- 발주서 목록 필터
  - 종류 필터
  - 상태 필터
- 발주 상세 기본정보
  - 구분
  - 공급처
- 발주 품목 입력표
  - 단위 선택

## 보정 사항

Radix Select는 빈 문자열 값을 직접 item value로 사용할 수 없으므로, `AppSelect` 내부에서 빈 값 옵션을 private sentinel 값으로 변환하고 외부 API에는 다시 빈 문자열로 돌려주는 처리를 추가했다.

## 유지 사항

- 발주서 저장 흐름 변경 없음
- 상태 전환 흐름 변경 없음
- 자재 계산/잔여 수량 계산 변경 없음
- PC/tablet/mobile 레이아웃 구조 변경 없음
