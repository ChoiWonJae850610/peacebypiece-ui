# 0.19.73 환경설정 회사 계정 요청 이력 표시

## 목적

고객사 관리자 환경설정의 회사 계정 정보 영역에서 회사 정보 변경 요청과 계정 비활성화 요청이 접수된 뒤 화면상으로 사라져 보이지 않도록 최근 요청 이력을 표시한다.

## 적용 범위

- `GET /api/admin/settings/company-account-requests` 추가
- 회사 계정 요청 최근 5건 조회 repository 추가
- 환경설정 회사 계정 정보 패널에 최근 요청 이력 섹션 추가
- 요청 접수 성공 후 최근 이력 자동 재조회
- ko/en i18n 문구 추가

## 비적용 범위

- 요청 승인/반려 처리 기능 구현 없음
- 시스템관리자 검토 화면 구현 없음
- DB schema 변경 없음
- 결제/요금제/회사 정보 실제 변경 로직 없음

## 확인 기준

1. 환경설정 > 회사 계정 정보에서 요청 이력 섹션이 표시된다.
2. 요청 이력이 없으면 빈 상태가 표시된다.
3. 회사 정보 변경 요청 또는 계정 비활성화 요청 접수 후 최근 이력에 새 요청이 표시된다.
4. 이력 조회 실패 시 요청 접수 기능은 계속 사용할 수 있고 실패 안내만 표시된다.
5. 기존 요청 작성, 기준정보, 요금제·저장공간, 약관·정책, 서비스 건의 탭 동작이 유지된다.
