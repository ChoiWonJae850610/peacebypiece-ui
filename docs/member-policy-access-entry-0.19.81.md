# 0.19.81 일반 멤버 정책 열람 진입점 보강

## 목적

0.19.80에서 `/workspace/legal` 고객 공개 정책 화면이 추가되었으므로, 일반 멤버가 정책 문서를 직접 찾아 들어갈 수 있는 진입점을 보강한다.

## 반영 범위

- 개인 설정 화면에 약관·정책 열람 카드 추가
- 일반 멤버 업무 홈 카드 모델에 약관·정책 카드 기준 추가
- ko/en 문구 추가

## 정책

- 고객 공개 정책 문서는 고객사 관리자와 일반 멤버가 모두 열람할 수 있다.
- 정책 동의 이력 저장은 아직 연결하지 않는다.
- DB/API 변경 없이 화면 진입점만 보강한다.

## 후속 작업

- 정책 동의 이력 DB/API 연결
- 고객사 승인 요청 전 필수 동의 저장
- 중요 정책 변경 시 재동의 화면 연결
