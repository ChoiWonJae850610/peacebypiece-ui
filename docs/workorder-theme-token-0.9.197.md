# 0.9.197 작업지시서 테마 토큰 준비 1차

## 목적

작업지시서 화면을 이후 색상 테마 변경과 언어 변경에 대응하기 쉽도록, 공통 UI부터 CSS 변수 기반 테마 토큰을 사용하도록 정리한다.

## 이번 버전 범위

- 전역 CSS 변수에 표면, 테두리, 텍스트, 강조, 위험, 경고, 그림자 토큰을 추가했다.
- 하단 중앙 토스트, 공통 모달, 작업지시서 공통 카드 컴포넌트 일부를 토큰 기반 클래스에 연결했다.
- 개별 작업지시서 화면 전체 색상 교체는 하지 않았다.

## 유지 원칙

- 기능 동작 변경 없음.
- 색상/톤 변경은 공통 컴포넌트부터 단계적으로 진행한다.
- Tailwind 색상 class를 한 번에 전부 제거하지 않는다.
- 다음 단계에서는 버튼, 배지, 섹션 헤더, 테이블 순서로 확장한다.

3. 작업지시서 좌측 카드 active/idle 상태
4. 우측 디자인/첨부/메모 카드 empty state
5. PDF/발주요청 모달 내부 문서 preview 톤
