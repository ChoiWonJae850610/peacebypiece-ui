# 0.18.45 협력업체관리 필터 toolbar wide 정렬 보정

## 목표

협력업체관리 필터 toolbar가 PC/태블릿 가로 화면에서 왼쪽에 몰려 보이는 문제를 줄이고, 좁은 화면에서의 검색창 전체폭 + 유형/상태 2열 구조는 유지한다.

## 변경

- 검색 영역과 필터 영역을 wide 화면에서 좌/우 그룹으로 분리했다.
- 검색창은 왼쪽에서 적정 폭을 유지한다.
- 유형 select와 사용 상태 select는 오른쪽 그룹으로 정렬한다.
- 모바일/태블릿 세로에서는 기존처럼 검색창 전체폭, 유형/상태 2열 구조를 유지한다.

## 비변경

- 협력업체 목록 wide table / compact card 반응형 구조는 변경하지 않는다.
- 검색/필터/정렬/등록/수정 모달 흐름은 변경하지 않는다.
- WorkspaceShell과 DB/API 저장 흐름은 변경하지 않는다.
