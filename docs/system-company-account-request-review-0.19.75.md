# 0.19.75 시스템관리자 회사 계정 요청 검토 화면 1차

## 목표

고객사 관리자가 환경설정에서 접수한 회사 정보 변경 요청과 계정 비활성화 요청을 시스템관리자가 확인할 수 있는 검토 화면을 1차로 연결한다.

## 적용 범위

- `/system/account-requests` 화면 추가
- `GET /api/system/company-account-requests` 추가
- 회사 계정 요청 전체 조회용 repository 추가
- 시스템관리자 홈의 고객사 운영 카드에 회사 계정 요청 메뉴 추가
- 요청 목록, 상태 필터, 상세 검토 패널 표시

## 제외 범위

- 승인 처리
- 반려 처리
- 회사 정보 실제 변경
- 계정 비활성화 실제 적용
- DB schema 변경
- 이메일/알림 발송

위 항목은 0.19.76 이후 별도 버전에서 처리한다.

## 검토 기준

- 고객사명
- 요청 유형
- 요청 상태
- 요청자
- 접수일
- 요청 상세 메시지
- 처리 메모가 있는 경우 처리 메모 표시

## 후속 작업

0.19.76에서 승인/반려 처리 API와 시스템관리자 처리 UI를 연결한다.
