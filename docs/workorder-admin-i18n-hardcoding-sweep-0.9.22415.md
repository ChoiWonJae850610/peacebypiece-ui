# 0.9.22415 고객관리자/작업지시서 i18n 하드코딩 전수 점검 1차

## 목적

저장소관리 정리 이후 남아 있던 고객관리자와 작업지시서 화면의 한영 혼합 문구를 1차로 정리한다.

## 반영 내용

- 작업지시서 상세 데스크톱 섹션 그룹 제목/설명을 i18n dictionary로 이동했다.
- 진행 단계 라벨과 workflow action 버튼 표시를 현재 locale의 dictionary 기준으로 표시하도록 보정했다.
- 작업지시서 목록 카드의 상태 배지를 현재 locale 기준으로 표시하도록 보정했다.
- 발주 정보의 상태 요약, 발주 유형, 미선택 값 표시를 locale 기준으로 변환했다.
- 원단/부자재, 외주 공정의 미선택/기본값 표시를 locale 기준으로 변환했다.
- 검수 모달과 기본정보 모달의 하단 버튼이 static default locale이 아니라 현재 locale을 따르도록 보정했다.
- 고객관리자 상단 제목이 server 기본 한국어 문자열에 고정되지 않도록 known title을 현재 admin dictionary로 변환했다.

## 범위

이번 버전은 캡처에서 확인된 `/admin`, `/admin/partners`, `/admin/files`, `/worker`의 주요 한영 혼합 지점을 우선 정리했다.

## 제외

- DB에 저장된 작업명, 고객사명, 공장명, 담당자명, 카테고리명 자체는 사용자 데이터로 보고 자동 번역하지 않는다.
- 삭제일시 시간대 표시 문제는 다음 버전에서 별도 처리한다.
