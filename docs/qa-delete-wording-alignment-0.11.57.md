# 0.11.57 삭제 안내 문구 통일

## 목적

작업지시서 삭제 모달과 첨부파일 삭제 모달에서 서로 다르게 보이던 삭제 안내 문구를 `삭제한` 기준으로 맞춘다.

## 반영 내용

- 작업지시서 삭제 모달 안내 문구를 `삭제한 작업지시서는 휴지통으로 이동하며, 30일 이내 복원할 수 있습니다.`로 정리했다.
- 첨부파일 삭제 정책 안내는 `삭제한 파일은 휴지통으로 이동하며 30일 동안 복원할 수 있습니다.` 기준으로 정리했다.
- 저장소 작업지시서 묶음 설명도 `삭제한 작업지시서` 기준으로 맞췄다.
- 저장소 presentation description의 `삭제된` 표현도 고객관리자 화면 기준 문구에서 `삭제한`으로 맞췄다.

## 변경하지 않은 내용

- 삭제 정책 자체는 변경하지 않았다.
- 작업지시서 삭제, 첨부 삭제, 휴지통 이동, 복원, 삭제 요청, R2 purge 흐름은 변경하지 않았다.
- DB schema 변경은 없다.

## 확인 항목

1. 작업지시서 삭제 모달에서 `삭제한 작업지시서는 휴지통으로 이동하며, 30일 이내 복원할 수 있습니다.` 문구가 보이는지 확인한다.
2. 첨부파일 삭제 모달 또는 파일 정책 안내에서 `삭제한 파일은 휴지통으로 이동하며 30일 동안 복원할 수 있습니다.` 문구가 보이는지 확인한다.
3. `/admin/files` 작업지시서 저장소 설명과 summary label이 `삭제한 작업지시서` 기준으로 보이는지 확인한다.
4. 삭제/복원/R2 purge 동작이 기존과 동일한지 확인한다.
