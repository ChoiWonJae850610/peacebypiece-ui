# 0.10.6 고객관리자 카드형 홈 문구 정리

## 목적

고객관리자 카드형 홈에 남아 있던 설계/개발자용 표현을 줄이고 실제 운영 화면에 맞는 카드 문구로 정리한다.

## 반영 기준

- `권한 기반 확장 후보` 같은 표현은 사용자 화면에서 쓰지 않는다.
- `기본 카드 4개 + 멤버관리 후보` 같은 내부 설계 문구는 화면에 노출하지 않는다.
- 멤버관리, 단위표준, 외주공정, 생산품유형은 준비 중 상태로 두되 설명은 실제 업무 기준으로 표현한다.
- 카드 영역은 더 작은 패딩과 간격을 사용해 운영 대시보드 첫 화면에서 메뉴 접근성이 높아지도록 한다.

## 보류

- 실제 멤버관리 화면 구현
- 권한 DB 및 권한 편집 UI
- 기준정보 화면 구현
