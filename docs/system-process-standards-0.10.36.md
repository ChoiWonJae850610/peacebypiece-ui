# 0.10.36 시스템관리자 외주공정 유형 관리 1차 화면

## 목적

고객관리자 환경설정에서 외주공정 유형을 직접 추가하지 않고 시스템 표준 목록 중 사용 여부만 선택하는 정책을 적용하기 위해, 시스템관리자가 관리할 외주공정 유형 원장 1차 화면을 추가한다.

## 이번 버전 범위

- `/system/standards/processes` 경로 추가
- 외주공정 유형 원장 화면 추가
- 기준 필드 표시
  - 공정명
  - 분류
  - 설명
  - 사용 예시
  - 정렬 순서
  - 상태
- `/system/standards` 설계 화면에서 외주공정 유형 관리 화면으로 이동 링크 추가
- 시스템 콘솔 빠른 링크에 외주공정 유형 관리 추가

## 제외 범위

- 실제 추가/수정/삭제 저장
- DB schema 변경
- 고객사별 사용 여부 저장
- 감사 로그 기록
- 고객관리자 기준정보 저장 로직 변경

## 후속 작업

- `system_outsourcing_process_standards` 원장 설계 확정
- `company_enabled_process_standards` 고객사별 사용 여부 구조 확정
- 시스템관리자 CRUD 연결
- 기준정보 변경 감사 로그 연결
