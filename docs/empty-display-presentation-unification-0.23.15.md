# 0.23.15 빈 값 표시 기준 통일

## 목적

작업지시서와 자재 발주 화면에서 빈 값을 직접 `-` 문자열로 처리하던 코드를 WAFL 공통 `EMPTY_DISPLAY` 기준으로 통합한다.

## 반영 범위

- 작업지시서 PC 헤더의 담당자·기본정보 요약 빈 값
- 작업지시서 모바일·태블릿 헤더의 담당자·기본정보 요약 빈 값
- 작업지시서 상세 편집 셀의 빈 값
- 작업지시서 상세 ViewModel의 담당자 빈 값
- 자재 발주 작성일 표시의 빈 값

## 표시 정책

- 일반 데이터 필드가 비어 있으면 `EMPTY_DISPLAY`를 사용한다.
- 실제 빈 상태 카드와 안내 문구는 기존 의미 있는 문장을 유지한다.
- `미지정`, `없습니다`처럼 업무 의미가 있는 선택·상태 문구는 기계적으로 `-`로 바꾸지 않는다.

## 영향

- UI 표시만 변경한다.
- 저장 데이터, API 계약, DB 구조는 변경하지 않는다.
