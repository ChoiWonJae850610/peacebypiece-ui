# 작업지시서 표시/문구 정리 2차 (0.19.59)

## 목적

작업지시서 상세 상단 요약 카드의 남은 화면 문구를 i18n 기준으로 분리하고, 사용자에게 노출되는 생산/자재/원가 요약 표현을 정리한다.

## 변경 범위

- `WorkOrderDetailVisualSummary`의 하드코딩된 한국어 문구를 `lib/i18n/*/workorder.ts`로 이동
- 생산 요약 표현을 `봉제/외주`에서 현재 화면 용어에 맞는 `제작/추가 공정` 기준으로 보정
- 원가 요약 설명을 관리자 비용 요약 표시/권한 숨김 기준으로 정리
- `actionFlow` 일부 문구 키의 들여쓰기만 정리

## 비변경 범위

- 작업지시서 상태 전환 로직 변경 없음
- DB/API/R2/PDF/첨부/메모 저장 흐름 변경 없음
- 권한 판단 로직 변경 없음
- package.json/package-lock.json 변경 없음

## 화면 확인 지점

- 작업지시서 상세 상단 요약 카드 3개가 정상 표시되는지 확인
- 제작/추가 공정이 있는 경우 생산 카드 설명이 `제작 n · 추가 공정 n` 형태로 표시되는지 확인
- 추가 공정이 없는 경우 생산 카드 설명이 `제작/추가 공정`으로 표시되는지 확인
- 원가 권한이 있는 계정과 없는 계정에서 원가 카드 문구가 각각 정상 표시되는지 확인
