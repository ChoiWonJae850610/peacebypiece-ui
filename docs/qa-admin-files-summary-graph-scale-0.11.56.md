# 0.11.56 저장소 요약 그래프 확대

## 목적

고객관리자 저장소 관리 화면의 상단 요약 카드에서 요금제 용량 원통형 그래프와 파일 유형 도너츠 그래프가 화면 대비 작게 보이는 문제를 보정한다.

## 반영 내용

- 요금제 용량 원통형 그래프의 너비와 높이를 확대했다.
- 사용률 badge와 진행 bar의 시각 비중을 높였다.
- 파일 유형 도너츠 그래프의 SVG viewBox, radius, stroke width, 표시 영역을 확대했다.
- 문서/디자인 막대 영역은 도너츠가 더 크게 보이도록 상대 폭을 줄였다.
- 상단 요약 카드 3종의 최소 높이와 padding을 키워 PC 화면에서 더 묵직하게 보이도록 정리했다.

## 변경하지 않은 내용

- 저장소 snapshot 계산 로직은 변경하지 않았다.
- 파일 유형 분포 계산 로직은 변경하지 않았다.
- 휴지통 복원/삭제 요청/R2 purge 흐름은 변경하지 않았다.
- DB schema는 변경하지 않았다.

## 확인 항목

- `/admin/files` 상단 요약 카드 3종이 PC 화면에서 더 크게 보이는지 확인한다.
- 요금제 용량 원통형 그래프가 기존보다 크게 표시되는지 확인한다.
- 파일 유형 도너츠 그래프가 기존보다 크게 표시되는지 확인한다.
- 문서/디자인 막대가 카드 밖으로 밀리지 않는지 확인한다.
- 모바일 폭에서 요약 카드가 세로로 자연스럽게 쌓이는지 확인한다.
