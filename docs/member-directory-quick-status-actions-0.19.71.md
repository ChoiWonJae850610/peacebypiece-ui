# 0.19.71 멤버관리 행별 빠른 상태 처리 버튼

## 목적

멤버 상세 모달을 열지 않아도 멤버 목록에서 자주 쓰는 상태 변경을 빠르게 처리할 수 있도록 행별 액션 버튼을 추가했다.

## 반영 범위

- 재직중 멤버: `비활성` 빠른 처리
- 비활성 멤버: `재직중` 복구
- 탈퇴 요청 멤버: `탈퇴 완료` 처리, `재직중` 복구
- 탈퇴 완료 멤버: `재직중` 복구
- 탈퇴 완료 처리 시 브라우저 확인창 표시
- 기존 멤버 상세 모달 저장 API 재사용

## 보존한 범위

- DB schema 변경 없음
- API route 추가 없음
- 초대/승인/권한 저장 흐름 변경 없음
- 고객사 관리자 멤버관리 목록 제외 정책 유지

## 구현 주의사항

빠른 상태 변경은 기존 `PATCH /api/admin/members/{memberId}`를 사용한다. 이 API는 부분 업데이트처럼 보이지만 기존 repository 구현상 `phone`이 누락되면 null로 저장될 수 있으므로, 빠른 상태 변경 시 현재 멤버의 이름/연락처/역할/권한을 함께 전달한다.
