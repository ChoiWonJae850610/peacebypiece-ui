# 0.10.42 시스템 생산품 유형 템플릿 분류 수정 UX 보완

## 목적

0.10.41에서 생산품 유형 기본 템플릿의 조회, 템플릿 추가/수정, 분류 추가, 분류 사용 상태 변경을 연결했지만 이미 생성된 1차/2차/3차 분류명을 수정하는 UX가 부족했다.

0.10.42에서는 각 분류 항목에 명시적인 수정 버튼을 추가해 이름과 정렬값을 인라인으로 수정할 수 있게 한다.

## 변경 내용

- 1차 분류 수정 버튼 추가
- 2차 분류 수정 버튼 추가
- 3차 분류 수정 버튼 추가
- 분류 수정 시 인라인 입력 폼 표시
- 분류명과 정렬값 수정 가능
- 저장/취소 버튼 제공
- 기존 사용/미사용 전환 기능 유지
- 기존 템플릿 추가/수정/상태 변경 기능 유지

## 변경하지 않은 것

- DB schema 변경 없음
- 신규 고객사 생성 시 템플릿 복사 로직 연결 없음
- 고객관리자 생산품 유형 화면 연결 없음
- 기존 감사 로그 이벤트 흐름 유지
