# 0.21.53 발주서 목록 카드 구조 단순화

## 목표

발주서 목록 카드를 작업지시서 목록 카드의 상태 뱃지 흐름과 맞추고, 미지정/공급처 미선택/품목 미입력 같은 설명형 문구를 줄입니다.

## 기준

- 카드 상단에는 작업지시서 카드와 같은 상태 뱃지를 표시합니다.
- 상태 뱃지 아래에는 생성일시를 표시합니다.
- 본문에는 자재 종류와 공급처만 표시합니다.
- 자재 종류가 없으면 `자재 종류 선택 전`으로 표시합니다.
- 공급처가 없으면 `공급처 선택 전`으로 표시합니다.
- `품목 미입력`, `30수 면 외 1건` 같은 품목 요약은 목록 카드에서 제거합니다.
- 발주 품목 정보는 가운데 상세 패널에서 확인합니다.
- 작성중 발주서의 `...` 메뉴는 우측 상단에 유지합니다.
