# 작업지시서 대화형 메모 제거 0.22.92

## 결정

작업지시서의 작성자별 메모, 댓글, 답글, 삭제 이력 기능을 제거한다.
작업지시서 협업 대화는 외부 커뮤니케이션 도구를 사용하며 제품 UI에는 중복 협업 기능을 유지하지 않는다.

## 제거 범위

- memos 테이블과 작업지시서 memo 컬럼
- 메모 API와 클라이언트 저장 흐름
- 메모 작성, 댓글, 답글, 작성자별 삭제 UI
- 메모 첨부 범위와 R2 경로 정책
- 메모 휴지통, 복원, 영구삭제 집계
- 메모 이력 생성 코드와 관련 권한

## 유지 범위

- 디자인 파일
- 일반 첨부파일
- 재고 조정 사유 등 다른 업무 도메인의 memo 필드
- 협력업체와 자재 등 다른 도메인의 일반 비고 필드

## 다음 단계

공장 전달사항은 기존 메모를 재사용하지 않고 작업지시서당 하나의 공식 문서형 데이터로 별도 구현한다.
