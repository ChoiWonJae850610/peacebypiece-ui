# 0.11.6 관리자 저장소 테이블 라벨 표준화

## 목적

`/admin/files` 저장소 화면의 테이블 안에서 개별 className으로 구현되던 파일 유형 라벨을 `AdminStatusBadge` 공통 컴포넌트 기준으로 전환한다.

## 반영 범위

- 문서/디자인 목록의 파일 유형 표시
- 휴지통 목록의 유형 표시

## 제외 범위

- 휴지통 정렬 헤더 버튼
- 체크박스 선택 컨트롤
- 저장소 API / R2 복원 / 삭제 요청 흐름
- 모달 구조

## 확인 항목

- `/admin/files` 문서/디자인 목록에서 유형 라벨이 기존 의미를 유지하는지 확인
- `/admin/files` 휴지통 목록에서 유형 라벨이 중앙 정렬로 유지되는지 확인
- 선택/삭제/복원 흐름이 기존과 동일하게 동작하는지 확인
