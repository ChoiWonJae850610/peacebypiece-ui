# 0.20.11 환경설정 상단 구조 정리 1차

## 목적

0.20.10에서 정리한 환경설정 IA 재설계 기준을 첫 화면 밀도에 반영한다. 이번 버전은 상세 탭 내부 구조를 크게 바꾸지 않고, 상단 진입 구조의 카드 과밀도를 먼저 낮춘다.

## 변경 기준

- 기존 상단 5개 대형 카드 내비게이션을 탭/세그먼트형 내비게이션으로 축소한다.
- 상단 메뉴에서 설명성 뱃지는 제거한다.
- 메뉴에는 제목과 핵심 하위 항목 1줄만 남긴다.
- 개인설정 안내 헬프메시지는 상단에서 제거한다. 개인설정은 우측 상단 사람 아이콘의 역할로 유지한다.
- 상세 탭 내부의 회사 정보, 요금제, 약관·정책 구조는 후속 버전에서 단계적으로 정리한다.

## 화면 판단

0.20.09 화면은 카드, 뱃지, 헬프메시지, 운영 기준 안내가 동시에 노출되어 실제 운영 화면보다 설계 문서처럼 보이는 문제가 있었다. 0.20.11에서는 사용자가 가장 먼저 보는 상단 영역부터 조작 가능한 탭 구조로 바꾸고, 실제 상세 정보는 선택된 탭 아래에서 확인하도록 분리한다.
