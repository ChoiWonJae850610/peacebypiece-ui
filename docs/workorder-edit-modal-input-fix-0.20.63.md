# 0.20.63 작업지시서 구성/공정 편집 모달 입력 UX 보정

## 목적

0.20.60~0.20.62에서 구성/공정 탭을 목록 카드 + 편집 모달 구조로 전환한 뒤 확인된 입력 UX를 보정한다.

## 반영 내용

- WAFL 공통 `AppSelect`에 열린 상태에서 trigger를 다시 누르면 닫히는 동작을 추가했다.
- WAFL 공통 `AppNumberInput`을 추가했다.
- 구성/공정 편집 모달의 숫자 입력을 `AppNumberInput`으로 교체했다.
- 숫자 입력은 0 클릭 시 빈칸으로 전환되고, 입력 중 세 자리 콤마가 표시되며, 빈값 blur 시 0으로 복귀한다.
- 공정 편집 모달의 추가공정 업체 입력을 텍스트 입력에서 업체 선택형으로 변경했다.
- 구성/공정 편집 모달의 입력 전/선택 전 blur 처리를 유지해 모바일·태블릿 포커스 꼬임을 줄였다.

## 변경하지 않은 범위

- 저장 API
- DB schema
- R2 업로드/첨부/메모/휴지통/purge 흐름
- 작업지시서 상태전환/권한 흐름
