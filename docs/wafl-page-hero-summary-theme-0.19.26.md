# WAFL Page Hero / Summary Card theme standard 0.19.26

## 목적
고객사 관리자 주요 화면의 상단 hero, 요약 카드, 섹션 안내 문구를 WAFL theme token 기반 공통 규격으로 정리한다.

## 적용 화면
- 통계정보
- 멤버관리
- 협력업체관리
- 저장소관리

## 적용 기준
- 화면별 직접 배경색/그라데이션 대신 `WaflPageHero` 공통 컴포넌트를 사용한다.
- 공통 hero와 notice는 `var(--pbp-*)` theme token만 사용한다.
- 요약 카드는 `AdminSummaryMetricCards`를 유지하되 카드 배경/좌측 강조선을 theme token 기반으로 보정한다.
- 사용자에게 노출되는 문구에서 "모달이 열립니다" 같은 구현 중심 표현을 제거한다.
- 멤버관리의 중첩 제목은 줄이고 내부 섹션을 "멤버 목록" 기준으로 정리한다.
- PC에서는 초대 관리 카드와 초대 링크 테이블을 숨기고, 모바일/태블릿 초대 사용 안내만 표시한다.

## 후속
- 0.19.27에서 갤럭시탭 회전 대응과 테이블 공통 컴포넌트 사용 여부를 점검한다.
