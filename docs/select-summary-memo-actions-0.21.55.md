# 0.21.55 select 상태·발주 요약·작업메모 액션 정리

## 범위
- 작업지시서 제작/추가 공정 모달 select 옵션 정리
- 발주서 공급처 select 비활성 기준 통일
- 발주 기본정보 카드 하단 비용 요약 카드 추가
- 작업메모 카드 액션을 더보기 메뉴로 정리

## 기준
- 업체 select는 `업체 선택` 다음 실제 업체 목록으로 이어지며 `선택 안함`을 업체 후보로 반복 표시하지 않는다.
- 추가 공정 항목 select의 `선택 안함`은 하나만 유지한다.
- 공급처/업체가 없으면 해당 select는 비활성화하고 `선택 가능한 ... 없음` 상태를 표시한다.
- 발주 기본정보 설명 문구를 제거하고 총 비용·원단 합계·부자재 합계를 카드로 표시한다.
- 작업메모 카드는 수정/댓글/삭제 액션을 `...` 메뉴로 모으고, 댓글이 있을 때만 하단 구분선을 표시한다.
