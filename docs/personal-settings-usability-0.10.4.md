# 개인 설정 사용성 보정 — 0.10.4

## 목표

`/me/settings` 개인 설정 화면에서 선택한 값이 즉시 반영되는 범위를 명확히 하고, 사용자가 기본 진입 화면으로 바로 이동하거나 설정을 기본값으로 되돌릴 수 있도록 한다.

## 반영 내용

- 저장된 개인 설정을 불러올 때 언어, 색상, 화면 밀도 값을 문서 루트에도 함께 적용한다.
- 개인 설정 저장/초기화 로직을 `lib/me/personalSettings.ts`로 모아 이후 계정 기반 저장소로 이동하기 쉽게 정리했다.
- 기본 진입 화면 설정값에 따라 상단의 `기본 진입 화면 열기` 버튼이 `/workspace` 또는 `/worker`로 이동하도록 정리했다.
- `기본값으로 되돌리기` 버튼을 추가해 브라우저에 저장된 개인 설정을 초기값으로 복원할 수 있게 했다.
- 화면 밀도 설정은 이번 단계에서 `/me/settings` 화면 여백에만 즉시 반영한다. 작업지시서, 관리자 홈 등 전체 업무 화면 적용은 후속 단계에서 별도로 확대한다.

## 보류

- 사용자 계정별 DB 저장
- 조직 전체 기본값과 개인 override 병합
- 로그인 후 기본 진입 화면 자동 redirect
- 전체 AppShell에 대한 화면 밀도 일괄 적용
