# 환경설정 기준정보 설정 UI 재정리 0.20.15

## 목적

환경설정의 기준정보 설정 화면에서 설명 카드와 버튼 카드가 과도하게 보이는 문제를 줄이고, 실제 운영자가 기준값을 확인하고 관리하는 화면처럼 보이도록 정리한다.

## 변경 기준

- 생산품 유형, 단위 표준, 외주 공정 유형을 내부 탭으로 나눈다.
- 선택된 탭의 현재 항목을 목록/테이블 형태로 바로 보여준다.
- 관리 버튼은 선택된 탭 기준으로 하나만 노출한다.
- `관리 가능`, `조회 전용` 같은 설명성 뱃지는 반복 노출하지 않는다.
- 기준정보가 비어 있는 경우의 안내는 짧게 유지한다.
- 기존 기준정보 관리 모달과 저장 API 흐름은 그대로 유지한다.

## 적용 범위

- `components/admin/standards/AdminStandardsSection.tsx`
- 환경설정 > 기준정보 탭
- 생산품 유형/단위 표준/외주 공정 유형 표시 밀도

## 제외 범위

- 기준정보 DB schema 변경
- 기준정보 저장 API 변경
- 작업지시서 생성/협력업체 선택 흐름 변경
- 정책 관리, 저장 정책, 알림 정책 모달 구조 변경

## 화면 구조

```txt
기준정보 설정
작업지시서 생성에 쓰는 기준값을 탭별로 관리합니다.      [선택 항목 관리]

[생산품 유형] [단위 표준] [외주 공정 유형]

생산품 유형
사용 5 / 전체 7

이름              단계      상태      순서
상의              1         사용      1
티셔츠            2         사용      2
```

## 후속 확인

- E2E에서 환경설정 진입이 유지되는지 확인한다.
- 기준정보 탭 클릭 후 내부 탭 전환이 정상인지 확인한다.
- 선택 항목 관리 버튼이 기존 모달을 여는지 확인한다.
- 모바일/태블릿에서 표가 과도하게 깨지지 않는지 확인한다.
