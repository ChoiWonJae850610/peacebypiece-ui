# 0.23.93 URL·라우팅·세션 안정화

- Google 로그인 시작·콜백에 안전한 내부 `returnTo` 경로를 전달한다.
- `/workspace`, `/worker`, `/system`, `/me` 하위 경로만 복귀 대상으로 허용한다.
- 외부 URL과 `//host` 형태를 차단해 open redirect를 방지한다.
- 보호 layout은 세션이 없을 때 로그인 화면으로 이동하며 기본 업무 복귀 경로를 보존한다.
- 전역 404와 오류 복구 화면을 추가한다.
- 세부 화면별 정확한 URL 복원, 브라우저 뒤로가기, query string 유지 여부는 체크포인트 통합 테스트에서 확인한다.
