# 0.9.22414 저장소 휴지통 상세 모달 i18n 정리

## 목적

저장소 관리 화면의 휴지통 상세 모달에 남아 있던 한글/영문 혼합 문구와 타입별 안내문 재사용 문제를 정리한다.

## 반영 내용

- 작업지시서 휴지통 상세 모달 제목, 단계 라벨, 배지, 단계명, 통계 라벨을 i18n key 기준으로 정리했다.
- 파일 휴지통 상세 모달 제목, 미리보기 버튼, 상세 필드 라벨, 안내문을 i18n key 기준으로 정리했다.
- 파일 상세 모달에 작업지시서 묶음 복원 안내문이 잘못 표시되지 않도록 파일 전용 안내문을 분리했다.
- 작업지시서 묶음 첨부의 복원/삭제 요청 제한 사유를 서버 문자열 대신 UI i18n formatter에서 계산하도록 정리했다.
- 서버 action의 휴지통 제한 사유 문자열은 더 이상 사용자 노출 문구 저장소로 쓰지 않도록 null 처리했다.
- ko/en admin dictionary에 휴지통 상세 모달 전용 key를 동기화했다.

## 확인 항목

1. 한국어 locale에서 작업지시서 상세 모달이 전부 한국어로 표시된다.
2. 영어 locale에서 작업지시서 상세 모달이 전부 영어로 표시된다.
3. 한국어 locale에서 파일 상세 모달이 전부 한국어로 표시된다.
4. 영어 locale에서 파일 상세 모달이 전부 영어로 표시된다.
5. 파일 상세 모달에는 작업지시서 묶음 복원 안내문이 나오지 않는다.
6. 작업지시서 상세 모달에는 문서/디자인/메모 묶음 처리 안내문이 나온다.
