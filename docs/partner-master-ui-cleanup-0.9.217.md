# 0.9.217 — 협력업체/기준정보 UI 정리

## 목적

협력업체 관리 화면의 용어와 조작 방식을 간단하게 정리한다. 이번 버전은 화면 사용성 보완이며 DB schema, API route, package 의존성은 변경하지 않는다.

## 반영 내용

1. 버튼명 정리
   - `협력업체 추가` → `업체추가`
   - 신규 등록 모달 제목도 `업체추가`로 통일한다.

2. 상태 필터명 정리
   - `전체 상태` → `전체`
   - 사용/미사용 필터와 같은 줄에서 더 짧게 보이도록 정리한다.

3. 협력업체 관리 안내 문구 교체
   - 기존 설명성 문구를 줄이고, 행 선택 수정 방식과 외주공정 관리 기준을 안내한다.

4. row 선택 수정 모달 기준 반영
   - 목록의 별도 `수정` 버튼을 제거한다.
   - 업체 행을 선택하면 수정 모달이 열린다.
   - PC 테이블의 가로 폭을 줄이고, 수정 동작을 목록 행 자체에 연결한다.

## 보류 항목

다음 항목은 후속 버전에서 분리한다.

- 생산품유형 CRUD 화면 정리
- 단위표준 CRUD 화면 정리
- 외주공정유형 CRUD 화면 정리
- 실제 권한 기반 표시/차단
- 기준정보 관리 전체를 독립 탭 구조로 재배치

## SQL DDL 필요 여부

불필요.

## 전체 리셋 필요 여부

불필요.

## 테스트 케이스

1. `/admin/partners` 접속
2. 상단 버튼명이 `업체추가`로 표시되는지 확인
3. 상태 필터가 `전체 / 사용 / 미사용`으로 표시되는지 확인
4. 협력업체 안내 문구가 행 선택 수정 기준으로 표시되는지 확인
5. 업체 목록 행을 클릭하면 수정 모달이 열리는지 확인
6. 별도 `수정` 버튼 컬럼이 사라졌는지 확인
7. 기존 등록/수정 저장 동작이 유지되는지 확인
