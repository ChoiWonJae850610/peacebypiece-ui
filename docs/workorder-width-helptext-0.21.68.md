# WAFL 0.21.68 작업지시서 폭 정렬 및 헬프메시지 정리

## 목적
- 작업지시서 목록 카드 폭을 생성 버튼 기준에 맞춘다.
- 작업지시서 상세 가운데 패널의 비용/공정/자재 섹션 폭을 진행단계 카드 기준으로 맞춘다.
- 작업지시서 관련 모달의 불필요한 헬프메시지를 제거한다.
- 작업메모 empty 문구를 `메모 없음`으로 단순화한다.

## 변경
- 작업지시서 목록 scroll 영역의 오른쪽 padding을 제거해 카드 폭이 생성 버튼 기준과 맞도록 보정했다.
- desktop 상세 섹션의 비용요약/제작공정/원단·부자재 영역을 header와 같은 좌우 inset 안에 배치했다.
- 작업지시서 생성/기본정보/담당자/공정/자재 모달 설명 문구를 비워 모달 본문 밀도를 낮췄다.
- 작업메모 빈 상태 문구를 `메모 없음`으로 정리했다.
