# WAFL Settings Detail Sections 0.19.44

## 목표

고객사 관리자 환경설정 화면의 상세 섹션을 회사 정보, 요청 관리, 운영 기준, 요금제/저장공간, 약관/정책, 개발 건의 단위로 분리했다.

## 적용 원칙

- 상위 영역은 `WaflSectionPanel`을 유지한다.
- 상세 묶음은 `WaflSettingsSectionGroup`으로 분리한다.
- 개별 값/액션 카드는 `WaflSettingCard`를 사용한다.
- 회사 정보 변경 요청, 계정 비활성화 요청, 기준정보 설정, 요금제 조회 API 흐름은 변경하지 않는다.
- 개인 언어와 색상 테마는 환경설정이 아니라 개인 설정 영역에서 관리한다는 안내를 유지한다.

## 테스트 포인트

- `/workspace/settings` 계정 정보 탭에서 회사 정보, 요청 관리, 요청 작성, 운영 기준 섹션이 분리되어 보이는지 확인한다.
- 회사 정보 변경 요청과 계정 비활성화 요청의 textarea, 취소, 요청 접수 동작이 기존과 동일한지 확인한다.
- 기준정보 설정 탭의 기존 기준정보 관리 기능이 그대로 동작하는지 확인한다.
- 요금제·저장공간 탭의 읽기 전용 정보와 요청 흐름 안내가 분리되어 보이는지 확인한다.
- 약관·정책, 개발 건의 탭의 준비중/이메일 안내 카드가 화면 폭에 맞게 줄바꿈되는지 확인한다.
- PC, iPad, Galaxy Tab, iPhone에서 섹션 폭이 넘치지 않는지 확인한다.
