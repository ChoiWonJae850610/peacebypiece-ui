# 시스템관리자 헤더 작업지시서 이동 버튼 제거 — 0.10.98

## 목적

시스템관리자 `/system` 홈 우측 상단에 노출되던 작업지시서 화면 이동 버튼을 제거한다.

## 변경 내용

- `/system` 홈 헤더 우측 상단에서 `/`로 이동하는 버튼을 제거했다.
- 버전 표시 pill은 유지했다.
- 운영 메뉴/관리·점검 메뉴 카드형 진입 구조는 유지했다.
- 시스템관리자 홈 외의 route와 작업지시서 화면은 변경하지 않았다.

## 확인 항목

- `/system` 접속 시 우측 상단에 버전만 표시되는지 확인한다.
- 작업지시서 화면 이동 버튼이 표시되지 않는지 확인한다.
- 시스템관리자 카드 링크들이 기존처럼 동작하는지 확인한다.
