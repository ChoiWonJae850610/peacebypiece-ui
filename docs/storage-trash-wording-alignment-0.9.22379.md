# 0.9.22379 저장소 휴지통 문구/정렬 정리

## 목적

고객관리자 저장소 휴지통 화면에서 개발자 관점의 표현을 줄이고, 휴지통 목록 컬럼의 시각 정렬을 통일한다.

## 반영 내용

- 고객관리자 화면에서 `부모 작업지시서` 표현을 제거했다.
- 파일 단독 복구 제한 문구를 `삭제된 작업지시서에 연결된 파일` 기준으로 바꿨다.
- 휴지통 테이블 컬럼 헤더를 가운데 정렬했다.
- 휴지통 테이블 본문 내용을 가운데 정렬했다.
- 휴지통 컬럼 폭을 선택 / 대상 / 삭제일시 / 작업지시서 / 유형 / 크기 기준으로 보정했다.
- `삭제일자` 라벨을 `삭제일시`로 변경했다.

## 확인 기준

1. `/admin/files` 저장소 관리 화면으로 진입한다.
2. 휴지통 탭을 확인한다.
3. 컬럼 헤더와 목록 내용이 가운데 정렬되는지 확인한다.
4. `삭제일시` 라벨이 표시되는지 확인한다.
5. 첨부파일 복구 제한 메시지에서 `부모 작업지시서` 표현이 보이지 않는지 확인한다.

- R2 Worker 흐름 변경 없음.
- package.json / package-lock.json 변경 없음.
