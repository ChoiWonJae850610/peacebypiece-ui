# 0.19.93 개인프로필 언어 전환 개발전용 숨김

## 목표

개인 설정 화면의 한국어/영어 전환 UI를 고객 화면에서 숨기고, 추후 다국어 정책과 지원 언어가 확정될 때 다시 공개할 수 있도록 개발 전용 노출 플래그로 분리한다.

## 변경 기준

- 고객 화면 기본 상태에서는 언어 선택 카드를 렌더링하지 않는다.
- `NEXT_PUBLIC_APP_RUNTIME_MODE=development`일 때만 개발 확인용 언어 선택 카드를 렌더링한다.
- 언어 선택 UI가 숨겨진 상태에서는 기존 localStorage에 `language: "en"`이 남아 있어도 화면 적용 언어를 한국어로 보정한다.
- 개인 테마 변경 기능은 유지한다.
- 개인 프로필 저장/탈퇴 요청/약관·정책 진입 기능은 변경하지 않는다.

## 테스트 기준

- `/me/settings`에서 개인 프로필과 테마 섹션은 표시된다.
- 고객 화면 기본 상태에서 `English`, `한국어` 언어 버튼은 표시되지 않는다.
- localStorage에 영어 설정이 남아 있어도 `html.lang`은 `ko`로 보정된다.

## 후속 재공개 조건

다국어를 다시 공개할 때는 단순히 버튼을 복구하지 말고 다음 항목을 먼저 확정한다.

- 지원 언어 목록
- 정책/약관 문서 다국어 제공 범위
- PDF/발주서/작업지시서 문구 번역 범위
- 고객사 단위 언어와 개인 언어의 우선순위
- 시스템관리자 화면의 언어 지원 여부
