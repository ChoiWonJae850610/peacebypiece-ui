# 0.23.14 숫자 및 빈 값 표시 기준 통일

## 목적
- WAFL 숫자 입력과 요약 영역에서 천 단위 구분 및 0 표시 기준을 공통 formatter로 통일합니다.
- 금액 표시는 항상 `0원` 형태를 유지하고, 수량은 공통 단위 formatter를 사용합니다.
- 화면별로 중복 구현된 숫자 포맷 코드를 제거합니다.

## 적용 범위
- `WaflNumberInput`의 표시값 formatter
- 작업지시서 숫자 표시 및 금액 요약
- 공통 통화 표시 유틸리티

## 표시 기준
- 숫자: `1,234`
- 소수 허용 입력: `1,234.5`
- 금액: `1,234원`
- 0 금액: `0원`
- 빈 텍스트: 기존 공통 `EMPTY_DISPLAY` 정책 유지
