# 0.18.83 통계정보 기간 컨트롤 컨테이너 폭 기준 보정

## 목적

통계정보 기간 분석 컨트롤의 시작일/종료일, 7일/30일, 초기화/적용 버튼 배치가 기기별 viewport breakpoint에 따라 서로 다르게 보이는 문제를 줄인다.

## 변경 내용

- `AdminStatsPeriodControls`에 `useElementSize` 기반 컨테이너 폭 측정을 적용했다.
- 컨테이너 폭이 충분한 경우 시작일/종료일 필드와 4개 버튼을 한 줄에 배치한다.
- 컨테이너 폭이 부족한 경우 날짜 필드와 버튼 영역을 두 줄로 분리한다.
- 날짜 필드는 한 줄 모드에서 compact width를 유지하도록 제한했다.
- 7일/30일 프리셋, 초기화/적용 버튼의 동작과 스타일은 유지했다.

## 비변경 범위

- 통계 계산 로직 변경 없음
- 기간 적용 URL 흐름 변경 없음
- 달력 팝오버 구조 변경 없음
- WorkspaceShell 스크롤 구조 변경 없음
- DB/API 통계 조회 흐름 변경 없음
