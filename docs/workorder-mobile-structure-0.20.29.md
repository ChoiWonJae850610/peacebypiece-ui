# 작업지시서 모바일 구조 2차 (0.20.29)

## 적용 범위

- 작업지시서 모바일 상세 내부 탭 라벨을 `기본 / 공정 / 구성` 기준으로 정리했다.
- 모바일 상세 하단 액션바에서 `첨부 / 디자인 / 메모`를 각각 직접 열 수 있게 분리했다.
- 모바일 관련 정보 시트 내부에서도 `첨부 / 디자인 / 메모` 세그먼트 탭으로 전환할 수 있게 했다.
- 첨부와 디자인은 기존 `WorkOrderAttachmentPanel`을 재사용한다.
- 메모는 기존 `WorkOrderMemoPanel`을 재사용한다.
- 디자인 드로잉 모달은 기존 세션 상태 유틸을 재사용해 모바일 분리 패널에서도 유지했다.

## 유지한 범위

- PC 3패널 구조는 변경하지 않았다.
- 태블릿 2패널 구조는 변경하지 않았다.
- 상태전환 로직은 변경하지 않았다.
- 권한 로직은 변경하지 않았다.
- API/DB/R2/첨부/메모/휴지통/purge 흐름은 변경하지 않았다.

## 다음 검토 후보

- 첨부/PDF/이미지 미리보기는 이후 파일 미리보기 공통화 단계에서 full-screen modal 기준으로 재정리한다.
- 메모 입력은 현재 기존 패널을 유지하고, 이후 bottom sheet 또는 별도 입력 화면 후보를 비교한다.
- empty/loading/error 상태는 현재 기존 컴포넌트를 유지하고, 모바일 QA 단계에서 공통 상태 UI로 추가 정리한다.
