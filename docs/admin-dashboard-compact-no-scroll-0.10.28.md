# 0.10.28 통계 화면 무스크롤 목표 압축 보정

## 목적
PC 기준 통계 화면에서 생산 구성/기간 분석 탭이 화면 아래로 밀리는 문제를 줄이고, 탭 구조를 유지하면서 한 화면 운영판에 가깝게 압축한다.

## 반영
- 상단 누적 카드 4개의 최소 높이와 내부 여백을 줄였다.
- 현재 저장소 사용량 카드의 MB 보조 문구는 화면에서는 숨겼다.
- 통계 섹션의 DASHBOARD SECTIONS eyebrow 문구를 제거했다.
- 통계 섹션 외부 여백과 탭 콘텐츠 상단 여백을 줄였다.
- 생산 구성 도넛 그래프 영역을 위로 당기고 하단 보조 문구 간격을 줄였다.
- 기간 분석 Top5 카드의 기준 설명 문구를 제거했다.
- 기간 분석 필터 영역과 카드 간격을 줄였다.

## 비변경
- 통계 계산 로직 변경 없음.
- 통계 API/repository 변경 없음.
- DB schema 변경 없음.
- 감사 로그 흐름 변경 없음.
