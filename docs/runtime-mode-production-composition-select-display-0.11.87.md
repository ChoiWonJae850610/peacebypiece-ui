# 0.11.87 작업지시서 생산구성 선택 표시 보정

## 목적

작업지시서 생산구성에서 원단, 부자재, 외주공정이 `선택 안함` 상태일 때 거래처/외주처 선택 UI가 빈 줄처럼 보이지 않도록 표시 기준을 통일한다.

## 적용 기준

- 원단/부자재 유형이 `선택 안함`이면 거래처 dropdown 후보는 `등록된 업체 없음`으로 유지한다.
- 외주공정이 `선택 안함`이면 외주처 dropdown 후보는 `등록된 업체 없음`으로 유지한다.
- 사용자가 실제로 선택한 값이 없는 상태의 표시값은 `-`로 통일한다.
- `등록된 업체 없음`은 편집용 후보 표시로만 사용하고 실제 선택값으로 저장하지 않는다.

## 수정 범위

- PC 생산구성 table
- 모바일 생산구성 card
- 태블릿 생산구성 card
- 선택값 표시용 공통 presentation helper

## 제외 범위

- DB schema 변경 없음
- full_reset.sql 변경 없음
- R2/첨부/메모/휴지통/purge 변경 없음
