# 0.11.40 관리자 저장소 모바일 화면 보정

## 목적

`/admin/files` 저장소 관리 화면을 모바일/태블릿 폭에서 사용할 때 상단 요약 카드, 휴지통 액션 버튼, 목록 영역, 확인/상세 모달이 화면 밖으로 밀리지 않도록 보정한다.

## 반영 범위

- 관리자 저장소 페이지 외곽 wrapper의 모바일 overflow 보정
- 저장소 요약 카드 3종의 모바일 padding, radius, stacking 보정
- 휴지통 action bar 버튼의 모바일 폭 분배 보정
- `AdminTable` body의 가로/세로 overflow 처리 보정
- 저장소 목록/작업지시서 저장소 목록 wrapper의 모바일 padding 보정
- 휴지통 confirm/detail modal footer 버튼의 모바일 세로 배치 보정

## 변경하지 않은 것

- 고객관리자 휴지통 복원 로직
- 고객관리자 선택 삭제 요청 로직
- 시스템관리자 R2 purge 로직
- 저장소 snapshot API
- DB schema
- 파일 업로드/다운로드/R2 key 정책

## 화면 확인 항목

### `/admin/files`

- 모바일 폭에서 상단 요약 카드가 1열로 내려오는지 확인한다.
- 태블릿 폭에서 요약 카드가 너무 좁게 찌그러지지 않는지 확인한다.
- 휴지통 버튼 4개가 모바일에서 줄바꿈되어 터치 가능한 크기로 표시되는지 확인한다.
- 휴지통 목록이 화면 밖으로 잘리는 대신 내부 스크롤로 접근 가능한지 확인한다.
- 휴지통 empty 상태 문구가 중앙에 안정적으로 표시되는지 확인한다.

### 휴지통 모달

- 휴지통 비우기 확인 모달의 예/아니오 버튼이 모바일에서 세로로 안정적으로 표시되는지 확인한다.
- 작업지시서 복원/삭제 범위 확인 모달의 footer 버튼이 화면 밖으로 밀리지 않는지 확인한다.
- 휴지통 상세 모달에서 파일명과 작업지시서명이 긴 경우에도 footer 버튼이 접근 가능한지 확인한다.
