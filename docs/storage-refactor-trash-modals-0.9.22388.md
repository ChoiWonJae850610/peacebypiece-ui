# 0.9.22388 저장소 휴지통 모달 컴포넌트 리팩토링

## 목적

고객관리자 저장소 휴지통 화면의 상세/확인 모달 렌더링 책임을 `FileTrashSection.tsx`에서 분리한다.

## 변경 내용

- 휴지통 비우기 확인 모달을 `EmptyTrashConfirmModal`로 분리
- 작업지시서 복원/선택 삭제 범위 확인 모달을 `WorkOrderActionPreviewModal`로 분리
- 휴지통 상세 모달을 `TrashDetailModal`로 분리
- 상세 필드 계산과 미리보기 링크 문구를 모달 전용 파일 내부로 이동
- 기존 휴지통 row 생성, 컬럼 렌더링, 선택 상태 계산, 복원/선택 삭제/비우기 액션은 유지

## 유지 정책

- DB schema 변경 없음
- R2 Worker 기반 흐름 유지
- package.json / package-lock.json 수정 없음
- 휴지통 버튼명, 비우기 확인 문구, 작업지시서 묶음 안내 문구 유지

## 확인 항목

1. `/admin/files` 휴지통 목록 표시
2. 비우기 버튼 클릭 시 예/아니오 확인 모달 표시
3. 작업지시서 row 복원/선택 삭제 범위 확인 모달 표시
4. 파일/작업지시서 상세 모달 표시
5. 복원 / 선택 삭제 / 비우기 동작 유지
