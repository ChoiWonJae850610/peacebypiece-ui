# 0.9.22389 저장소 휴지통 삭제 모달/리오더 표시 보정

## 변경 목적

작업지시서 삭제 모달의 안내 문구를 첨부파일 삭제 모달과 같은 빨간 안내 문구로 통일하고, 저장소 휴지통 목록과 상세 모달에서 리오더 차수가 포함된 작업지시서명을 표시한다.

## 반영 내용

- 작업지시서 삭제 모달의 기존 안내 문구 2개를 제거했다.
- 작업지시서 삭제 모달 안내 문구를 `삭제된 파일은 휴지통으로 이동하며, 30일 이내 복원할 수 있습니다.`로 통일했다.
- 안내 박스 색상을 첨부파일 삭제 모달과 같은 빨간 계열로 맞췄다.
- 저장소 휴지통 snapshot에서 작업지시서 표시명을 `title/base_title/reorder_round/work_order_kind/is_rework` 기준으로 계산하도록 보정했다.
- 휴지통 작업지시서 row, 연결 첨부파일의 작업지시서명, 상세 모달 description/본문에 같은 표시명이 사용되도록 했다.

## 확인 항목

1. `/worker`에서 리오더 작업지시서 삭제 모달을 연다.
2. 기존 노란 안내와 회색 안내가 사라졌는지 확인한다.
3. 빨간 안내 박스에 `삭제된 파일은 휴지통으로 이동하며, 30일 이내 복원할 수 있습니다.`가 표시되는지 확인한다.
4. `/admin/files` 휴지통 목록에서 리오더 작업지시서명이 `리오더 n차`까지 포함되는지 확인한다.
5. 작업지시서 휴지통 상세 모달과 연결 첨부파일의 작업지시서명도 목록과 같은지 확인한다.
