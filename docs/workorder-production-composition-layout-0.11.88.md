# 작업지시서 생산구성 레이아웃 보정 0.11.88

## 목적

작업지시서 상세 화면의 발주정보와 생산구성 영역에서 내부 카드 depth와 테이블 폭을 정리한다.

## 반영 내용

- 생산구성 상위 카드 padding과 내부 section 간격을 축소했다.
- 원단/부자재 테이블에 최소 폭을 부여해 좁은 PC 폭에서 컬럼이 과도하게 압축되지 않도록 했다.
- 외주공정 테이블에 최소 폭을 부여하고 단가 기준, 단가, 금액 컬럼 폭을 보정했다.
- 발주정보 테이블에도 최소 폭을 부여해 날짜, 수량, 공임비, 로스비 컬럼의 압축을 완화했다.
- 태블릿 생산구성 영역은 바깥 카드와 내부 카드의 시각 depth를 낮추고 spacing을 축소했다.

## 확인 기준

- PC 작업지시서 상세 화면에서 원단/부자재 단위 값이 잘리지 않는지 확인한다.
- PC 작업지시서 상세 화면에서 외주공정 단가 기준 값이 잘리지 않는지 확인한다.
- 태블릿 작업지시서 상세 화면에서 생산구성 카드가 과하게 겹쳐 보이지 않는지 확인한다.
- 모바일 화면은 이번 변경의 직접 대상이 아니며 기존 카드형 구조를 유지한다.
