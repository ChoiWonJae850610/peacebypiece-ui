# 0.11.66 관리자 환경설정 탭 콘텐츠 1차 보정

## 목적

0.11.63에서 환경설정 IA를 탭형 카드와 하단 패널 구조로 바꾼 뒤, 요금제·저장공간 / 계정 정보 / 개발 건의 패널의 PC 화면 밀도와 콘텐츠 배치를 1차 보정한다.

## 반영 내용

- 환경설정 상단 메뉴 카드 높이를 축소했다.
- 메뉴 카드의 radius와 detail chip 간격을 조정해 PC 화면에서 과하게 큰 카드처럼 보이지 않게 했다.
- 요금제·저장공간 패널을 지표 영역과 요청/문의 영역의 2단 구조로 재배치했다.
- 계정 정보 패널을 조직/개인 설정 지표 영역과 요청/이동 영역의 2단 구조로 재배치했다.
- 개발 건의 패널을 접수 항목과 적용 예정 안내 영역으로 분리했다.
- 각 패널의 최소 높이를 낮춰 하단 패널이 화면을 과도하게 차지하지 않도록 했다.

## 유지한 내용

- 기준정보 설정 탭의 기존 `AdminStandardsSection` 흐름은 변경하지 않았다.
- 요금제·저장공간은 읽기 전용 정책을 유지했다.
- 계정 정보는 읽기 전용 정책을 유지했다.
- 개발 건의는 이메일 접수 흐름을 유지했다.
- API, DB schema, 저장 로직은 변경하지 않았다.

## 확인 항목

- `/admin/settings` 진입 시 상단 탭 카드가 기존보다 과하게 크지 않은지 확인한다.
- 요금제·저장공간 탭에서 지표와 요청 카드가 PC에서 좌우로 자연스럽게 분리되는지 확인한다.
- 계정 정보 탭에서 개인 설정 이동 카드가 정상 링크로 동작하는지 확인한다.
- 개발 건의 탭에서 이메일 작성 버튼이 기존처럼 동작하는지 확인한다.
- 모바일 폭에서 모든 패널이 1열로 자연스럽게 쌓이는지 확인한다.
