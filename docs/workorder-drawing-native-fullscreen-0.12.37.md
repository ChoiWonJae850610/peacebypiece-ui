# 작업지시서 직접 그리기 full-screen editor 전환 — 0.12.37

## 목적

직접 그리기 화면을 일반 모달이 아니라 태블릿 작업용 전체 화면 편집기처럼 사용하도록 정리한다.

## 변경 기준

- 배경 작업지시서 화면이 보이지 않도록 직접 그리기 panel을 viewport 전체 크기로 확장한다.
- 상단 설명 문구를 제거하고 제목과 닫기 버튼만 남긴다.
- 중앙 캔버스가 남는 화면 높이를 최대한 사용하도록 한다.
- 하단 도구 모음과 저장 액션은 항상 보이도록 같은 하단 영역 안에 둔다.
- 태블릿 세로 화면에서는 하단 도구 모음이 두 줄 이상으로 감길 수 있게 한다.
- backdrop click 닫기 방지는 유지한다.
- 브라우저 뒤로가기 방지와 저장 전 닫기 확인 흐름은 유지한다.

## 제외한 내용

- 확대/축소
- 손바닥 이동
- 이미지 위에 그리기
- 기본 의류 템플릿 배경
- DB schema 변경
- R2/API 흐름 변경

## 확인 항목

1. 태블릿 가로/세로에서 직접 그리기 화면이 거의 전체 화면으로 보이는지 확인한다.
2. 상단 설명 문구가 사라지고 header 높이가 줄었는지 확인한다.
3. 캔버스와 하단 도구/저장 영역이 스크롤 없이 함께 보이는지 확인한다.
4. 하단 도구가 좁은 폭에서 자연스럽게 줄바꿈되는지 확인한다.
5. 저장 후 기존 디자인 첨부 목록에 PNG가 추가되는지 확인한다.
