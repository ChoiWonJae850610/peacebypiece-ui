# 0.18.81 통계정보 분석 섹션 탭 정렬 보정

## 목적

통계정보의 분석 통계 header에서 생산 구성 / 업체 성과 / 기간 분석 탭이 태블릿 일부 화면에서 제목 영역 하단에 붙어 보이는 문제를 보정합니다.

## 변경 내용

- 분석 통계 header의 좌측 제목 영역과 우측 탭 영역을 md 이상부터 같은 row로 배치했습니다.
- md 이상에서 탭 영역이 제목 블록의 상단 기준에 맞춰 정렬되도록 보정했습니다.
- 모바일 좁은 화면에서는 제목/설명 아래에 탭이 내려오는 기존 흐름을 유지했습니다.
- 통계 계산, 탭 전환, 기간 선택/적용, DB/API 흐름은 변경하지 않았습니다.

## 확인 항목

- PC에서 제목/설명과 탭이 같은 상단 기준으로 보이는지 확인합니다.
- 태블릿 가로/세로에서 탭이 제목 블록 하단이 아니라 상단 기준으로 정렬되는지 확인합니다.
- 모바일 세로에서 레이아웃이 깨지지 않는지 확인합니다.
