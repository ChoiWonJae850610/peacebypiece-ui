# 0.19.16 멤버 초대 목록 컬럼과 disabled icon action 보정

## 목적

멤버관리 초대 목록에서 초대 링크 일부 표시를 제거하고, 관리자가 실제로 판단해야 하는 상태·생성일·만료일·작업만 남긴다.

## 반영 내용

- 초대 링크 컬럼 제거
- 생성일은 날짜와 시간:분:초까지 표시
- 만료일은 날짜만 표시
- 작업 컬럼은 글자 없는 icon action button만 사용
- 사용 가능한 초대는 복사/취소 icon button 활성
- 취소됨/만료됨/사용됨 등 사용할 수 없는 초대는 복사/취소 icon button을 disabled 상태로 유지
- 버튼이 완전히 사라져 행별 작업 위치가 흔들리는 것을 방지

## 변경하지 않은 것

- 초대 생성 API
- 초대 취소 API
- 초대 링크 복사 동작
- 멤버 승인/권한 저장 흐름
- DB schema
